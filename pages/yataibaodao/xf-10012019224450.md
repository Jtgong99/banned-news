 <p>
  十月一号，当北京举行国庆大典，歌舞升平的时候，香港则是警民暴力升级，发生警察真枪实弹打伤抗议者的事件。本台记者郑崇生整天都在抗议现场进行报道，我们请她介绍她的所见所闻。
  <br/>
  <br/>
  记者：郑崇生，你好。
  <br/>
  <br/>
  郑：嘉远,你好。
  <br/>
  <br/>
  记者：请你跟我们描述一下十月一号当天香港街头发生的抗议情况好吗？
  <br/>
  <br/>
  郑崇生：好的。香港今天可以说完全没有感受到中共在北京庆祝建政70周年的喜悦， 而是发生了反送中运动以来首次喋血街头的情况。今天抗议活动在香港的18个区遍地开花，尤其在九龙半岛包括了荃湾、屯门、深水埗附近，从下午开始就不断的有激烈的警民的对峙和冲突的状况。除了抗议民众有扔掷汽油弹之外，在荃湾发生了香港警方对抗议民众实弹开枪的情景。警方的说法是因为这些抗议的民众他们形容是暴民，他们对警方的执法产生了生命的威胁。也因此香港这座城市的在今天整个下午弥漫着汽油弹以及催泪弹的味道。我在现场感受到的是香港警方确实执法开始不断地升级。尤其在轩尼诗道上不断地平射释放催泪瓦斯来清场，这是过去没有的一个状况。
 </p>
 <p>
  记者：据你了解，有多少人受伤，受伤 程度怎么样？
  <br/>
  <br/>
  郑崇生：到今天晚上8点香港医管局统计是有51个人受伤，送医院。其中有两个人是性命垂危，有两个人是伤势严重。两个性命垂危其中之一应该就是在荃湾发生的香港警方开实弹射击的受伤学生。最新的消息是，根据香港的媒体报道，他已经在进行手术了。伤者中年纪从11岁到75岁都有。这就是截至目前到今天晚上的最新的情形。
  <br/>
  <br/>
  记者：你刚才提高有75岁的民众受伤。这些人是示威者还是旁观者呢？
  <br/>
  <br/>
  郑崇生：目前香港警方并没有说，医管局并没有详细的资料。不过根据我们现场采访经验看到的是，像今天下午6点多一些比较年长者的抗议人士参加完游行活动之后，在湾仔著名的修顿球场上稍作休息就要返家。不过因为在轩尼诗道上开始不断地出现了释放汽油弹情况之后，香港警方丢掷催泪弹，甚至是无差别就是没有特定目标。警方甚至从街道上从修顿球场内部丢掷催泪弹，引起许多在旁边休息的长者破口大骂，说香港警方这样的执法根本就是慌乱无章、没有战术、也没有战略，那根本就是伤及无辜。警方的执法引起很多民众情绪相当的激动。截至到晚上，刚刚在铜锣湾站都还发生了气愤的民众在地铁站口的焚烧物品的情形。
  <br/>
  <br/>
  记者：香港政府或者特首是在北京参加国庆活动。其他的香港政府官员，高级官员今天有没有对这一事件表态？
  <br/>
  <br/>
  郑崇生：香港特首林郑月娥是率领200多人的代表团在天安门广场上和中国国家主席参与了中国国家主席习近平主导的阅兵以及纪念建国70周年的大庆。而对香港的这些民众来说，他们相当的气愤，没有感受到喜悦。尤其在荃湾发生的有民众受到警方开枪之后。等到林郑(月娥)回来她要如何修补警民之间高度对峙以及不信任感呢，我觉得这是一个很大的挑战。尤其是在刚刚之前，警民会的前委员还为今天下午的执法替警察说话。他们认为开枪是合理且合法的。这也是为什么抗议的民众情绪到了晚间仍然是非常的激动。他们一直喊出的口号是“五大诉求缺一不可”。其中一个就是要成立独立的针对警方的调查委员会。林郑会怎么回应我们还在看。不过香港目前的情绪除了警民的不信任感高度对峙之外，我们还观察到有支持警方以及反对警方这两股的势力也常常在街头，就可以听到零星的骚动啊、口角争执。我觉得香港现在已经从过去大家非常熟悉的美食飘香的东方之珠成为了是催泪弹弥漫的城市。接下来香港这座城市要怎么修补伤口可能需要各方要付出很大的努力。像今天中国国家主席习近平他在天安门广场上致词的时候还特别提到了要坚持一国两制。然后接下来他讲到的是和平统一。如果说坚持一国两制是讲给香港人听，那和平统一是讲给台湾人听。在下午一国两制下的香港已经发生了街头的流血事件。我想在香港和台湾的民众都会有深刻的感受。北京要如何对待港台，尤其是现在香港的情况，处理起来我相信是更为棘手的。
  <br/>
  <br/>
  记者：好的。谢谢郑崇生给我们所作的描述和报道。
  <br/>
  <br/>
  郑崇生：谢谢。
  <br/>
  <br/>
  责编：申铧 网编：郭度
