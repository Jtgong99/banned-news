 </strong>
 <br/>
 本人王新平，在中国海军上海水警区服役16年，2014年底退役回乡，在此之前是共产党灌输的政治思想教育。回到家乡以后，2018年六月初因为修建新204国道，我家两处房屋需要拆迁，在宣传的口号里，共产党声称法律面前人人平等，现实中，法律只是共产党手里的大棒，不管政策是否合理，只要制定了政策，打压一片反对的声音，当地共产党政府在没有依据法律程式的情况下，以政治思想教育为前提，多次找我谈话，不堪骚扰。声明是纪检委，香河街道山庄村村委会，以我的党员身份要绝对服从组织安排，以我的党员身份无条件签订拆迁空白协议，不然强制拆除房屋，每平方补助680元人民币，安置两处宅基地（到现在也没有安置），本人和我母亲刘美英拒绝签订（本人有我母亲空白协定手机照片），六月中旬当地共产党政府单方面公布拆迁补偿数额，多次怂恿亲朋好友，实行连带劝说，不然以党纪国法处理我，威胁我，最后我和我母亲只有签订空白协定。我服役16年，按照退役要求，我符合安置工作条件，只因为老家没有关系没有市政直属单位的亲属，不属于第一批安置，只有要求自主择业。
 <br/>
 中国是一个不尊重人权，没有民主的国家，人权只有在政治舆论里是高高挂起的，法律保护不了太多的民众，却伤害了太多民众的利益，法律是空白的，无力的，暗淡的，我们还有呐喊的权利？没有！共产党不会听到反对的声音，共产党没有民主，共产党的官员都不是民众选举出来的，他们的权力大于法律，操纵法律打压反对和呐喊！
 <br/>
 从房屋被拆迁后，我和我母亲一直上访，到市信访局多次，到东港区信访局多次，最后层层推诿，到了我们香河街道政府，还是让我们服从安排！打日照市市长热线，没人理会，去日照市纪检委反映拆迁违规违法，答复是纪检委不接受拆迁方面的违纪问题，去大众论坛写帖子公布于众，最后香河街道一个孙主任暴怒，直接打村委会威胁，在中国共产党人民网的地方领导留言板多次留言，回复是本人发帖涉嫌违法，在国务院互联网+督察上留言反映问题，得不到任何反映和答复，本人不管是留言，上访，发帖子，还是打电话反映事实问题，都是实名，还有告知本人的家庭住址和身份证号码，绝不匿名。
 <br/>
 这是一个什么样的的党？2019年五月，根据山东省委巡视组到达岚山区巡视，本人多次反映问题，他们依据实情给予记录，却得不到任何答复，这是一个怎样的国家？是一个国富民弱的国家，是一个存在特权阶层的国家，他们可以操控法律，肆意制定所谓的政策打压民众！
 <br/>
 本人在此声明，退出中国共产党！2019年6月7日端午节
