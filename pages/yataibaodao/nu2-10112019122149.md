 <p>
  正当美国政府向盟国就中国科技巨头华为的安全威胁提出警告之际，美国联邦调查局也在把注意力投向美国各大学校园，提醒和培训大学管理人员如何防止知识产权被偷窃。
  <br/>
  <br/>
  一位美国联邦调查局 (FBI) 特工在给美国伊利诺伊大学厄巴纳-香槟分校 (University of Illinoise-Urbana-Champaign) 负责研究事务的副校长的一份电邮中，询问他们是否认为中国公司华为偷窃了该校的知识产权。作为试图阻截中国政府指派的研究人员大规模偷窃美国的知识产权和商业机密的努力的一部分，联邦调查局近来正在与全国各地的高校进行联系。
 </p>
 <div>
 </div>
 <div>
  美国媒体美联社向美国50个州多家大学发出提阅公共信息的申请后，获取了很多联邦调查局和这些大学的管理人员之间的电子邮件，并从中了解到美国政府在防止知识产权盗窃方面所做的努力。
  <br/>
  <br/>
  这些电邮信件显示，各大学校方官员对联邦调查局的警告表达了欢迎的态度，还提出要联邦调查局人员为他们做进一步讲解的请求。但同时，他们似乎也在合法的国家安全担忧，以及避免影响研究工作和抹黑清白学者之间寻求平衡。
 </div>
 <div>
  <br/>

 <div>
  为此，联邦调查局在各大学里进行了有关的辅导，并向各大学校方官员介绍了有关情况。他们还向大学生们散发了相关的传单。
  <br/>
  <br/>
  联邦调查局特工人员的这些电邮还显示，在过去两年间，他们曾要求得到华盛顿大学两位研究人员的电子邮件，还向俄克拉荷马州立大学询问了它在某些特定研究领域里的研究人员，还向科罗拉多大学波尔得分校问询了某位教授是否存在滥用学术经费的问题等。
  <br/>
  <br/>
  美国司法部表示，它理解校方所面临的满足国家安全需求和保护学术自由和研究人员之间的矛盾心情。他们只是希望帮助各大学把少数偷窃知识产权者从大多数合法从事学术研究的人员中分辨出来。
 </div>
 <div>
  <br/>

  <br/>
  联邦调查局也表示，他们并不鼓励高校对研究人员进行基于国籍的监督，而是采取应有措施保护研究结果，并对可疑行为提高警惕。
  <br/>
  <br/>
  联邦调查局开展这番工作的同时，美国国防部和能源部也推出相关措施，对他们给各大学提供的研究经费设置一些限制。
  <br/>
  <br/>
  美国政府长期以来一直指控中国盗窃美国的知识产权，但是中国政府一直否认。
  <br/>
  <br/>
  美联社的报道援引美国政府反间谍部门的负责人威廉·伊凡尼娜的话说，在情报搜集这方面来讲，中国是美国最大的威胁。
  <br/>
  <br/>
