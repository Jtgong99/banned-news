 <p>
  美中两国新一轮贸易谈判周五结束。特朗普当天宣布，两国已经达成了一份涉及农产品和关税在内的局部协议。中方近期释出了一系列积极信号，推动双方在部分问题上达成共识。
  <br/>
  <br/>
  特朗普周五在会见了中国副总理刘鹤后对媒体表示，美中谈判双方已经达成了一份“非常充实的初步协议”。中方将在农产品方面作出一些让步，而美国将推迟关税加征时间。
  <br/>
  <br/>
  这份协议将推迟原定于本月15日上调的对部分中国商品的税率。作为回报，中国同意购买价值400亿到500亿美元的美国农产品，并进一步对外开放金融市场。但美国贸易代表莱特希泽表示，特朗普尚未决定是否会按原计划执行于12月15日加征的新一轮关税。
 </p>
 <div>
 </div>
 <div>
  此前，美国计划在本月15日将2500亿美元的中国商品的关税税率从25%上调至30%，并于12月15日对1600亿美元的中国商品加征15%的关税。
  <br/>
  <br/>
  特朗普周五一早发推说，美中贸易谈判正在取得很好的进展。与此前一段时间相比，双方都释出了更友好的信号。他说，大家都期待看到一些大事发生。
  <br/>
  <br/>
  在此之前，美国商会常务副会长薄迈伦(Myron Brilliant)在听取了两国谈判小组简报后，周四曾对媒体表示，两国可能会首先达成汇率协议，遏制中国蓄意贬值人民币的行为。几个月来，外界认为双方可能会达成某种临时协议，但特朗普随后表示他更希望达成全面协议。
  <br/>
  <br/>
  据悉，双方此次并未能达成汇率协议，但美国财政部长姆努钦表示相关谈判几乎已经完成了。
 </div>

 <div>
  薄迈伦周三在华盛顿会见刘鹤时表示，贸易战只有输家，没有赢家，美国工商界不希望看到加征关税。刘鹤则表示，中方此次抱着极大的诚意而来，愿意与美方就贸易平衡、市场准入等问题进行深入交流，推动谈判取得进展。
  <br/>
  <br/>
  中国金融学者贺江兵认为，双方希望在此时达成所谓的“迷你协议”，是因为他们都不急于签署一份人心所向的全面协议。
  <br/>
  <br/>
  “双方最大的利益共同点就是都不达成全面协议，他们都必须要拖下去，但对内对外都不能这样讲，可是双方都不能把这个责任揽到自己身上。”
  <br/>
  <br/>
  他补充说，民主党目前主导美国众议院，而特朗普并不愿意为了尽早结束贸易战，就中国政府进行结构性改革的要求作出过多的让步，以免造成不必要的国内政治风险。
  <br/>
  <br/>
  外媒引述薄迈伦说，他认为双方这次有望达成汇率协议的意图之一，就在于促使美国政府决定暂缓在本月15日上调对华商品的关税。
  <br/>
  <br/>
  华盛顿信息与战略研究所所长李恒青分析说，人民币几个月来大幅贬值主要是市场因素导致的。如果当局没有动用大量外汇购入人民币、任由市场决定的话，美元兑人民币汇率“破八”都是相当容易的。
  <br/>
  <br/>
  因此，李恒青表示，两国达成汇率协议不成问题，但关键点仍然在于如何确保中国今后不会反悔。
 </div>

  “如果中国今后再去人为操纵汇率的话，那么国际货币基金组织和美元体系会把中国踢出去。它一定有这样的一个执行机制，而这件事不容易达成共识。”
  <br/>
  <br/>
  除了汇率协议之外，业内人士期待美中双方本次能在另外一些问题上达成共识，比如中国承诺采购更多的美国农产品、美国放宽对中国通信巨头华为的出口禁令。
  <br/>
  <br/>
  与此形成鲜明对比的是，在本次谈判前夕，美方的一系列动作被认为可能会惹恼中国。
  <br/>
  <br/>
  美国商务部周一把28家中国公安机关和协助当局监控新疆的中企列入了“实体名单”，禁止他们在未经许可的情况下向美企购买零部件。次日，美国国务院宣布将对镇压新疆穆斯林的中方官员实行签证限制。此外，NBA火箭队总经理莫雷(Daryl Morey)上周发推挺香港，引发了中国社会的强烈不满和抵制。
  <br/>
  <br/>
  正因如此，金融学者贺江兵说，双方这次肯定会达成一份“短期协议”。
  <br/>
  <br/>
  “两国都发生了很多剧变。美国主要是弹劾川普，中国发生了太多事了，又是香港、又是NBA。双方的压力不同，但都需要一份‘短期停火协议’。”
  <br/>
  <br/>
  对此，李恒青也表示两国很有可能达成此类协议。他解释说，中共将于近日召开十九届四中全会，届时会对中国国家领导人习近平过去7年的执政表现进行总结。他表示，刘鹤此行的首要目的就是暂缓下周即将上调的中国商品税率，让习近平“开成会”。
  <br/>
  <br/>
