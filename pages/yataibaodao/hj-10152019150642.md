 <p>
  美国国会众议院15日以口头表决的方式, 无异议通过三项涉及香港的议案，为持续5个月的香港“反送中”抗争运动注入强心剂。
  <br/>
  <br/>
  美国众议院全院星期二就《香港人权及民主法案》、《保护香港法案》（Protect Hong Kong Act），还有承认香港与美国的双边关係、谴责中国政府干预香港事务，支持港人示威权利的《与香港站在一起》决议案（Stand With Hong Kong Resolution）进行讨论审议。三项议案当天下午轻松获得一致通过。
  <br/>
  <br/>
  美国众议院外交委员会亚太小组委员会首席民主党议员舍尔曼(Brad Sherman)在会议上强调，上述三项法案意味着美国众议院与香港示威者站在一起，并支持他们所提出的五大诉求。
  <br/>
  <br/>
  “（香港政府提出的）《逃犯条例》修订是在侵犯香港人的权利、腐蚀香港高度自治（原则）。中国政府不但没有承认香港的示威行动是因其而起，反而指责美国煽动这场抗争，美国拒绝这样的说法。”
 </p>
 <div>
 </div>
 <div>
  值得注意的是，美国国防部负责印太安全事务的助理部长薛瑞福（Randall Schriver）星期二在华盛顿智库詹姆斯敦基金会举行的研讨会上也表示，美国不会为了避免被称为香港抗议的“幕后黑手”而改变自己的原则，美国“百分之百”支持那些呼吁尊重1984年《中英联合》声明所赋予的基本权利的香港人。
  <br/>
  <br/>
  薛瑞福也提到，美国对香港抗议者的某些做法感到担忧，同时也担心香港和中国政府对抗议者的重手干预。
  <br/>
  <br/>
  《香港人权与民主法案》主要提案人之一、共和党众议员史密斯（Chris Smith）则称赞香港人说：“他们在示威运动中体现出的决心、勇气和坚韧像太阳一样在黑暗中闪耀。”
 </div>

 <div>
  史密斯议员特别提到中国国家主席习近平日前有关“任何人企图在中国任何地区搞分裂，都会粉身碎骨”的言论。他认为，中国官方对待异议人士手段残暴，大规模监禁新疆穆斯林这样的反人类罪行却甚少被追责。
  <br/>
  <br/>
  史密斯说，他希望这些针对香港的法案能为美国采取有意义的行动提供一张蓝图。
  <br/>
  <br/>
  在谈到《保护香港法案》和《与香港站在一起》决议案时，美国众议院外交事务委员会主席恩格尔（Eliot Engel）表示，美国与香港的关系是基于双方拥有共同的价值观，其中包括言论自由和集会权利。美国对香港警方暴力镇压示威者的情况感到极度担忧。
  <br/>
  <br/>
  “那些承诺保护市民的香港警察向要求民主的示威者发射催泪弹、胡椒喷雾、橡胶子弹。《保护香港法案》将确保美国企业不会助长那些针对香港勇敢示威者的暴力行为。”
  <br/>
  <br/>
  众议院议长南希·佩洛西（Nancy Pelosi）在众院发言说，自30年前天安门广场镇压发生以来，美国两党议员曾多次就中国的人权问题发声谴责，但每一次都因牵扯到商业利益而没有成功。
  <br/>
  <br/>
  “如果我们（美国）因为商业利益而不就中国的人权状况发声的话，我们将失去批评其他地方所发生人权侵犯事件的道德权威。”
  <br/>
  <br/>
  据了解，《香港人权及民主法案》、《保护香港法案》以及《与香港站在一起》仍需经由参议院全体会议通过，并在特朗普总统签署后才能成为法律。
  <br/>
  <br/>
  （记者：韩洁 责编：申铧 网编：郭度）
