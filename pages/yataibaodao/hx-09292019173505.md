 <p>
  周日，台北约十万人在大雨中参加了“撑港、反极权”大游行。香港歌手何韵诗在参与过程中被泼红漆。总统蔡英文在脸书指示依法严惩肇事者。
  <br/>
  <br/>
  香港歌手何韵诗是在大游行之前，接受媒体联访时遭到袭击的。当时，一名穿戴着“反送中”典型装备的男子，持一瓶红油漆往她头上泼洒，何韵诗的头发、脖子和衣服都沾染红漆。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  由于事发突然，媒体记者和旁观者一阵尖叫，何韵诗则相当镇静，先是低头不语，伸手摸头发确定被泼的是什么东西。之后何韵诗稍微擦拭后继续对媒体讲话。
 </p>
 <p>
 </p>
 <p>
  <iframe frameborder="0" height="620" scrolling="no" src="https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRFAChinese%2Fvideos%2F678035526017648%2F&amp;show_text=0&amp;width=622" width="622">
  </iframe>
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  何韵诗说：“这个很明显可以看到亲中人士非常低级的素质，我们在香港其实每一天都在面对这样子的威胁，我们的安全其实在香港很多社会运动人士每天都在面对这种状况。我觉得这个其实就是很明显的打压跟恐吓。我觉得香港人面对这种状况是不会退下，也不会因此被吓怕的。”
  <br/>
  <br/>
  <strong>
   何韵诗：台湾拥有真正民主 却出现红色恐怖
  </strong>
  <br/>
  <br/>
  何韵诗稍后在游行主办单位之一、台湾公民阵线发起人江旻谚和立委林昶佐陪同下在济南教会召开记者会。
 </p>
 <p>
 </p>
 <p>

  <br/>
  何韵诗强调，这个状况本来以为只会在香港发生，因为香港现在就是面临一国两制被完全破灭的状况，但是发现在台湾，一个拥有真正民主的地方，竟然会出现红色恐怖的状况，有这种暴力的行为。中正一分局分局长沈炳信接受自由亚洲电台采访说，“他是被现场民众制伏以后转交给现场的警方，警方在现场就将该男子跟另一个同伙带回分局，何小姐有提告，已经来我们分局做完提告。”警方表示，将依涉及违反集会游行法、公然侮辱罪等罪名将两人送办。
  <br/>
  <br/>
  <strong>
   台媒：泼漆男子为统派政党干部
   <br/>
  </strong>
  <br/>
  有台湾媒体报道，泼漆的胡姓男子是中华统一促进党分支党部的主委，曾在独派聚会闹场，还曾对独派自由台湾党主席蔡丁贵喷生发剂。另一名同伙梁男则疑有黑帮背景和前科。但对媒体的这些报道，中正一分局分局长沈炳信只表示：“我这边的资料还没有，我们还在查明中。”
 </p>

  <br/>
  蔡英文今天在脸书表示，对游行集会蓄意挑衅破坏、施行暴力的人，已经被警方逮捕，随即要移送司法侦办、依法严惩，“请务必严肃看待我们守护台湾民主的意志与决心”。
  <br/>
  <br/>
  <strong>
   10万人黑衣上街 喊“光复香港、时代革命”
  </strong>
  <br/>
  <br/>
  这场在台北的游行，由在台港人组织香港边城青年及台湾多个NGO组织合办，响应全球二十国、四十多个城市“撑港、反极权”行动。除了台北大游行，台中、高雄、新竹、台南同步有集会行动。主办单位公布台北游行估计有十万人参加，远超出预期。
 </p>

  <br/>
  <br/>
  游行前台北下起大雨，雨势一度变小，在集会的济南路上出现一道彩虹，和现场民众高举的七彩连侬旗相互辉映。游行者此起彼落喊着“光复香港、时代革命”，并高唱着“荣光归香港”。有坐轮椅、有推娃娃车，有港人带美国国旗和青天白日满地红旗，说要谢谢美国和台湾，也有台湾人受访请香港人保重。
  <br/>
  <br/>
  曾在美国参与刺杀蒋经国总统的人权工作者黄文雄也在九二九游行之中，他接受自由亚洲电台采访表示，对香港的情势，一路在担心，中国本身有很多自己的矛盾，在解决他们自己的矛盾时，难免会牵连到香港，现在情形非常复杂，正是因为这样要站出来声援香港基本人权。

  <br/>
  另一名在台湾受到瞩目的是港独先锋、香港民族党前召集人陈浩天。他29号中午参加IR Taiwan举办的“独立自决，台港活路”人权餐会，与多个独派政党、团体呼吁台湾组成“台港联盟”对抗纳粹中国（Chinazi）。陈浩天在当天游行的晚会上说，他的心情非常沉重，

  “刚刚我一个兄弟跟我讲他中弹，然后我非常担心，我打给他，是什么子弹，他说是橡胶子弹，没关系，啊，又打过来了，然后就挂掉了，所以，心情非常沉重，也非常担心。今天很多人被捕，有些人被打到昏迷，会不会死掉也不知道。”
  <br/>
  <br/>
  陈浩天曾在台湾遇袭，此行他受访表示，从下飞机就有人一路跟随，进一步了解，应是台湾政府方面的“保护”或跟随。
