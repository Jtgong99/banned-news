<hr/><p>
 【大纪元2019年06月03日讯】（大纪元记者骆亚报导）今年是“
 <a href="http://www.epochtimes.com/gb/tag/%E5%85%AD%E5%9B%9B.html">
  六四
 </a>
 ”三十周年，世界各地的纪念活动层出不穷，目前
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%AD%E7%BE%8E%E8%B4%B8%E6%98%93%E6%88%98.html">
  中美贸易战
 </a>
 也全面升级，中共政权极度恐惧，开始在全国多地测试
 <a href="http://www.epochtimes.com/gb/tag/%E6%96%AD%E7%BD%91.html">
  断网
 </a>
 以应对未来的紧急状况。网民担心与外界隔绝的时代快来了。
</p>
<p class="p3">
 <span class="s1">
  据北京一位不方便署名的媒体人向大纪元透露，近日来广东、上海、重庆等地都相继出现
  <a href="http://www.epochtimes.com/gb/tag/%E6%96%AD%E7%BD%91.html">
   断网
  </a>
  ，并从中共的有关部门获得证实，这是“全国一键断网测试”。
 </span>
</p>
<p class="p3">
 <span class="s1">
  对方还介绍，“北京地区这几天很多手机和网络信号都非常差，以致有些媒体想做连线报导也不得不搁置。因信号实在太差，有外媒记者还抱怨连电子邮件也很难发出去。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  外媒记者还不得不借助东南亚国家的网络卡才能跟外面保持通畅一些。对方估计中共的贸易战白皮书发表之后，中共可能要做出新的动作了。
 </span>
</p>
<p class="p3">
 <span class="s1">
  据“金融界”消息，
 </span>
 <span class="s3">
  5
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  31
 </span>
 <span class="s1">
  日上午
 </span>
 <span class="s3">
  11
 </span>
 <span class="s1">
  点多广东电信出现大规模断网故障，显示网络连接正常，但是无法上网。包括汕头接龙湖、金平、濠江、潮阳等不同区域大面积用户报
 </span>
 <span class="s3">
  IP
 </span>
 <span class="s1">
  城域网电路没办法使用，初步定位
 </span>
 <span class="s3">
  DNS
 </span>
 <span class="s1">
  解析异常引起。
 </span>
</p>
<p class="p3">
 <span class="s1">
  腾讯转发《东江时报》的消息报导，
 </span>
 <span class="s3">
  31
 </span>
 <span class="s1">
  日早上，不少小伙伴一觉睡醒发现全惠州断网，实际是全省断网。
 </span>
</p>
<p class="p3">
 <span class="s1">
  另据“中国经济网”消息，
 </span>
 <span class="s3">
  29
 </span>
 <span class="s1">
  日上午
 </span>
 <span class="s3">
  11
 </span>
 <span class="s1">
  点左右上海移动崩了，不仅
 </span>
 <span class="s3">
  4G
 </span>
 <span class="s1">
  无法接收，连打电话发短信这些基本功能都失效，网民们陷入“手机恐慌”。
 </span>
</p>
<p class="p3">
 <span class="s1">
  而重庆电信断网更早。据百度百科资料，
 </span>
 <span class="s3">
  5
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  20
 </span>
 <span class="s1">
  日晚
 </span>
 <span class="s3">
  11
 </span>
 <span class="s1">
  点开始，多名重庆电信用户反映网络中断。微信朋友圈被“电信宽带无法连接”刷屏。网络中断持续半小时左右。
 </span>
</p>
<p class="p3">
 <span class="s1">
  独立学者“冷山”在其自媒体推特上总结说：“仅有防火墙还不够，中国各地开始演习大面积断网了，应对贸易战后吃草民众的集结！”
 </span>
</p>
