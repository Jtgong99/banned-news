 <p>
  中国欧盟商会周三警告说，在中国的外资企业对于即将推出的社会信用体系带来的严厉的制裁和监控没有足够重视，并且准备不足。
  <br/>
  <br/>
  根据这一新的企业排名系统，中国的国内外公司将被要求在其工作场所安装监控摄像头并与政府共享数据，他们还将根据企业税务记录和遵守一系列现行法律的情况进行评级。
 </p>
 <p>
  <br/>
  据法新社报道说，在北京的欧盟商会周三发表研究报告指出，违反规定的企业将被列入“黑名单”，并受到“立即和严厉的惩罚”。报告并补充说，制裁不仅限于处罚，还包括更频繁的检查、海关延误、没有补贴或退税以及公开羞辱。报告也提到，企业社会信用体系的一系列标准由中国政府制订，只有在中国政府眼中值得信赖的市场行为，企业才能获得较高评分。欧盟商会主席伍德克认为企业社会信用体系将决定公司的生死，而欧洲商界极度缺乏准备工作，令人深感担忧。
  <br/>
  <br/>
  德国柏林一家咨询公司负责人孔弼永受访时则表示，中国针对企业的社会信用体系不再只是一个构想，它已搜集了大量数据，对企业行为进行信用评级，并影响企业营运。孔弼永提到，中国的社会信用体系，其实还包括由政府对企业施加更大政治影响的评分标准，例如中国媒体对目标企业的报导。
  <br/>
  <br/>
  在中国的外国公司经常抱怨缺乏公平的竞争环境，特别是当地方官员对国内竞争对手的违规行为视而不见时。“社会信用”一词使外国公司感到紧张，因为它还与另一个庞大的中国体系有关，该体系旨在根据一系列政府定义的标准对个人进行评级。
  <br/>
  <br/>
  记者：陈爱珍 责编：申铧 网编：郭度
