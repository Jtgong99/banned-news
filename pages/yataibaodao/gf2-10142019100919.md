 <p>
  上海一名公民因为透过媒体公开声援香港反送中示威，被以“寻衅滋事罪”关押了整整一个月。甚至被贴上“反党反社会主义分子”的标签。
  <br/>
  <br/>
  香港爆发反送中运动以来，大陆当局一直严厉控制相关信息进入内地，但仍然有不少民众透过其他渠道获悉真相。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  以上海维权人士顾国平为例，除了透过社交媒体紧跟反送中运动的脉搏，还公开在网上发文支持示威者，两个月前更接受《大纪元时报》和“新唐人电视台”等外媒采访，因而触动了当局的神经。上月他在河北省北戴河逗留期间，被当地便衣扣押，带到看守所接受盘问。
  <br/>
  <br/>
  顾国平：“他们问什么时候采访你的？什么时候采访的？我说，这无可奉告，我忘了，年纪大。这视频里放的内容是不是你？我说，你们以这种方式指控我，录取我的所谓口供，你们休想得到违法侵权的目的。我今年给你们的最终结果，就是‘零口供’3个字。”
  <br/>
  <br/>
  在拘留期间，狱方多次出言恐吓，有狱警更骂他是“反党反社会主义分子”，又说案情敏感，他可能不能回家了。
 </p>
 <p>
 </p>
 <p>

 </p>
 <p>
  <br/>
  顾国平：“由于海外媒体率先采访报道了我这个事情，上海地方公安想利用这样一个机会来打压我，想置我于死地而后快这样一个目的，把一个无辜的、正常的、表达自己观点、看法的一个退休老人，关进了看守所。”
  <br/>
  <br/>
  在压力面前，顾国平仍然坚决支持香港示威者。
  <br/>
  <br/>
  顾国平：“譬如说‘双普选’，他们争取选举权没有任何违背宪法的地方。如果这个诉求正当合法我都支持，香港居然有上百万的人出来维权，难道这几百万人都错了？”
  <br/>
  <br/>
  在秦皇岛看守所羁押了1周后，警察把他送到上海长宁看守所，直到上周六才放人。
  <br/>
  <br/>
  2001年，上海当局接管和强拆顾国平一家的私有房产。顾国平指当局的行动违法，而且没有提供任何赔偿，他父母也因此而被迫在养老院终老。过去18年不断上访维权，使顾国平成为当局的眼中钉，曾多次遭受暴力对待和拘留。
  <br/>
