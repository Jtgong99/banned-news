<hr/><p>
 【大纪元2019年06月28日讯】（大纪元记者徐简报导）6月27日，美国白宫发布了总统
 <span href="http://www.epochtimes.com/gb/tag/%E5%B7%9D%E6%99%AE.html">
  川普
 </span>
 （特朗普）这次G20峰会之行的目标，拟从六个方面促进全球经济增长和繁荣。
</p>
<p>
 <span href="http://www.epochtimes.com/gb/tag/%E5%B7%9D%E6%99%AE.html">
  川普
 </span>
 总统说：“让我们共同通过我们的力量、资源和声音，不仅为我们自己，更是为我们的人民——减轻他们的负担，提升他们的希望，并赋予他们梦想。”
</p>
<h4>
 发展世界经济
</h4>
<p>
 在G20峰会上，将鼓励其它国家以美国为例，促进增长、促进就业和创新。
</p>
<p>
 • 美国经济复兴，这是其它国家如何实现增长的榜样。
 <br/>
 • 像美国的税制改革一样，促增长政策增加了数百万个新工作岗位，带回了制造业，并恢复了（人们）对美国经济的信心。
 <br/>
 • 当世界经济增长时，美国在繁荣 ，确保美国商品和服务业占据强大和稳定的市场。
 <br/>
 • 各国应重点实施有助于刺激经济的促增长政策，例如改革税法，精简法规和促进货币稳定。
 <br/>
 • 此外，各国可以通过培训高技术工人、让工人进行某些高需求行业的再培训，更好地释放该国全部经济潜力。
</p>
<h4>
 实现公平和互惠贸易
</h4>
<p>
 在G20峰会上，川普将致力于促进公平和互惠贸易。
</p>
<p>
 • 促进自由、公平和互惠的贸易协议，是美国增加就业和增长战略的关键组成部分。
 <br/>
 • 努力开辟美国商品和服务的新市场，同时确保任何交易都是可执行的，并为美国的工人和公司创造公平的竞争环境。
 <br/>
 • 美国正在与盟国和合作伙伴合作，以应对不公平的贸易行为，包括知识产权盗窃、不公平的使用劳工和强制技术转让。
 <br/>
 • 鼓励美国的盟友和伙伴，降低关税和非关税壁垒，进行自由、公平和互惠贸易的贸易。
 <br/>
 • 美国敦促对世界贸易组织（WTO）进行改革和现代化，同时确保世贸组织按原计划运作。
</p>
<h4>
 鼓励创新
</h4>
<p>
 另外，川普总统致力于鼓励数字市场的发展和支持新兴技术，以促进未来繁荣。
</p>
<p>
 • 新兴技术如人工智能，有可能对美国和世界的生活标准和生产力产生积极、有意义的影响。
 <br/>
 • 认识到这些技术的重要性，川普总统在今年早些时候签署了一项行政令，保持（美国）在人工智能领域的领导地位。
 <br/>
 • 过于繁琐的监管可能会扼杀创新，并限制这些变革性技术推动经济增长的潜力，为此美国致力于推动支持创新的政策，支持数字经济的增长，并为技术创新铺平道路。
 <br/>
 • 美国敦促20国集团成员共同努力，推进一个开放、公平、以市场为基础的数字经济，这些数据的自由流动使所有国家受益。
</p>
<h4>
 促进能源发展
</h4>
<p>
 第四点，川普总统将利用G20峰会推进他的“全面”能源发展方法。
</p>
<p>
 • 获得负担得起的可靠能源，是川普总统推动美国经济发展和促进全球繁荣的中心策略。
 <br/>
 • 在20国集团会议上，美国将寻求对所有能源在促进经济增长、能源安全和能源获取方面所发挥的作用的认可，同时支持健康的环境。
 <br/>
 • 美国已成为世界上最大的石油和天然气生产国，发展经济的同时保护环境。美国的能源和环境记录保持全球最好的记录之一。
 <br/>
 • 促进能源技术投资——如液化天然气（LNG），以及高质量的开发能源的基础设施。
 <br/>
 • 美国现在向五大洲的35个国家出口液化天然气，其中16个国家是在川普政府期间加入的。
</p>
<h4>
 在全球范围内赋予妇女权力
</h4>
<p>
 川普总统认为，赋予妇女权力是明智的经济和安全政策。
</p>
<p>
 • 赋予妇女参与世界经济的权力，对经济繁荣、国家安全和全球稳定至关重要。
 <br/>
 • 美国致力于消除法律和文化障碍，使得妇女能完全自由地参与当地经济。
 <br/>
 • 各国应认识到，只有妇女和男子都能够参与各级的经济和政治，才能实现稳定。
 <br/>
 • 通过妇女全球发展和繁荣（W-GDP）计划，来帮助引领这一趋势。
 <br/>
 • 川普总统在2019年2月启动了这一具有里程碑意义的倡议，旨在到2025年，为发展中国家的5000万妇女提供经济援助。
</p>
<h4>
 支持全球明智发展（Smart Development）
</h4>
<p>
 川普总统支持透明、负责性的和基于市场的金融发展原则。
</p>
<p>
 • 致力于促使私营部门在全球范围内，实现更可持续的经济发展。
 <br/>
 • 以私营部门为主导的可持续经济，替代国家领导的经济，后者伴随着许多束缚条件。
 <br/>
 • 美国仍然致力于经济合作与发展组织（OECD）的税收流程。
 <br/>
 • 美国谴责那些通过采取单方面行动对数字公司征税，破坏经合组织进程的做法。
 <br/>
 • 总统正在支持打击腐败、提高盟友和伙伴透明度的努力。
 <br/>
 • 各国应通过实施和执行强有力的反腐败法律，来平衡发展的竞争环境。
 <br/>
 • 美国支持20国集团努力促成透明、负责任和可持续性的基础设施发展融资，这种方式不会使负债累累的发展中国家陷入困境。#
</p>
