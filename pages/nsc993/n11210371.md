### 莲圆：桂枝香.曙光
------------------------

<p>
 【大纪元2019年04月24日讯】
</p>
<p>
 和平上访，
 <br/>
 为大法鸣冤，
 <br/>
 人间希望。
 <br/>
 白鹭翩翩四野，
 <br/>
 浩歌云荡。
 <br/>
 遥遥天际芸芸渺，
 <br/>
 骇惊涛，
 <br/>
 悲心难忘。
 <br/>
 理当回复，
 <br/>
 此心不负，
 <br/>
 万千同向。
 <br/>
 念念善音天路荡，
 <br/>
 众目锁京城，
 <br/>
 法子悲壮。
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E4%BA%8C%E5%8D%81%E5%B9%B4.html">
  二十年
 </a>
 悠悠去，
 <br/>
 <a href="http://www.epochtimes.com/gb/tag/%E4%B9%9D%E8%AF%84.html">
  九评
 </a>
 天剑。
 <br/>
 揭红魔画皮人醒，
 <br/>
 鬼慌慌、
 <br/>
 落红惨状。

 法今如是，

 旌旗处处，
