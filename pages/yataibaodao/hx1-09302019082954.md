 <p>
  香港歌手何韵诗在台北参加929撑港游行遭统派泼漆，台湾检警30号声押禁见胡姓、梁姓两主嫌，再逮捕7名中华统一促进党成员，共逮捕到案9人，检警朝组织暴力方向侦办。执政党民进党立委呼吁重判，勿因民主纵容暴力，丢台湾的脸。
  <br/>
  <br/>
  香港歌手何韵诗在台北参加撑香港游行，接受媒体联访时，遭伪装成支持反送中的胡姓蒙面歹徒，持预藏油漆罐，从何韵诗头上倒洒红油漆。何韵诗提告。受高度关注。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  台湾警政署长陈家钦30日召开记者会指出，经专案小组连夜侦查，已查扣相关中华统一促进党犯案工具，包括水枪、水球、厨余、旗帜、标语、大声公(手持扩音器)、名片、背心等证物，并将朝组织性犯罪侦办，务求一网打尽。
 </p>

 <p>
  <b>
   台警政署长：预谋性、计划性、组织性的犯罪
  </b>
  <br/>
  <br/>
  陈家钦说：“这个处所，就是本案暴力攻击行为的密谋处所，及策划的处所，胡嫌在这个地点，召开相关策划这次行动、暴力攻击；整个证据显示，本案是一个有预谋性、计划性，和组织性的犯罪行为。”
  <br/>
  <br/>
  陈家钦强调，台港大游行活动，经合法申请核准，依法受保障，遗憾特定团体以暴力手段对何姓歌手泼漆，妨害合法集会活动进行，对人身攻击造成危害，严重伤害台湾国际形象及民主法治。除了929现场逮获胡姓、梁姓两主嫌，检察官已声押禁见，并彻查幕后策划者。

  <br/>

  台北市长柯文哲30日受访被问到何韵诗遭泼红漆的行为表示：“本来就不对啊，废话，这还要问吗！”柯文哲说，要查清楚是组织性攻击行为，还是个人行为。
  <br/>
  <br/>
  国民党总统参选人韩国瑜竞选办公室表示，谴责暴力，支持执法单位严办到底，韩国瑜不乐见这样的暴力冲突发生，未来当选后，也不容许类似的脱序行为。

  <b>
   绿委：统派攻击香港民运非第一桩
  </b>
  <b>
   轻判造成纵容
  </b>
  <br/>
  <br/>
  王定宇说：“以往在机场，白狼的儿子攻击黄之锋，甚至于之前的跟监行动，这都是以组织犯罪的方式执行中国北京的命令，这是我们所不能允许的。我们的法律过轻，以往都只能处理行为人，公然侮辱罪也不过就拘役或役科罚金300元（约人民币67元），所以他们攻击不同意见的人，付出的代价很小，但是得到中国的奖赏很大，形成这些黑帮伪装成政党，不惜破坏台湾，丢尽台湾的脸。”

  王定宇说，难怪基层员警常感叹，捉到嫌犯送进地检署，一下就回派出所呛声。立法院2017年12月完成了“组织犯罪条例”的修法可重判，修法重点在“将难以查证的『牟利性』，从必要条件改成非必要条件”，“发起、主持、操纵或指挥犯罪组织者，处三年以上十年以下有期徒刑，得并科新台币一亿元（约人民币2300万）以下罚金；参与者，处六月以上五年以下有期徒刑，得并科新台币一千万元（约人民币230万）以下罚金。”
  <br/>
  <br/>
  滞台中国大陆异议人士龚与剑也分析，若对这类施暴红统嫌犯轻判，他们会伺无忌惮，台湾政府永远没办法解决这个问题。
  <br/>
  <br/>
  龚与剑说：“他们付出的代价可能是罚多少台币，但是他们可能在中共那边领到的那些奖赏、那些在中国大陆可以获得的荣誉，远远地可以抵过在台湾这样的罚锾的话，每一个中国人，不要小看中国人，中国人都是很现实的人，他们都很愿意做。因为在中国大陆，如果说共产党不好的话，他可能会被消失，在台湾的话，向何女士泼漆、向蔡英文总统扔鸡蛋的话，他们可能仅仅只是一个罚锾（处份）。”

  <b>
   何韵诗：泼红漆
  </b>
  <b>
   曝露暴权手上染的鲜血
  </b>
  <br/>
  <br/>
  事发后，何韵诗在脸书粉丝页，接连发了两篇文章“报平安”，何韵诗博文说：“被泼油漆，是麻烦了点，洗也洗不掉，皮肤被残留物刺到红红肿肿的，但假设这样能让更多人明白现在台湾和香港处境的严重性，但愿也是这样一个意外的公义彰显。”
  <br/>
  何韵诗还说：“泼红漆，卑劣又愚昧。想把我们染红，却只完完全全暴露了这个暴权手上染的鲜血。”
  <br/>
  <br/>
  谈到提告或许有人觉得不值，何韵诗说：“这就是他们如此嚣张狂妄的本钱吧。断定你们都不会提告，断定大家都怕麻烦，不想浪费时间，所以才会如此横行无忌。然而，一个人独善其身退下，将纵容更多类似的事件发生，更多人将会受害。”

  在何韵诗留言按赞超过10余万人次，大批台湾网友道歉说：“对不起！让誏你受到这样的污辱！”、“希望妳的提告让统促党消声匿迹！”有港人说：“鲜红油漆是妳的勋章”、“直接将个头染红”、“香港人的心态就是怕事，才有今天的果，绝不能再做同一错事阿！”、“梅姐在天之灵一定以你为傲！”、“感谢妳为香港人的付出，现在受到的屈辱，敌人日后将要万倍承受。”

  对何韵诗遭泼漆，何泳彤接受自由亚洲电台采访说：“不是怕，而是愤怒，如果害怕，早几个月，要怕都怕了，现在直接情感是愤怒，连我们到台湾也要这样骚扰我们。”

  <br/>
  <br/>
  何泳彤表示，在台港生现在担心人身安全，会结伴而行，不过，这种恐吓手法对港人无效：“香港人连警察这样追、催泪弹这样射都不怕的时候，不可能因为这样一点事就退下来。”

   统促党：胡男所为与统促党无关
