 <p>
  中国人权律师陈建刚的妹妹和14岁的外甥女16号在北京陈建刚的家中被警察带走，羁押了5个半小时后才获释回家。陈建刚因为代理中共中央政法委原书记周永康的儿媳黄婉的案子而遭到北京律师协会的官员威胁“被失踪”。他和家人最近在人权组织的帮助下，已经逃离中国，抵达美国华盛顿。陈建刚的妹妹为什么遭警方如此对待？本台记者申铧为此采访了正在华盛顿美利坚大学做访问学者的陈建刚。
  <br/>
  <br/>
  记者：首先请您谈一下您妹妹被抓走(的经过)，她等于算传唤，是吗？
  <br/>
  <br/>
  陈：因为我妹妹跟我说，到现在（警方还没有给她）任何法定的手续或者法定的理由，也没有向她出示任何身份证件。就是在外面砸门，把门锁损坏之后撬门进来，然后把我妹妹和我14岁的外甥女控制，把手机全部抢走，然后在我家进行搜查。至于什么法定的手续，什么法律的程序，是搜查还是传唤等等，还需要吗？现在中国已经不需要了。
  <br/>
  <br/>
  记者：是什么理由呢？为什么要把您妹妹抓过去呢？您妹妹有没有参与您的维权活动呢？
  <br/>
  <br/>
  陈：我妹妹什么都不知道。她住在山东，是一个普通的妈妈，有两个孩子。她对我任何事情都不了解。她到北京来是因为我到美国来留学，我家里的东西需要她收拾一下弄回家，因为北京租来的房子要退给人家了，仅此而已。中共的警察到我家来撬门进去告诉我妹妹说的是，你只要在你哥哥家里住，你就是违法的，因此要把你带走，就是这样。没有任何法定的手续，或者文件，什么都没有，直接就是抢劫。现在中国，在习近平的领导之下，官方特别是这些警察，像土匪一样，全部是穿着官衣的土匪。
 </p>
 <div>
 </div>
 <div>
  记者：你妹妹大概被羁押了五个半小时，晚上八点半被释放的，是吧？
  <br/>
  <br/>
  陈：是的。
  <br/>
  <br/>
  记者：那你有没有跟她通话，然后了解她在派出所被问了些什么事情呢？
  <br/>
  <br/>
  陈：这个我没有详细地了解，因为王宇律师，包龙军律师都在她身边。（警察）对她说的也就是你来干什么呀，我大概了解的是这些问题。把我的财产从我妹妹手里抢走之后，说让你哥哥来取。
  <br/>
  <br/>
  记者：据我了解，王宇律师和包龙军律师夫妇在您妹妹受到警察威胁时赶到了现场，他们做了些什么呢？
  <br/>
  <br/>
  陈：他们到的时候我妹妹刚被抓走十多分钟的样子。我妹妹在和王宇的通话过程中，门被撬开，然后电话被他们抢走。他们来了之后门锁被他们换掉了。中共的爪牙就是这样一帮土匪，他们没有任何手续入侵我个人的家，然后抢劫我的财产，没有任何法定手续。还把门锁换掉，现在我家里的门钥匙在他们控制之下。
 </div>

  记者：所以你家里人进不去你自己的家？
  <br/>
  <br/>
  陈：是这样的。因为我家里的锁被换掉了嘛。他们在五个多小时后把我妹妹释放的时候给了我妹妹一把就叫做安装的钥匙。现在的门锁通常情况下有一把安装的钥匙，等锁换完之后，会使用正常的钥匙，那个时候这把装修用的钥匙就作废了。所以现在我的家在被中共的警察控制，钥匙在他们手中。
  <br/>
  <br/>
  记者：那您现在妹妹和侄女的情况怎么样呢？
  <br/>
  <br/>
  陈：他们人是平安的。帮我收拾东西也收拾好了，北京时间16日的晚上会离开北京，我希望他们能够平安回家。
  <br/>
  <br/>
  记者：因为您离开中国不是通过正常途径出来的，那他们是不是为了这个进行报复呢？
  <br/>
  <br/>
  陈：有这个可能。当然了，我不到美国来留学，也会遭遇这种处境。其实每一个中国人几乎都面临这种危险。
  <br/>
  <br/>
  记者：您之所以出境是因为您代理前中共政法委书记周永康的儿媳黄婉的案子，受到威胁，是因为这个出来的。那么您现在到美国以后已经开始在美利坚大学的学业了，您能给我们介绍一下您的近况吗？
 </div>
 <div>
  <br/>
  <img class="image-inline captioned" src="resolveuid/004bce43-21c3-4c63-bc74-b04219f772d4"/>
 </div>
 <div>
  陈：我刚刚和学校对接上。因为我已经错过了四个月学习英语的机会，所以我的英语比较差。但是学校仍然愿意接受我，并且给我提供了很多的便利。到了美利坚大学的华盛顿法学院，正式开始法律的课程。学校愿意接受我，我对此非常感谢。我到法学院这边晚了几天，其他同学也是刚刚开始。我们这一次一共12位访问学者，分别来自不同的12个国家。前天我们大家见面做自我介绍，和学校里的工作人员见面，总之就是刚刚开始。
  <br/>
  <br/>
  记者：学习时间有多长？
  <br/>
  <br/>
  陈：还有至少十个月。
  <br/>
  <br/>
  记者：您希望在这次学习中学到什么东西呢？
  <br/>
  <br/>
  陈：作为一个律师，我感兴趣的当然是民主、法治、宪政，保障人权，权力的制衡，以及刑事诉讼，庭审，调查询问等刑事审判技术的问题。简单来说，在普世价值之下，正常社会法律是什么样子的，我想详细地，设身处地地亲自了解一下，就是这些内容。
  <br/>
  <br/>
