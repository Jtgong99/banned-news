 <p>
  在中国，医院百态可谓是无奇不有，医闹、红包、黄牛党都榜上有名。近日，微博上传出一名女子用豪车堵塞一所北京医院紧急通道的视频，引发网民对京城权贵符号的热议，而当事人也不得不对广大群众道歉。
  <br/>
  <br/>
  视频中，这位姓单的京城女子情绪激动，与到场执法的交警发生对峙。
  <br/>
  <br/>
  单女士还说，她不会把车挪走，自己也没有驾驶证。
  <br/>
  <br/>
 </p>
 <div>
 </div>
 <div>
  视频显示，一辆车牌号为“京A88519”的白色劳斯莱斯停在了北京妇产医院东院区的应急通道上。据国内媒体报道，这辆豪车的车型为“曜影”，内地售价高达五六百万元人民币。
  <br/>
  <br/>
  这段“豪堵”视频周三在微博上疯传，一些网友把注意力放在了这辆豪车的车主身上。有网民说，“京A8”开头的车牌是“京城权贵阶层的符号”，并举例说“京A80”开头的是国务院的车，“京A82”开头的是北京市委的车，而视频中的这辆以“京A88”开头的是钓鱼台、警卫局的车。
 </div>

  <br/>
  还有网友顺藤摸瓜，发现这辆豪车的车主名叫“和则文”，与北京中科宇航通科技发展有限公司的董事长同名。这家公司的注册资本为1亿元人民币，进行航天科技和航天器的研发。
  <br/>
  <br/>
  不管车主有怎样的家底，网民却炸开了锅。一位网友说：“这个社会真不是仇富，而是需要公平正义。”另一位网友直言：“中国太缺乏素质教育。”
  <br/>
  <br/>
  事后，当事人单女士在接受媒体采访时哭诉说，她家是个体户，这辆豪车是她的一个朋友的。由于她错过了周三的预约时间，就驾车走应急通道，却被工作人员拦下。她随后情绪失控，于是就发生了视频中的那一幕，她恳求大家原谅。
  <br/>
  <br/>
  周四晚间，朝阳警方发布消息，由于单女士严重扰乱医院公共秩序，依法对其处以行政拘留五天的处罚。
