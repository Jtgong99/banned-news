 <p>
  一名美国联邦快递公司的飞行员，因携带BB枪塑料子弹，在广州机场被中国公安部门拘留。有评论人士认为，中国这个行动应该和美中目前敏感的双边关系有关。
  <br/>
  <br/>
  联邦快递飞行员陶德·霍恩(Todd. Hohn)，9月12日在广州机场被拘捕。《纽约时报》报道说，霍恩被公安警察问话之后被保释，目前他住在广州的酒店。他被告知在调查结束前不得离开中国，并上交了护照和手机。
  <br/>
  <br/>
  中国外交部9月20日表示，广州白云机场海关以涉嫌走私武器弹药罪，对霍恩采取拘传、取保候审的强制措施，案件仍在侦办当中。
  <br/>
  <br/>
  《华尔街日报》的报道说，当局在霍恩托运的箱子中，发现了用于低压仿真气枪的非金属气枪弹。
 </p>
 <div>
 </div>
 <div>
  在纽约执业的律师高光俊表示，霍恩目前的状况相当于已被软禁。
  <br/>
  <br/>
  “实际是House arrested（软禁）,不管你把人关在酒店还是监狱都是逮捕，所以他现在就是被House arrested”。
  <br/>
  <br/>
  霍恩原为美国空军上校，几年前退役加入联邦快递公司。目前他在联邦快递亚洲分公司担任飞行员，家住香港。事发时他刚刚驾驶飞机由日本抵达广州，并准备乘坐商业班机回香港。他所携带的所谓非金属气枪子弹，在日本和联邦快递的边检中都顺利过关。
  <br/>
  <br/>
  低压方针气枪，即国外儿童玩具BB枪，这种玩具和使用的塑料子弹，被中国视为正规武器并全面禁止。
  <br/>
  <br/>
  根据百度的介绍，BB枪是一种枪械外型的仿真玩具。而BB弹则是通过弹簧压缩空气或预压缩气体可发射不具杀伤力的塑胶子弹。
  <br/>
  <br/>
  美国媒体认为，目前美中之间微妙的关系和香港紧张的局势，可能都和事件有关。
  <br/>
  <br/>
  高光俊认为，事件确实很可能和政治有关，而非普通刑事案件。
  <br/>
  <br/>
  “我相信这个是有政治上的因素。因为BB 枪的子弹这种事，就算是违规了，数量不多的话，通常把东西收了就完了。另外他也不是运入中国，而是过境，是要去香港的。这种事情以前连说都不会拿来说的。”

  事件发生之后，美国联邦快递公司发表声明说：“广州的中国当局在我们的一名飞行员登上一架商业班机前在其行李内发现某物件，随后将其拘留，之后将其保释。我们正在与有关当局合作，以更好地了解事实真相。”
  <br/>
  该公司对本台的询问电邮未作出回应。
  <br/>
  <br/>
  美国评论人士李洪宽认为，这可能是中国政府对付美国的一种方式。
  <br/>
  <br/>
  “这是中国经常玩的，就是成为要挟的人质，成为一个棋子。只要能让美国相关方面知道这是一个要挟就对了，这是它要达到的目的。”
  <br/>
  <br/>
  在美中最近一年多的争端中，联邦快递遭中方多次指责，包括华为产品运送差错，以及有人从美国向中国快递枪支等。不久前，中国官媒还在指责联邦快递向香港“运送刀具”，不过联邦快递表示，该刀具是联邦快递主动交给有关当局的。
  <br/>
  <br/>
  霍恩被捕，使得美中之间的你来我往更加复杂，而他美军退役军官的身份，也使事件受到广泛关注。
  <br/>
  <br/>
  自由亚洲电台记者石山华盛顿报道。 责编：申铧 网编：郭度
