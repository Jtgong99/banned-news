 <p>
  香港“反送中”运动在8·31抗议示威爆发警民冲突后，情势持续升温。据路透社报道，香港特首林郑月娥上周坦承，如果她能做选择她会辞职下台。香港观察人士认为，林郑月娥的去留掌握在北京手上，她只是北京的一个木偶。
  <br/>
  <br/>
  路透社报导，林郑月娥在上周一场与企业界人士的闭门会谈上坦言，香港局势在美中紧张持续升温的国际情势下，这场动乱已经上升为中国国家安全与主权问题的危机，她作为特首，解决问题的选择非常、非常有限。她还对她是否辞职首次表达了深深的无奈：
  <br/>
  <br/>
  林郑月娥：“作为行政长官，为香港带来了如此巨大的动荡，是不可原谅的。如果我能做出选择，我做的第一件事就是表示深刻的道歉并辞职。”

  <b>
   林郑只是北京的木偶
  </b>
  <br/>
  <br/>
  路透社取得这场会议的录音，并获得三名人士的证实。她的这番表白在中共建政十一庆典前夕传出，香港《开放杂志》主编金钟告诉自由亚洲电台，林郑月娥的去留，由不得她：
  <br/>
  <br/>
  金钟：“她完全是北京的一个玩偶，一个工具，她的进退是在北京的控制下，解决方法还是要北京当局、也就是习近平当局做出决策。”
  <br/>
  <br/>
  报导还说，林郑月娥在会中透露，北京当局“绝对没有”计划出动解放军到香港街头，因为代价太大。
  <br/>
  <br/>
  对林郑月娥的这段讲话，香港特首办公室一名发言人回复路透社说，她上周参加的两场会谈都是私人性质，特首办公室无法对会议中的谈话发表评论。这意味着特首办没有证实或否认报导。
  <br/>
  <br/>
  此外，路透社上星期还曾独家报导，香港政府在“反送中”运动开始之后，曾向北京提交一份报告，林郑月娥建议要“撤回”《逃犯条例》的修订，以解决香港当前争议，不过，却遭到北京反对，还下令她不得与抗议民众妥协。
  <br/>
  <br/>
  香港时事评论员谭自强告诉自由亚洲电台，林郑月娥有请辞念头，并不出奇，但北京的态度也很明白。是林郑的性格，决定了香港局势的演变。
  <br/>
  <br/>
  谭自强：“她（的性格）八个字讲完‘刻薄寡恩、刚愎自用’。而且我了解的最新消息，习近平的最新指示是说‘谁搞出来的事情，谁自己解决，不要巴望中央帮你’。这个很清楚，所以，现在整个压力压到林郑月娥身上。”
  <br/>
  <br/>
  谭自强还指出，接下来观察的重点是中秋节前、也就是9月13日香港局势能否稳定下来，否则，对北京来说，庆祝建政70周年的大庆，如果扮随着香港的抗议声浪，全世界都紧盯，而这不会是中国国家主席习近平想看到的。

  <b>
   上周末香港再爆警民冲突
  </b>
  <br/>
  <br/>
  另外，8月31日，香港再次爆发严重的警民冲突。香港地铁站用粤语、普通话及英文接连广播警告，8月31日当晚，地铁太子站内警铃大作，全是因为香港警方要展开搜捕行动，警棍齐飞、浓浓胡椒喷雾下，车厢内外传出阵阵哀嚎声。
  <br/>
  <br/>
  一幕幕香港警方与民众激烈对抗的画面，除了8·31血溅地铁太子站，街头上“为香港罪人祈祷”游行之外，汽油弹也在香港街头燃烧起来。
  <br/>
  最新消息还传出，香港众志副主席郑家朗2日晚间返家途中，遭三名男子突袭殴打双眼送医。
  <br/>
  <br/>
  <b>
   黄之锋和周永康撰文呼吁美国通过香港立法
  </b>
  <br/>
  <br/>
  香港雨伞运动的两大要角黄之锋和周永康8月31日当天联名投书美国纽约时报，文章标题就强调，《香港人民不会被中共吓倒》。
  <br/>
  <br/>
  在8·31游行前夕，香港警方先发动逮捕行动，包括黄之锋在内一共9人遭捕后获得假释。黄之锋和周永康在文章中也指出，逮捕了他们还会有更多人前仆后继。两人在文章一开头更引用电影《饥饿游戏：嘲笑鸟》的台词“要想烧死我们，那就同归于尽”，誓言香港人民不会默不出声。
  <br/>
  黄之锋和周永康也呼吁国际社会、尤其是美国，要对中国独裁者发出坚定讯号，两人希望美国国会的《香港人权与民主法草案》能够尽快通过，并收回香港享有的特殊经济待遇。
  <br/>
  <br/>
  即将于9月9日开议的美国国会两党议员已经对香港问题有一致表态。联邦众院外交委员会主席民主党的恩格尔（Eliot Engel）及共和党首席议员麦考尔（Michael McCaul）在8·31游行当天发表共同声明，敦促北京当局，在1989年天安门大屠杀事件满30周年的时刻，中国共产党有义务向自己的人民与世界展示，会选择一条和平发展的道路。
