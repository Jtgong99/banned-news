 <p>
  5年前中国全国人大常委会的“8.31决定”，为之后的"雨伞运动"埋下伏笔。为纪念这个日子，香港成千上万示威者在周六上街抗议，并与警方发生激烈冲突。在巴黎，游旅法港人也号召集会，支持香港的反对运动。
  <br/>
  <br/>
  香港， 自由  ! 香港， 自由 !
  <br/>
  <br/>
  这是8月31号，旅居法国的香港人在巴黎共和广场集会，手牵手组成“香港之路”的队形时所喊出的口号，以声援香港的民主运动。
 </p>

 <p>
  负责组织工作的旅法香港人解释了这场活动的原因，“原本8月31号在香港有一个和平游行，但因为香港警察提出反对而被迫取消，这是对香港游行集会自由一个很大的打击，同时有好几个民主会的议员立法会议员、区议员和政党领袖先后被捕，这是在製造白色恐怖，阻止民众上街抗议。所以，我们在巴黎的香港人，更要站出来声援香港的同路人，我们相信民主和免于恐惧的自由是世界各地人民，包括法国人民、波罗的海三国人民努力争取而来的普世权利。”他们说，这次的集会概念是来自30年前的“波罗的海之路”。1989年8月23号，波罗的海三国爱沙尼亚、拉脱维亚、立陶宛，约二百万人手牵着手，组成长达675公里的人链。这条被称为「波罗的海之路」的人链穿越三国首都，表达民众要脱离苏联、独立建国的诉求。七个月后，立陶宛于1990年3月11号率先宣布独立，其馀两地也先后脱离苏联建国。到了1991年底，大多数西方国家都已承认波罗的海三国的独立地位。
 </p>

  <br/>
  <br/>
  适逢“波罗的海之路”30週年，21万港人在维多利亚港两岸以及象徵港人不屈不挠精神的狮子山上，组成总长度达50公里的人链，以表示港人争取民主自由的坚定决心。因此，在巴黎，他们号召组成“香港之路”，支持争取民主自由的港人。
  <br/>
  <br/>
  组织集会的旅法港人分析了香港的哦最新状况：香港的示威者不仅要面对警察暴行，也害怕受“乡黑”袭击或遭他人报复。
  <br/>
  <br/>
  暴徒袭击示威者，警方却视而不见，造成不必要的严重人身伤害，令社会瀰漫一片白色恐怖气氛；多位国泰及港龙航空公司的机师和空勤人员，在社交媒体平台上表态支持抗争而惨遭解僱；港府暗示会对曾经参与抗争的公务员秋后算帐；世界各地的集会也遭亲中人士挑衅。
 </p>

  <br/>
  31号巴黎的集会，除了声援的法国人之外，还有来自新疆维吾尔、西藏、蒙古和台湾的旅法民众，共襄盛举。
  <br/>
  <br/>
  举着国旗的旅法台湾民众就说，“因为其实香港对我们来说是一个前车之鑑，所以香港发生的所有事情，我们都认为有可能会发生在台湾身上，所以对我们来说非常重要，一定要持续的关心。”
  <br/>
  <br/>
  过去两个月来，香港人坚决维护法治、争取实现「双普选」的勇气，引起国际注目，也受到越来越多来自全世界各地民众的支持。
