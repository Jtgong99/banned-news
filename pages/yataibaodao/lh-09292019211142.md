 <p dir="ltr">
  在中国即将在十月一号举办国庆70周年的大典前夕，数百名来自美国各地的关注中国前途的人士齐聚中国驻美国大使馆门前，发出与中国国庆主旋律不同的声音，呼吁关注香港、台湾、新疆、西藏等地的人权状况，反对中共独裁。
 </p>
 <p dir="ltr">
  9月29号，在中国驻美国首都华盛顿的大使馆门前，举行了“反对赤纳粹、声援自由香港集会”。据主办方说有四百人左右出席了这次集会。
 </p>
 <p>
 </p>
 <p dir="ltr">
 </p>

 </p>
 <p dir="ltr">
  <strong>
   代表面广 展现团结精神
  </strong>
 </p>
 <p dir="ltr">
  “今年能来四百人我感觉非常的好。”
 </p>
 <p dir="ltr">
  这是华盛顿的人权组织公民力量的创办人杨建利。公民力量是这次活动的组织者之一。
 </p>
 <p dir="ltr">
  “好在不仅仅是人数，代表的面也非常的广。有中国的人权人士来了，还有基督徒、法轮功（修炼者）、藏人、维族人、蒙古族人、香港人、台湾人、越南人、北韩人、古巴人等等。（他们）或者直接受共产党迫害，或者他们受到共产党支持的政权的迫害。”
 </p>
 <p dir="ltr">
 </p>

 <p dir="ltr">
  杨建利还说，这次活动最成功之处就是展现了各族裔团结的精神。
 </p>
 <p dir="ltr">
  活动现场有很多代表各种诉求的旗帜，有藏人的雪山狮子旗，有代表东土耳其斯坦（新疆）的蓝色旗，还有法轮功修炼者的“我们要人权”的巨大横幅。参与者中有很多人带着口罩，身穿印有“香港加油”字样的T恤衫。
 </p>
 <p dir="ltr">
  <strong>
   声援香港是主题之一
  </strong>
 </p>
 <p dir="ltr">
  活动刚一开始，参与者模仿“香港之路”，背对着大使馆的大门，手拉手筑城一道人链，以表达对香港反送中人士的支持。
 </p>

  <br/>
  香港抗议学生代表Issac Wong 在集会上发言，对大家出席这个活动声援香港表示感谢。他还说：
 </p>
 <p dir="ltr">
  “我呼吁大家继续支持香港，对抗中国共产党这个政权，希望香港成为中国共产党的第一场大战，然后把自由扩散到其他受压迫的地方。”
  <br/>
  <strong>
  </strong>
 </p>
 <p dir="ltr">
  <strong>
   蒙族代表发言声音哽咽
  </strong>
  <br/>
  <strong>
  </strong>
 </p>
 <p dir="ltr">
  蒙古族代表都兰发言时，声音一度哽咽：
 </p>
 <p dir="ltr">
  “我们蒙古族，还有新疆、西藏这些少数民族，在中共的压迫下，真的是无法生存下去。现在我们的草原，我们的文化，还有我们的家乡被破坏得什么都没有了。”
  <br/>
  都兰呼吁香港人和大家团结起来，反抗中共暴政。
 </p>

 <p dir="ltr">
  中国知名的异见人士陈光诚也到场表示对中共专制政权的不满：
 </p>
 <p dir="ltr">
  “所以今天我们各界、各族裔的朋友要携起手来，不能让中共这个邪恶专制政权继续祸害中国人民。中共不等于中国。中共是窃国贼。让我们一起努力！”
 </p>
 <p dir="ltr">
  <strong>
   美国高中生呼吁美国多关注中国人民
  </strong>
 </p>
 <p dir="ltr">
  在集会上发言的还有世界维吾尔大会副主席乌麦尔•卡纳特（
  <i>
   Omer Kanat
  </i>
  ），
  <i>
   国际声援西藏运动

  副主席布琼次仁（Bhuchung Tsering）,还有一位台湾太阳花运动的参与者。华盛顿地区的越南裔代表也有发言。颇为引人注目的还有一位只有17岁的华盛顿附近的高中生亚历山大·邓普顿（Alexander Templeton）：

 <p dir="ltr">
  “我们看到中国有集中营，关押了数百万的人。我们看到在香港，警察使用暴力和恐怖手段（对付抗议者）。有人还把香港人称为‘蟑螂’。还有中国内地十几亿中共压迫的受害者。我们必须结束这一切。美国应该少关心中国的钱，而应该多关心中国人！”
  <br/>
  <strong>
  </strong>
 </p>
 <p dir="ltr">
  <strong>
   众人踩踏“赤纳粹”旗
  </strong>
 </p>
 <p dir="ltr">
  活动最后，众人排队踩踏铺在地上的“赤纳粹”旗。这是一面红色的旗帜，上面有中共的党徽和黄色的五角星组成代表纳粹的卐字形状。这面旗帜首先在香港反送中运动中出现。
 </p>
 <p dir="ltr">
  这次活动除了公民力量之外，还有22个机构共同组织，其中包括美国兰托斯基金会、对华援助协会、自由西藏学生运动、共产主义受难者基金会、美国人维吾尔裔协会等等。
 </p>
