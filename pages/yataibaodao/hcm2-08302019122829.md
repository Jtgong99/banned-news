 <p>
  在台湾，鸿海集团创办人郭台铭阵营30日终于松口，如果郭台铭决定参选的话，会以“独立候选人”身份投入2020总统大选，不过，时间仍未定。另外，近期香港“反送中”运动，使得寻求连任的蔡英文总统的信任度急速上升。
  <br/>
  <br/>
  郭台铭风尘仆仆结束三天中东访问行程返台后召开了记者会。这是他自国民党初选败阵之后，第一次向媒体发表长篇讲话。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  郭台铭表示，这次应沙特阿拉伯高层官员邀请前往沙国访问。在三天两夜紧凑行程中，见到沙国七位部长与部长级官员。沙国正值产业转型，对台湾的电子、精密机械、半导体工业深感兴趣，希望能引进这些工业。
 </p>

  郭台铭：“国家经济政策，我一贯主张东进、西和、南拓、北接。我希望未来台湾能走出去，在经济发展过程中，新的市场能够涵盖到中东以及全球的回教市场。”
  <br/>
  <br/>
  郭台铭洋洋洒洒报告中东行的成果，就是没有松口谈参选与否的问题。随后，郭台铭阵营发言人，永龄基金会副执行长蔡沁瑜接受媒体提问。有媒体揣测郭台铭将在9月13日“中秋起义”。蔡沁瑜表示，9月17日(连署登记截止日)以前，慎重思考再做决定。
  <br/>
  <br/>
  蔡沁瑜：“如果今天郭台铭先生有决定参选的话，他会以独立参选来做参选人。时间点我们的确都还在研议当中，没有确切的情况，毕竟郭台铭先生才刚下飞机。”
  <br/>
  <br/>
  郭台铭可能以独立参选模式投入大选，这意味着他将会“脱党参选”，而郭台铭所属的国民党已经提名高雄市长韩国瑜。国民党主席吴敦义日前被问及时表示，如果有违反党纪的规定，当然依照规定处理。吴敦义强调，说不定还不是登记参选，可能只要连署，有违纪的动作可能就会触犯党的规定。
 </p>

  当国民党陷入分裂危机时，现任总统蔡英文的人气却随着香港“反送中”运动节节高升。根据《美丽岛电子报》30日公布的最新民调显示，48.2%的人信任蔡英文，与上个月相比大幅增加7.4个百分点，不信任度为44.0%。这是蔡英文两年来，首次出现信任度超过不信任度；而蔡英文的满意度也达到44.7%，比上个月跃增6个百分点。
  <br/>
  <br/>
  主持这项民调的戴立安接受本台访问时分析，今年一月以来，蔡英文两次在信任度跟满意度大幅弹升，超过七个百分点，分别是六月跟八月。正好碰上香港“反送中”与香港抗议持续升温，都让蔡英文的支持度大增。
  <br/>
  <br/>
  民调专家戴立安：“从资料上来看，民众对共产党的反感超过七成的时候，是个很强大的意见反应，这时候蔡英文的信任度跟满意度会反向大幅提升。”
  <br/>
  <br/>
  戴立安指出，从民调可以看出，20到29岁年轻族群，比较支持蔡英文的想法跟态度。在这次民调中，这些年轻人甚至认为，在历任总统里，蔡英文是最能保卫台湾的领导人。戴立安认为，这跟这个族群的历史记忆有关。
