 <p>
  近日，前知名媒体人张贾龙被维稳部门带走，随后以涉嫌寻衅滋事罪名被刑事拘留。
  <br/>
  <br/>
  据维权网报道，8月12日晚九点前后，贵阳公民、前知名媒体人张贾龙被辖区派出所和维稳部门从家中带走，随后被以涉嫌寻衅滋事罪刑事拘留，现关押在贵阳市南明区看守所。
 </p>
 <p>
 </p>
 <p>
  张贾龙出生于1988年，贵州省贵阳市人，前腾讯财经频道编辑，是活跃的推特用户和博客作者，曾报道过艾未未工作室被强拆的新闻。
  <br/>
  <br/>
  2011年4月，因在社交网络上发表一条“今天在出租车上听司机说五一期间三分之二的北京出租车司机将举行罢工”的推文，张贾龙被北京市公安局传唤24小时并抄家。随后因“在境外网站推特发布虚假信息，扰乱社会秩序”，被行政拘留10天。
  <br/>
  <br/>
  张贾龙这次被捕，原因尚不得而知。张贾龙的好友，北京异议人士胡佳在接受本台采访时说：“张贾龙已经相当一段时间没有发表太被人关注到的公众言论，据他妻子的了解就是没有。我个人和他是微信好友，他发微信朋友圈并不频繁，对于现在的许多热点问题没有什么特别重要的言论。”
  <br/>
  <br/>
  胡佳还表示，根据张贾龙妻子的说法，警方是根据他过去的言论，以寻衅滋事的罪名抓了他，扣押了他的电脑等物品，具体的案情尚不清楚。胡佳认为，鉴于最近的香港局势和即将到来的国庆70周年，当局也许加大了维稳力度。
  <br/>
  <br/>
  2014年2月15日，在腾讯财经工作的张贾龙受美国驻华大使馆邀请，作为中国青年博主的代表，与美国国务卿约翰·克里交流互联网自由等问题。张贾龙在会议上表示，希望美国能支持渴望自由的中国人，帮助他们一起推倒中国的网络防火长城。同时他也谴责部分美国公司协助中国政府阻止网民登陆境外网站。
 </p>

 <p>
  张贾龙还在会见时表示，很担心中国“良心犯”的处境，特别是人权活动人士许志永和作家刘晓波，前者于2014年1月被判四年有期徒刑，后者在被控“煽动颠覆国家政权”后遭到监禁，想要知道克里是否会探访刘晓波的妻子刘霞，并称刘霞健康状况不佳。
  <br/>
  <br/>
  2014年2月19日，张贾龙在《外交政策》杂志上发表文章《我想告诉克里的所有事》，呼吁美国拒绝给中国从事防火长城相关工作的人员发放签证，例如防火长城创始人方滨兴。
  <br/>
  <br/>
  2014年5月，腾讯公司以“泄露商业秘密等保密敏感信息行为”为由，与张贾龙解除劳动合同。
