 <p>
  自美国去年将中国定义为战略竞争对手后，美国对华政策出现大转弯。美国国防部印太安全事务助理部长薛瑞福（Randall Schriver）星期三重申，在美中两国利益出现分歧的情况下，美国将按规则与中国进行竞争。
  <br/>
  <br/>
  为了反制中国在国际范围内的影响力，美国总统特朗普去年签署了重新规划华盛顿在印太地区的长远战略方针的《亚洲再保证倡议法》。
  <br/>
  <br/>
  美国参议院外交关系委员下属的东亚、太平洋和国际网路安全政策委员会（Subcommittee on East Asia, The Pacific, and International Cybersecurity Policy）16日举行听证会，审议《亚洲再保证倡议法》实施情况以及美国的印度太平洋战略。
 </p>
 <div>
 </div>
 <div>
 </div>
 <div>
  美国参议院外交委员会亚太小组主席贾德纳（Cory Gardner）在开场发言中提到，中国正寻求在印太区域取代美国，并在全球范围内挑战美国的主导地位。
 </div>

 <div>
  “随着中国不断对外实现经济及军事扩张，美国今天面临的挑战是如何成功调整美国政策，以应对中国这个极权、‘近乎匹敌的竞争者’。我们不能让中国在毫无争议的情况下塑造未来。”
  <br/>
  <br/>
  美国国防部印太安全事务助理部长薛瑞福（Randall Schriver）出席听证会时说，《亚洲再保证倡议法》是保证美国可以维护自由、开放印太地区的重要政策。北京的言论和行为与美国的愿景相反。
  <br/>
  <br/>
  “我们曾耐心地希望中国共产党看到合作与参与的价值，但是他们在香港、新疆、南海及其他地方的行动使这一希望破灭。 我们之所以与中国竞争，是因为中国领导人认为中国在思想及能力上可以和美国竞争。”
  <br/>
  <br/>
  薛瑞福强调，竞争不等于冲突，美国和中国将在双方利益重叠时合作；在双方出现利益相悖时进行激烈竞争，美国同时会持续关注、发声谴责中国违背国际准则的行为。

 <div>
  据了解，《亚洲再保证倡议法》去年底在国会通过，正值美国的《台湾关系法》生效40周年。《亚洲再保证倡议法》倾注了相当可观的篇幅阐明美国对中国和台湾的战略目标，其中包括承诺美国会持续定期对台进行军售，并鼓励美国高官根据《台湾旅行法》的规定访问台湾。
  <br/>
  <br/>
  美国国务院亚太助理国务卿史迪威(David Stilwell)星期二作证时表示，美国关注中国政府以经济手段压制台湾，挤压其国际空间、挖走其邦交国等行为。
  <br/>
  <br/>
  “台湾海峡的和平与稳定符合美国利益。几十年来，美国一直支持台湾根据《台湾关系法》维持足够的自卫能力。与此同时，《亚洲再保证倡议法》也重申，美国将继续支持台湾保持有效的威慑能力，以应对现有的或可能在未来出现的威胁。”
  <br/>
  <br/>
  除了军事与战略方面，《亚洲再保证倡议法》也涉及人权、民生、文化等方面。
  <br/>
  <br/>
  史迪威说，由于《亚洲再保证倡议法》提到“在印太地区促进人权和尊重民主价值观与美国国家安全利益相符，美国国务院正致力于推广相关事项。
  <br/>
  <br/>
  “我们正向国际社会施压，要求中国停止对维吾尔人以及其他穆斯林少数族裔的镇压；我们坚持认为中国不应干预藏传佛教转世事务。”
  <br/>
  <br/>
  除此之外，美国国际开发总署（USAID）亚洲局代理局长斯蒂尔(Gloria Steele)也在听证会上介绍了该局针对西藏的援助计划。
  <br/>
  <br/>
  她说，美国国际开发总署着重于改善藏人社区的生活，加强当地机构的活力与凝聚力，同时致力于保护他们的文化和传统。
  <br/>
  <br/>
  （记者：韩洁 责编：申铧 网编：郭度）
