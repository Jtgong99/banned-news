### 老人遭冤判 国保抢钱 近六万现金不归还
------------------------

<p>
 【大纪元2019年04月24日讯】 黑龙江省延寿县法轮功学员
 <a href="http://www.epochtimes.com/gb/tag/%E7%BD%97%E5%BD%A9%E6%A3%AE.html">
  罗彩森
 </a>
 ，被非法关押在哈尔滨市看守所，至今8个多月。日前老人接到判决书，被
 <a href="http://www.epochtimes.com/gb/tag/%E5%86%A4%E5%88%A4.html">
  冤判
 </a>
 一年六个月，而且此前家中被警察掳走的近六万现金也得不到归还。
</p>
<p class="p4">
 <span class="s1">
  明慧网报导，
  <a href="http://www.epochtimes.com/gb/tag/%E7%BD%97%E5%BD%A9%E6%A3%AE.html">
   罗彩森
  </a>
  女士有三个儿子，都已成家立业娶妻生子。三年前丈夫去世后，罗彩森一直和小儿子生活在一起。家里生意忙时，勤劳的她帮儿子们带孩子、洗衣、做饭，偶尔也帮孩子们打理一些生意。儿孙绕膝，在享受着天伦之乐的同时，一家人生活得其乐融融。
 </span>
</p>
<p class="p4">
 <span class="s1">
  2018年8月，罗彩森在自己家中被邻县尚志县的国保李志国、高剑平等人跨地区无辜绑架并被抄家，家中的五万八千元现金也被一同掳走。
 </span>
</p>
<p class="p4">
 <span class="s1">
  家属多次去尚志县国保找到高剑平、李志国，想要回属于自己的钱，可是高剑平等人推诿搪塞，拒绝还钱，并且告诉家属，这钱得等到开完庭才能退给家属。
 </span>
</p>
<p class="p4">
 <span class="s1">
  但是，罗彩森的案子10月7日到阿城检察院时，这笔钱没有随案移交，等11月7日案子到了阿城法院时，这五万八千元也没有随案移交，到了12月7日阿城法院开庭时，法官当着律师和家属的面说：这笔钱没有随案移交，也和本案无关。
 </span>
</p>
<p class="p4">
 <span class="s1">
  庭审结束后，罗彩森的家属想既然已经开庭了，这钱与本案无关，就去找尚志国保取回这笔钱，并且家里做生意，进货急需这笔钱。然而高剑平却又说这钱得等到开庭后判决结果出来后才能还给家属。
 </span>
</p>
<p class="p4">
 <span class="s1">
  但是，现在判决结果已经出来，尚志国保还是不归还家属这五万八千元。
 </span>
</p>
<p class="p5">
 <span class="s1">
  中共江泽民集团自1999年7月迫害法轮功以来，公检法司机构抢劫法轮功学员钱财的案例举不胜举。
 </span>
</p>
<p class="p1">
 <span class="s1">
  法轮功学员李家芳，曾进京为法轮功鸣冤，被非法劳教3年。2002年，她失去了辽河油田的优越工作，靠自己的聪明才智，一直在高新企业做工程预算工作。
 </span>
</p>
<p class="p1">
 <span class="s1">
  2018年5月23日，山东临沂市
  <a href="http://www.epochtimes.com/gb/tag/%E5%85%AC%E5%AE%89%E5%9B%BD%E4%BF%9D.html">
   公安国保
  </a>
  将李家芳的五张银行卡抢走，一直不归还，还想用莫须有的罪名黑掉它们。
 </span>
</p>
<p class="p2">
 <span class="s1">
  法轮功学员公丕启，原山东省预备役高炮师副参谋长、军队退休上校。
 </span>
</p>
<p class="p1">
 <span class="s1">
  2017年10月17日，青岛市北区公安局警察在没任何合法手续的情况下到公丕启家非法抄家，从他身上及家中抢走现金12万多。此后，判决书中只出现了7万多，其余现金不知去向。
 </span>
</p>
<p class="p1">
 <span class="s1">
  法轮功学员田彩英，2016年4月22日，辽宁抚顺市永安台派出所警察将她家中欲交房贷的八九万多元现金以及内有几万元的银行卡和存折一同抄走。
 </span>
</p>
<p class="p1">
 <span class="s1">
  田彩英家人查看房子时，发现家中一片狼藉，如同遭到土匪抢劫一般。
 </span>
</p>
<p class="p5">
 <span class="s1">
  据明慧网不完全统计，仅2018年上半年，中共警察对129名法轮功学员抢劫、勒索现金达930,302元人民币。
 </span>
</p>
<p class="p5">
 <span class="s1">
  2018年7月份，法轮功学员被敲诈勒索的现金总额达238,690元，其中警察抄家勒索87,690元。8月份，法轮功学员被勒索的现金总额为252,550元，其中警察抢劫勒索125,550元。
 </span>
</p>
<p class="p6">
 <span class="s1">
  中共《刑法》第325条明确规定：“意图为自己或第三人不法之所有，而抢夺他人之动产者，处六月以上五年以下有期徒刑。
 </span>
 <span class="s4">
  ”
 </span>
 <span class="s4">
  《刑法》第328条规定，
 </span>
 <span class="s1">
  以强暴、胁迫或他法，而取他人之物或使其交付者，为强盗罪
 </span>
 <span class="s5">
  ，
 </span>
 <span class="s1">
  处五年以上有期徒刑，
 </span>
 但中共一直视法律为废纸。#
</p>
