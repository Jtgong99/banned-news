 <p>
  香港反送中运动持续将近四个月，对当地经济造成一定影响。香港首富李嘉诚周五宣布，他将捐出10亿港元，帮助香港中小企业度过难关。前段时间，他被指对“反送中”运动的态度过于暧昧，遭到中国媒体炮轰。那么他这个时候慷慨解囊又意图何在呢？
  <br/>
  <br/>
  李嘉诚基金会周五在官网上宣布，将会捐资10亿港元，配合特区政府的经济支持措施，支持香港中小企业度过难关。声明指出，具体细节仍待与政府商议。外界尚不清楚捐款的分配方式。
  <br/>
  <br/>
  基金会主席李嘉诚表示，这笔捐款配合港府新宣布的20亿中小企备用资金，希望这10亿元能起带头作用。
 </p>
 <div>
 </div>
 <div>
  李嘉诚的慷慨解囊让人们把目光再次投向了香港政治局势对当地经济造成的影响，而有中国官媒表示，当地近期的示威活动折射出的深层次社会问题之一，就是年轻人常抱怨的住房难。
  <br/>
  <br/>
  上个月，香港最大的政治组织民主建港协进联盟建议港府以作“公共用途”为由，引用《收回土地条例》，大量兴建公营房屋。没过多久，香港四大家族中的郑氏家族和李兆基家族就宣布会通过捐地的方式，纾解当地公屋短缺的问题，而郭氏家族也表示愿意配合政府的收地计划。
 </div>

 <div>
  拥有农地面积最少的李嘉诚家族旗下的长实集团则回应说，农地用作住房可能需要较长的时间才能受惠于人，集团会对此作出研究。
  <br/>
  <br/>
  中国金融学者贺江兵认为，比起捐地来说，李嘉诚资助当地企业更容易缓解实质问题。
  <br/>
  <br/>
  “我觉得李嘉诚捐资和捐地是有区别的。此举能够更直接地帮助中小企业，而这些企业可以让大多数人就业，能够从根本上解决社会稳定的问题。”
  <br/>
  <br/>
  随着“反送中”运动演变成自香港回归以来最大的政治危机，当地商界人士大多紧跟中央宣传步调，严厉谴责示威者的暴力行为，并支持警方严正执法。作为香港首富，李嘉诚的态度耐人寻味。
  <br/>
  <br/>
  8月，他以“一个香港市民”的名义在多家报纸刊登头版广告，在反对暴力的同时，呼吁自由、法治和包容。他还在上个月到访一家寺庙时，恳请当局对香港年轻人网开一面、大事化小。此言一出，李嘉诚立即遭到中国官媒炮轰，被指“纵容犯罪”、为了私利炒地产。
  <br/>
  <br/>
  因此，有舆论认为，李嘉诚在这个时间点捐钱是为了讨好北京当局。但美国乔治•华盛顿大学比较政治学博士生李源分析说，李嘉诚近年来从大陆大举撤资，他不再那么需要看中共的眼色行事。
  <br/>
  <br/>
  “我认为这对他在香港当地的形象倒是非常重要的一步。此举不光是在经济利益上支持香港近期受影响的中小企业，这对他的企业和个人形象来说，这笔钱花的也是挺值的。”
  <br/>
  <br/>
  受持续数月的“反送中”运动影响，香港经济面临前所未有的挑战，而旅游业和零售业受创尤为严重。香港政府周三公布的数据显示，当地8月的零售销售创下了有记录以来的单月最大跌幅，同比下跌23%。同月，访港旅客人数较去年同期减少了四成。
  <br/>
  <br/>
  港府8月表示，香港社会动荡、美中贸易摩擦等多种因素使得当地经济的下行压力加剧，政府据此把今年香港GDP增速的预测从5月的2-3%下调到0-1%。香港研究协会近期对千余名香港市民进行了调查，超过七成受访者对当地未来的经济走势感到悲观。
  <br/>
  美国加州企业家袁建斌表示，为了提振低迷的经济形势，港府当前的首要任务仍然是稳定社会情绪。
  <br/>
  <br/>
  “我认为，当局应当最先解决的就是与市民的沟通问题。香港的稳定其实在很大程度上来说，在于港府的一些具体措施，包括有一个良好的沟通渠道，并且一定要体现出（沟通的）广泛性。”
  <br/>
  <br/>
  港府周五宣布，将会引用“紧急法”制定《禁止蒙面规例》，从本周六开始禁止民众在参与公众活动时使用蒙面物品。本台报道，受此影响，当天全港各区都有商场和商铺提早关门，气氛“俨如戒严”。
  <br/>
  <br/>
  自由亚洲电台记者家傲华盛顿报道。 责编：申铧 网编：郭度
