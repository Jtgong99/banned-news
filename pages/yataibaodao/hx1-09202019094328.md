 <p>
  本周一台湾才被所罗门断交，周五再被基里巴斯断交。这样，蔡英文任内已失7个友邦，台湾只剩15友邦。蔡英文谴责基里巴斯选择做北京的棋子是项错误，并直指北京密集攻击台湾为扭转台湾明年大选战局。北京则对基里巴斯表示“高度赞赏”。同一时间美国军舰通过台海。
  <br/>
  <br/>
  “基里巴斯共和国政府自本日（20）正式通知我方，与我国断绝外交关系。对吉国政府不顾多年来中华民国（台湾）政府的多方协助与情谊，我方至感遗憾并予强力谴责。政府也宣布自即日起终止跟基里巴斯的外交关系。”
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  <iframe frameborder="0" height="620" scrolling="no" src="https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRFAChinese%2Fvideos%2F417959098853237%2F&amp;show_text=0&amp;width=622" width="622">

 <p>
  台湾外交部长吴钊燮5天内宣布“中华民国”台湾被第二个友邦断交。回顾5天前，所罗门和台湾断交时，吴钊燮被问到是否在南太平洋5个友邦掀起断交的“骨牌效应”时，自信满满表示“邦交都稳固”，直到基里巴斯和台湾断交，他才吐露实情。
 </p>
 <p>
  <b>
   台外长：所罗门和台湾断交
  </b>
  <b>
  </b>
  <b>
   基里巴斯总统决定
  </b>
  <b>
   “
  </b>
  <b>
   跟着走
  </b>
  <b>
   ”
  </b>
  <br/>
  <br/>
  吴钊燮说：“所罗门群岛（和台湾断交）的事情发生以后，基里巴斯的总统就决定要跟着一起走，所以这几天事情发展的非常快速。”
  <br/>
  <br/>
  吴钊燮说，2016年基里巴斯总统马茂上台后，就开始和中国方面有所往来。涉外人士指出，基里巴斯要求台湾提供3600万美元，作为购买民航机赠款遭拒，是断交关键。另有媒体报导，基里巴斯开口要价7000万美元的737客机，陆方原提出约3000万美元的国产C919进行杀价。
  <br/>
  <br/>
  吴钊燮在记者会上表示无法证实金额。尽管有外媒报导吐瓦鲁亲台总理下台对台不利，不过吴钊燮对台湾仅存的瑙鲁、吐瓦鲁、帕劳、马绍尔4个南太友邦仍打包票说没有问题。
 </p>
 <p>
 </p>

  <br/>
  外传中国领导人习近平因应10月召开的中共四中全会，主导预算无上限的“台湾介选计划”，在外交、军事与观光等各层面全面打压台湾，以缓解他在政治上遭党内问责的沉重压力，强挖台湾南太邦交国即为此计划一环。
 </p>
 <p>
  对于这一传言，吴钊燮说：“习近平在针对香港的议题、针对台湾的议题的时候，媒体都有报导，他说，该斗争就斗争，我相信这个就是其中的一部份，但有关细节我并没有掌握。”
 </p>
 <p>
  记者会上被问到“零邦交”的可能和因应？吴钊燮说：“我想任何的外交部长不会坐在这边跟你讨论这个问题。”
  <br/>
  <br/>
  外交部过去在断交记者会后会从大厅撤旗，不过基里巴斯与台湾断交事发突然，断交记者会后，基里巴斯的国旗还竖立在外交部大厅，来不及撤。
  <br/>
  <br/>
  <b>
   蔡英文：北京压迫台湾只能选择会向他低头的总统
  </b>
  <br/>
  <br/>
  人在桃园的总统蔡英文临时在当地发表谈话说：“对于基里巴斯政府所作的这个决定，我们觉得非常地遗憾。他们选择放弃一个真诚朋友，而选择去做中国的棋子。我们觉得是一个很大的错误。现在距离总统大选只剩下一百多天，这也是中国在这个时刻密集打击我们的原因。”
 </p>
 <p>
 </p>
 <p>

  <br/>
  蔡英文提到，针对国安单位掌握的情资，在明年一月台湾总统大选之前，中国对台湾持续会有很多打压和胁迫作为，动作只会更多，不会更少，目的就是想扭转总统大选的战局。
 </p>
 <p>
  蔡英文说：“他们试图用打压的行动告诉台湾人，我们不能买战机，我们不能撑香港，我们只能选择一个会向中国低头的总统。因此我要在这里代表中华民国台湾，坚定地告诉中国，对一国两制，我们的答案只有三个字，就是‘不可能’。”
 </p>
 <p>
  蔡英文强调，中国大陆这些恶霸式的压迫，不会让台湾人放弃坚持。如果台湾人选择放弃坚持，世世代代台湾人将不会有主权、不会有自由、不会有民主，将会一无所有。
  <br/>
  <br/>
  对5天内被两国断交，吴钊燮再度表态会辞职负起政治责任，但蔡英文定调，政府全程都掌握，外交团队也尽最大努力。这引发一片骂声。
  <br/>
  <br/>

  <br/>
  前中华民国驻新西兰大使介文汲接受自由亚洲电台采访指出，对“雪崩式断交”，蔡政府应虚心检讨提出务实对策，不能只是一再怪罪友邦、指责北京，还要在野党跟她一起骂中共，以为自己捡到枪，应深切检讨外交体系政治任命指挥专业外交官的问题。外交部长必须换人，应聆听专业意见。
 </p>

  介文汲说：“不能再这样下去，外交要整个崩盘了！最重要的是，不仅要检讨缺失，拟定有效方法，对现存邦交国尽快加强（关系）。所罗门悬缺大使3个月、基里巴斯目前没大使在大使馆。在关键时刻，外交人事安排都跟不上，别的更不要说。”
  <br/>
  <br/>
  中国文化大学政治学系讲座教授陈一新接受自由亚洲电台采访提到，当年陈水扁总统丢掉9个友邦，但抢回来3个友邦，蔡英文有没有能力拓展外交？
  <br/>
  <br/>
  陈一新说：“美国他想帮忙，不是无用，就是说中共出的资本大，政治现实、经济现实的问题。蔡英文、吴钊燮应该道歉，处理外交无力，一周断2个。陈水扁政府断9个还抢回另外3个邦交国，不至于全垮。陈水扁那时候还有点能力去跟中共抢外交，现在这个蔡英文政府根本抢也抢不过，又不承认‘九二共识’，谁理她？”
  <br/>
  <br/>
  国民党立委曾铭宗担心在中共十一国庆前夕，更多国家会跟台湾断交，呼吁蔡英文动员所有国安系统、外交部系统，全力巩固邦交。

  曾铭宗也呼吁中国大陆没有必要再挖“中华民国”外交墙角，这无助两岸良性发展。曾铭宗认为，“零邦交”的情况应该还不至于，但台湾真的可能在国际上被孤立。
 </p>
 <p>
  <b>
   中方高度赞赏 美舰穿越台海
  </b>
  <br/>
  <br/>
  中国大陆外交部发言人耿爽20号在例行记者会上则表示，中方对基里巴斯政府坚持承认“一个中国”原则，同台湾当局断绝所谓“外交关系”，并将同中国恢复外交关系的做法，表示高度赞赏，支持基里巴斯作为主权独立国家作出的这一重要决定。
  <br/>
  <br/>
  中国大陆一周内夺台湾2邦交国。台湾的国防部20号表示：“美军一艘作战舰今天由北向南航经台湾海峡自由航行，这也是美军舰今年以来第8度通过台湾海峡。国军运用联合情监侦作为，全程掌握周边海、空域舰、机相关动态，期间并无异状。”
  <br/>
  <br/>
  台湾中央通讯社报导，军方私下证实，该舰是“安提坦号（CG-54）”巡洋舰，同一型号战舰7月下旬曾航经台湾海峡。
