 <p>
  武汉新冠疫情的广泛蔓延在中国社会造成恐慌，使得普通民众风声鹤唳。不少地方出现了社会控制的土办法，对民众的人身自由进行严格控制，引起了广泛的不安。有网友表示，这让人又想起了文革。
  <br/>
  <br/>
  由于疫情的影响，中国人熟悉的搓麻将的声音现在很多地方已经听不到了。为了防止疫情传染，中国很多地方都禁止民众以打麻将的方式聚集。但防控疫情的社会控制手段远远不止禁止麻将这么简单。小区封闭、乃至封城，已经成了从湖北武汉向外扩散的普遍控制手段。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  <b>
   控制手段似文革再现
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>
  <div class="image-inline captioned" style="width:1200px;">
   <div style="width:1200px;">
    <img alt="一些民众不顾武汉肺炎疫情，依然聚集打麻将。（Public Domain）" src="https://www.rfa.org/mandarin/yataibaodao/huanjing/wy-02172020125644.html/1580234472_24435.jpg" title="一些民众不顾武汉肺炎疫情，依然聚集打麻将。（Public Domain）"/>

 </p>
 <p>
  身居美国纽约的前中国医生何岸全从医疗专业的角度赞同这种封闭方式：“湖北省或武汉市政府在疫情初期，没有及时管控，从医学理论上说，可以假设每个武汉人都是疑似感染者，所以，这些手段（封城、封小区）都是可以做的。”
  <br/>
  <br/>
  但他指出，目前封闭小区、防控疫情的具体做法还不够人道，“如果没有足够的医疗、后勤和医疗服务人员，封闭小区就是违反人道的。”
  <br/>
  <br/>
  由于后勤不足，不少城市的封闭小区出现了基本生活物资采买不足的现象，民众充满焦虑的心理。医疗条件不足更导致不少感染病例无法得到及时救治。
  <br/>
  <br/>
  与此同时，很多地方的封闭手法过激也在民众中引起了普遍的不满。身居辽宁某小区的一位不愿具名的先生告诉本台记者，现在小区物业防控疫情的具体做法有点类似文革时期，“他们全部是戴红袖章，你稍微有些不满，他们就把警察叫来，或者把保安叫来，就把这个人带走了。”
  <br/>
  <br/>
  这位先生说，非常时期，普通民众没法对这些过度的做法提出反对，不戴口罩都会被抓起来，甚至挨打，“不光是警察打你，你走在街上，如果不戴口罩，别人都要打你。”
  <br/>
  <br/>
  这位先生的说法印证了社交媒体上流传的部分视频和说法。为了防控疫情，不少地方出现了大量不顾法律底线的做法。中国官媒《人民日报》推特账户上公布了一段视频，是湖北孝感一户人家，因为在自己家打麻将，被当地防疫人员砸毁麻将桌，并被连扇三个耳光的情形。
  <br/>
  <br/>
  除此以外，有的地方在受到感染的居民窗户外安装铁栏， 任意封堵居民楼外出通道，在交通要道私设关卡，禁止通行，导致刚好外出的居民无法回家等等。
  <br/>
  <br/>
  中国官媒《人民日报》周一发表评论文章指出，那些简单粗暴、矫枉过正、用力过猛的行为，看似坚决有力，实则脱离疫情防控的实际，有的还涉嫌违法违规，副作用不容忽视。
  <br/>
  <br/>
  身在纽约的《中国战略分析杂志社》社长李伟东认为，这篇社评来得太晚了：“我看到《人民日报》那篇社评说，这样强制控制有违法治，但这些情况已经发生一个多月了。专制政府的问题就是，事情发生很久了，他们才反应过来，但普遍的人道灾难已经造成了。”
  <br/>
  <br/>
  <b>
   极权体制下的警察社会
   <br/>
  </b>
  <br/>
  美国《纽约时报》在近日的报道中指出，中国正在用毛式社会控制运动对抗疫情，很多防疫一线的工作交由了加强版的居民联防来进行。
  <br/>
  <br/>
  李伟东认为，出现这种情况主要的症结还是在习近平上台之后推行的极权体制，“（在这种体制下）全能主义，或者说大政府小社会的情况得到了极大的加强，基本上变成了一个警察社会，这次疫情更有助于他朝这个方向努力。而且疫情之后，这些严控措施也不可能短时间内消除。”
  <br/>
  <br/>
  与此同时，不少民众通过社交媒体展现出对某些过度严控措施的赞同。李伟东认为，疫情之下，民众可能更加顺从政府的强力管制，造成他们的依赖心理，这对于国家的民主化和人权保障是不利的。
  <br/>
  <br/>
  (记者：王允 编辑：申铧） 网编：郭度
