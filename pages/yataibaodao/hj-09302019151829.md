 <p>
  香港的“反送中”抗争正进行得如火如荼。10月1日中国国庆当天，香港可能会再次爆发大规模抗议。在香港民意汹涌以及国际谴责声浪此起彼伏的情况下，中国国家主席习近平的态度似乎有所软化。
  <br/>
  <br/>
  中国国家主席习近平星期一晚上出席新中国成立70周年招待时，再次重申了中央的对港政策。
  <br/>
  <br/>
  他说，“要继续全面准确贯彻‘一国两制’、‘港人治港’、‘澳人治澳’、高度自治的方针，严格按照宪法和基本法办事”，在“祖国的全力支持，广大爱国爱港爱澳同胞的共同努力”下，“香港、澳门一定能与祖国内地同发展共进步。”
 </p>
 <p>
  习近平这番向香港承诺“一国两制”的讲话被外界普遍认为是在放软姿态。前香港政府中央政策组顾问刘细良告诉记者，习近平想要“安抚”的对象并不是香港示威者。
  <br/>
  <br/>
  “我觉得大家不要对习近平此举抱有太大的幻想，这只是在宣示一个既有的立场来讨好国际媒体，同时也是针对美国提出的《香港人权和民主法案》。只要‘一国两制’消失，美国对香港的特殊政策也会消失。法案提到的独立关税地位、出入口管制、文化交流、签证等，才是中国政府最关心的问题。”
  <br/>
  <br/>
  纽约城市大学政治学教授夏明也认为，习近平希望缓解中国国内权贵的担忧。
  <br/>
  <br/>
  “因为这些人会以香港作为风向标。香港走多远、其自由的丧失程度有多深等问题，会直接影响到大陆的政治、经济、商业、媒体等各方面的生态。尤其是中国共产党的高官的财富和后路都已经在香港准备好了。如果香港变成一个‘臭港’，对中国的一些寡头来说是很不划算的。另外也涉及很多和习近平有着不同政治意见和取向的派别。”
  <br/>
  <br/>
  然而，中国最高领导人派“定心丸”的举动并没有消除香港社会对中央持续干预香港事务，导致香港政治空间紧缩、“一国两制”遭严重腐蚀的疑虑。
 </p>
 <p>
  <br/>

 <p>
  香港《南华早报》日前的报道引述消息人士表示，就在香港特首林郑月娥宣布正式撤回《逃犯条例》修订前，曾将撤回决定提交给习近平批准。
  <br/>
  <br/>
  虽然林郑月娥强调撤回修例是香港政府的决定，但路透社早前披露的一段林郑月娥与商界密谈录音中，林郑月娥表示，香港局势对北京而言已上升至“国家安全和主权议题”，她作为行政长官的“政治操作空间因此变得非常有限”。
  <br/>
  <br/>
  对此，刘细良分析说：“2012年之后‘一国两制’变样已经是一种新常态了。我们想想，如果这次撤回《逃犯条例》修订是需要习近平批准的话，那就说明提出这个修例的就是中国政府，这次政治风波的开始就是源于北京。这和（华为财务长）孟晚舟在加拿大被引渡，中国需要设立一个报复机制有关，和去年发生的香港女子在台湾被杀事件没什么联系，只是林郑月娥想出来用于包装、实现北京政治野心的方式——在香港这个国际商业城市设立引渡外商的机制。”
  <br/>
  <br/>
  有鉴于中央对香港问题的重视程度，香港《明报》30日发评论提到，主导港澳政策协调的角色似乎已经从港澳办转移至中央国家安全委员会下属的国安办，原因是港澳办主任张晓明并没有于日前在北京举行的国庆招待会上现身。
  <br/>
  <br/>
  刘细良认为，该消息很大程度上不是“空穴来风”，但属于正常、暂时性情况。
  <br/>
  <br/>
  “现在处于非常时期，所以才由总书记领导的国家安全部门负责，但日常的政策协调，中联办和港澳办的角色比以往更加重要。过去的话，港澳办主任只是在中央部委里协助特区政府‘打关系’，不是特区政府的直属上司，现在这层关系彻底改变，张晓明是林郑月娥的直属上司。现在是因为张晓明的失误，出现误判而引发大规模的政治危机，所以由国安办接手。”
 </p>

 <p>
  香港的“反送中”抗争已持续将近4个月，还没有消退的迹象。中央政府将会如何处理这场严重的政治危机，也引发不少猜测。
  <br/>
  <br/>
  林郑月娥在上述提到的闭门会谈录音中说道，中央政府“绝对没有计划”出动军队镇压抗议者。
  <br/>
  <br/>
  但路透社星期一的报道则指出，上月进驻香港进行“第22次轮换行动”的解放军原驻港部队并没有撤出香港返回大陆，据统计，目前驻守在香港的驻军人员人数增长了一倍，从原本的3000人至5000千人，增加到1万人至1万2千人。
  <br/>
  <br/>
  夏明接受本台记者采访时说，这意味着：“习近平应该还是担心香港失控，所以需要一个双保险。与此同时，给香港的警察一个安心，也给香港市民一个心理上的震慑。既然军队已经在了，如果要用也比较方便。”
  <br/>
  <br/>
  夏明同时对中央政府的不派军队的“承诺”持怀疑态度。他说，回想30年前的天安门事件，当时政府也表示绝对不会向人民开枪，但事实上政府确实开了枪，并狡辩称是对“暴徒”开枪。由此可见，中央若真的没有打算用军队镇压示威者，就不会派遣解放军前往香港。
  <br/>
  <br/>
