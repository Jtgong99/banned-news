 <p>
  中国大陆官方从八月一日起，暂停赴台个人游，业者预估，今年陆客来台人数恐较去年减少70万人次。旅游业者27日提出陈情，要求蔡英文政府对产业冲击提出对策。
  <br/>
  <br/>
  根据在台湾的移民署统计，今年上半年一至六月陆客总计125万2665人，其中个人游旅客占63万3353人，个人游占所有陆客比重超过五成。
  <br/>
  <br/>
  首当其冲的重灾区之一台北市，根据OTA(网路订房平台，Online Travel Agent)统计，台北市九月订房量衰减1/3。
 </p>
 <p>
 </p>
 <div>
 </div>
 <div>
 </div>
 <div>
 </div>
 <div>
 </div>
 <p>
 </p>
 <p>
  “台湾海峡两岸旅行发展协会”及“台北市旅馆商业同业公会”向国民党立院党团陈情并召开协调会，业者大吐苦水，海峡两岸旅行发展协会理事长姚大光指出，高雄市长韩国瑜承认“九二共识”，今年三到七月观光客人数每个月成长五成。现在个人游喊停，原来期待减少个人游，希望团客略有增加，不幸，中国大陆的团客只剩1/3。
 </p>
 <p>
 </p>
 <p>

  <br/>
  姚大光说，陆客每天在台的平均消费金额为219美元，平均停留7.4天，损失354亿(约人民币78.6亿)实质外汇收入外，相关产业上看台币上千亿(约人民币220亿)损失。他说，蔡政府推出的36亿(约人民币8亿)暖冬旅游补助虽好，但仍是杯水车薪。
  <br/>
  <br/>
  姚大光：“现在只能活着就好，能卑微地活下去，未来可能还有机会。我也希望两岸睿智聪明的领导人，不要把观光当成祭品，让观光能够存活，观光80%都是庶民产业，台湾不照顾庶民要照顾谁。”
  <br/>
  <br/>
  旅游业者和掌管观光的交通部以及负责两岸事务的陆委会等单位举办一场闭门协调会。根据国民党立委许毓仁转述陆委会主委陈明通说法。“陈明通主委有表达说，中国大陆单方面取消自由行的限制，这样的政策不是我们可以控制，只能见招拆招，面对这样的状况，其他部会提出相对应的反制措施。”
 </p>

 </p>

 <p>
  许毓仁认为，陈明通的说法，这是旅宿业的事情，感觉好像没办法帮忙，他认为蔡政府应该要更积极。
  <br/>
  <br/>
  陈明通在会后澄清，许毓仁的解读并不正确。陈明通说，中国大陆一直把开放旅客旅游，当作政策工具。
  <br/>
  <br/>
  “因为大陆人民出国需要管制，因为这一直是他的政策工具，这些风险本来就有所评估，所以这次中国大陆停止自由行，很快的，第二天36亿元扩大秋冬旅游补助就出来，这些都在评估之内。”
  <br/>
  <br/>
  中华民国旅行商业同业公会全国联合会召集人李奇嶽受访时指出，这次陆客停止个人游的重灾区包括，陆客最爱的101、西门町所在的台北市，以及东部的花莲，热门景点阿里山所在的嘉义。
 </p>

  李奇嶽：“台湾跟大陆政府两边都应该要努力把关系维护好，不是说妳台湾政府说都是大陆不来跟我们没关系，是他们的问题。大陆那边又说，是因为你台湾跟大陆关系不好，所以不要。两边政府不该流于把问题丢到对方去。”
  <br/>
  <br/>
  姚大光说，整个产业上下游从业人数约有200万人，如果观光产业团结起来，可以影响未来的选举。
  <br/>
  <br/>
  来自高雄的民进党立委刘世芳则认为：“冲击到谁的选情倒是很复杂。现在高雄市主政是韩国瑜，韩国瑜竞选讲，人进得来、货出得去、高雄发大财。没想到在他代表国民党要选总统的时候，中国用(控制)旅客的方式，两边都惩罚。惩罚民进党也惩罚国民党。”
  <br/>
  <br/>
  刘世芳说，中国至今对香港反送中运动与警民冲突完全找不到有效处理方式，这次禁止个人游来台，恐怕也是为了避免让中国旅客接触到相关资讯有关。
