 <p>
  在香港问题及美中贸易战升温的外部环境下，中共中央政治局30日召开会议，决定今年10月在北京举行中共第19届中央委员会第四次全体会议。观察家认为，外界期待已久的这次会议虽然是走过场，但香港问题和美中贸易战如何发展，届时将让中国国家主席习近平在会上面临压力测试。
  <br/>
  <br/>
  香港问题和美中贸易战持续升温之际，中国中央电视台30日的新闻联播中报导，中共中央政治局当天召开会议，决定中国共产党第19届中央委员会第四次全体会议将在10月于北京举行。
  <br/>
  <br/>
  报导表示，19届四中全会的主要议程是中共中央政治局向中央委员会报告工作，研究坚持和完善中国特色社会主义制度、推进国家治理体系和治理能力现代化若干重大问题。
 </p>
 <div>
 </div>
 <div>
  确定议程大方向的这些用词用语，听在美国哈佛大学政治学博士顾为群的耳里，他认为多是老调重弹。他说，会前安排酝酿这么长时间，应该是习近平要环顾党内情势，等待最佳时机，虽然现在看起来10月开会可能变化不大，但外在环境的情势发展，不是没有变数。
  <br/>
  <br/>
  顾为群：“之所以日期没有宣布，最大的变量就是香港，他（习近平）会观察，看香港到底摆不摆得平，如果摆平了，那就是胜利召开四中全会；但如果摆不平，那会继续推迟。”
  <br/>
  <br/>
  顾为群认为，香港局势的演变，会是四中全会将在10月份哪一天开的最大变数，而届时开会要观察的重点，还包括习近平在党内的权力是否巩固。
  <br/>

  <br/>
  顾为群：“（这）还是中共所谓的民主集中制，就是假民主、真集中，在形式上摆平整个中央委员会，同时他（习近平）也观察，看看中央委员会里面，有没有人真心反对他，估计表面上反对他都不敢，但透过开这个会，他想试图了解中央委员会还有没有反对他的人。”
  <br/>
  <br/>
  中共第十九届领导班子在2017年换届后，照往例应于2018年上半年召开二中全会，下半年召开三中全会。
  <br/>
  <br/>
  然而，习近平去年初推动修宪，取消国家主席任期限制，罕见地在1、2月接连召开19届二中及三中全会。当时外界多预测，19届四中全会应于去年下半年召开，却迟迟未举行，迄今为止中共已长达一年半未召开中央全会，也因为这个原因，这届四中全会可说是让外界“漫长等待”的一次会议。
  <br/>
  <br/>
  在即将迎来建国70周年之际，中共宣布召开四中全会，身处北京政治中心的中国历史学者章立凡也提到，除了香港问题，美中贸易谈判能否在十一建国庆典前拿出好成绩，习近平届时在会上也得向党内报告。
  <br/>
  <br/>
  章立凡：“我觉得还是存在很大难题，如果从现在起，9月、10月这两个月时间，不能对香港及中美贸易问题有个阶段性成果，或者出了更大问题，我觉得，这个会就可能开不好。”
  <br/>
  <br/>
  章立凡认为，美中贸易与香港会是习近平的两大压力。而此时决定四中全会要在10月召开，习近平能否藉由会议召开而稳固权力，也有待观察。
  <br/>
  <br/>
  章立凡：“这两个问题，如果他不解决，或是没有阶段性结果，可能这个会对他（习近平）来讲，不一定是好事。”
  <br/>
  <br/>
  四中全会是否演变成为“会无好会”的会议，外部因素、尤其是香港局势及美中贸易战的发展，在接下来2个月将会有连动性的影响。
  <br/>
