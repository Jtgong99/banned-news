 <p>
  警方周日（20日）在尖沙咀弥敦道以水炮车驱散期间，向清真寺正门喷射蓝色胡椒水剂，事后解释是误中清真寺。行政长官林郑月娥及警务处处长卢伟聪周一到访清真寺，有穆斯林团体代表透露，政府向他们正式道歉。但有民主派议员认为，香港政府应该向更多受影响的团体道歉 。
  <br/>
  <br/>
  行政长官林郑月娥及警务处处长卢伟聪周一上午11时多到达清真寺，与教长及代表会面。他们在清真寺逗留约半小时离开，并没有回应传媒提问。穆斯林团体代表Saeed Uddin在会后透露，林郑月娥及卢伟聪有向他们道歉，并会接受两人的道歉。他指穆斯林团体是和平的社群，没有牵涉任何事件，所以警方也没有理由去破坏清真寺。
 </p>
 <p>
 </p>

 <p>
  Saeed Uddin说：“我们是和平的社群。我们希望继续和平地在这里居住，因为香港是我们的家，清真寺没有被破坏，什么也没有被破坏，唯一做错的事，他们道歉了，我们接受了。”
  <br/>
  <br/>
 </p>
 <p>
  政府新闻处下午发稿，林郑月娥表示，已就事件向穆斯林团体代表致歉。林郑月娥又指，穆斯林社群向来视香港为家，为崇尚多元、包容、共融和开放的社会作出贡献，而面对香港社会近月的纷争，穆斯林教领袖多次呼吁市民和信众保持冷静和团结，与市民大众一起守护香港，林郑月娥对此表示感谢。
 </p>

  <br/>
  <br/>
  毛汉褚简宁说：“哪些是示威者，哪些是市民，你要用头脑做事，不要见人就放蓝色水。我当然不会接受。为什么水炮车走到清真寺门口停下来，你看看影片，停下来然后喷，那便是有心不是非故意。”

  毛汉褚简宁又说，有亲北京立法会议员亲身慰问过他，但同时向他提示，在传媒面前要小心说话，亦不要被其他人利用。
  <br/>
  <br/>
  警方在例行记者会向穆斯林团体致歉。西九龙总区指挥官卓孝业指，警方尊重宗教自由，并非存心冒犯任何宗教团体，将会继续和香港各宗教团体保持良好沟通。他又指，水炮车亦有对圣安德烈堂等其他建筑物造成影响，对此表示抱歉。

 <p>
  <br/>
  卓孝业说：我想再一次向九龙清真寺及穆斯林团体致歉。我们各个警区今早也主动接触区内的穆斯林社群，向他们解释情况。香港是个多元的社区，警队非常尊重宗教自由，以及和所有族群、不同的宗教团体，我们都是没有任何恶意，也非存心冒犯。
  <br/>
  <br/>
 </p>
 <p>
  同样遭受水炮车射蓝色水的立法会议员谭文豪表示不接受政府的道歉，作为负责任的政府应公开向受水炮车影响的清真寺、圣安德烈堂及商铺等致歉。他认为，政府深知事件的严重性，及国际社会对事件的关注，故首次作出道歉。
  <br/>
  <br/>
 </p>

 <p>
  谭文豪：你叫市民不要行私刑，但自己就找清真寺闭门会面，打算把事件给盖过去，这不是负责任政府的态度。第二、受影响的不止清真寺，也有其他附近商铺、无辜市民、也有圣安德烈堂，他们都受蓝色水影响。如果要道歉，林郑月娥应公开向上述人士道歉。
  <br/>
  <br/>
  反送中运动至今，政府首次就警方驱散行动“ 误中副车” 向宗教团体致歉。除清真寺以外，属二级历史建筑的圣安德烈堂也在同日遭受水炮车射蓝色水，而在早前的驱散行动中，警方也曾向黄大仙祠、车公庙发射催泪弹，但林郑月娥及卢伟聪未曾亲自到场致歉。
