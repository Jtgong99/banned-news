 <p>
  辽宁沈阳市民刘振去年在刑事拘留期间暴毙引起的风波，尚未平息。有人怀疑，辽宁两名维权人士近期失去联络，可能与关注刘振的死亡真相有关。甚至有消息传出，两人可能被控以“颠覆国家政权”等罪名。
  <br/>
  <br/>
  去年10月，因房屋拆迁问题维权十余年的沈阳居民刘振与10多人一起到北京的美国大使馆求助，事后他被沈阳警方送到看守所刑事拘留。
  <br/>
  <br/>
  在看守所度过2个多月后，刘振今年1月初突然被送到医院，其后医院声称，他因急性心脏病抢救无效死亡。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  家属回忆刘振在羁押期间精神状态并没有异常，其后查看遗体时，发现他胸口有明显外伤，质疑他的死另有内情。由于类似事件不是首次发生，辽宁不少异议人士在网上敦促政府和公安公布死亡真相。据了解，沈阳当局对此反应强烈，甚至以“颜色革命”形容异议人士的行动，其中林明洁更被视为重点监控对象。今年3月他在北京被沈阳警方拘捕，关押了一个多月才获释。
  <br/>
  <br/>
  到了上月中旬，林明洁与哥哥林明华先后失联。辽宁维权人士祝忠孝相信，两人正被当局羁押，但由于至今未能掌握当局的文书，涉及的罪名尚未明朗。
 </p>
 <p>
 </p>
 <p>
  <img class="image-inline captioned" src="resolveuid/b11b9ef5-ff0f-4c77-b72b-411089b58758"/>
 </p>
 <p>
  祝忠孝：“据说林明洁是9月18日被抓的，他哥哥也是。罪名据说是‘聚众扰乱公共秩序’和‘寻衅滋事’。抓他是因为很多人自发发起悼念刘振的活动。当局就认为是林明洁组织发起的。也有人说，是涉及到‘颠覆国家政权’。”
  <br/>
  <br/>
  林明洁以热心帮助访民维权著称。3年前曾因参与集体上访被沈阳市当局刑事拘留，同年状告沈阳市公安局局长徇私枉法，去年被裁定寻衅滋事罪成判刑2年半，扣除羁押期在数月后获释。
  <br/>
  <br/>
  关注事件的异议人士马波不排除，林明洁失联与集体上访有关。
  <br/>
  <br/>
  马波：“集体访是控告他们信访局或者公安部门、司法部门、政府部门不依法办案。他们（访民）甚至是几十个人去国家信访局，所以说，对辽宁的影响力可能大一些。”
  <br/>
  <br/>
  知情人士说，林明洁没有子女，而林明华唯一的女儿目前在北京打工，联系不上，目前只好期待林明华尽早获释，以家属身份为林明洁正式委托律师。
  <br/>
  <br/>
  记者：高锋   责编：胡力汉/申铧  网编：瑞哲
 </p>
