 <p>
  关注中国人权状况的“六四天网”的吉林义工王晶上周末刑满出狱。据知情人士透露，她此前在看守所多次遭到殴打，脑癌病情恶化。她出狱后，当局还警告她的亲朋好友不能接受媒体采访。
  <br/>
  <br/>
  维权网消息，因在2014年“两会”期间报道访民天安门自焚事件被判刑的“六四天网”公民记者王晶上周日刑满出狱。
  <br/>
  <br/>
  消息说，出狱当天，王晶就公开了她的近照，并表示“同志们我回来了。”据悉，她在看守所期间曾多次遭到警察殴打。
  <br/>
  <br/>
 </p>
 <div>
 </div>
 <div>
  一位知情人士向本台证实了王晶刑满出狱的消息，并表示，当事人目前承受着巨大压力，因为她身边的人都被告知不许接受媒体采访。而她本人也很恐惧，担心会再次被捕。
  <br/>
  <br/>
  这位知情人士还透露，王晶目前身体状况堪忧。在押期间，她的脑瘤明显扩大。出于安全考虑，这位知情者不愿透露身份，她的声音也经过处理。
  <br/>
  <br/>
  “（王晶）目前想尽快回日本治病。现在她的病情都复发了，身上一堆病，主要她的脊柱严重受伤。”
  <br/>
  <br/>
  王晶的丈夫在日本，所以她可以去日本治病。

  本台记者试图联系她的母亲孙燕华和她的辩护律师之一李静林，但并未成功。
  <br/>
  <br/>
  据悉，王晶的二姐于1993年上班期间莫名遇害，警方多年来一直没有破案。王晶为了姐姐的冤案长年上访，但遭到警方严酷打压，并多次被拘留。这些年间，王晶在与当局抗争的过程中，成为了一名关注访民的公民记者。
  <br/>
  <br/>
  2014年3月5日，在中国第十二届全国人大二次会议开幕当天，王晶和“六四天网”的其他两名义工，因在天安门广场拍摄访民自焚并发布相关消息，被控“寻衅滋事”，并被刑事拘留30天，此后被取保候审。
  <br/>
  <br/>
  同年12月10日世界人权日那天，王晶在北京中央电视台附近拍摄抗议照片时被拘留。2016年4月，吉林法院以“寻衅滋事罪”判处王晶有期徒刑4年零10个月。
  <br/>
  <br/>
  本台此前对王晶的遭遇作过多次报道。2015年，曾参与王晶庭审的四川维权人士杨秀琼透露，王晶对在场人员描述了她在审讯期间遭到的虐待。她引述王晶说，当时审讯人员几天都不让她吃饭、喝水或睡觉。在看守所时，她还被泼冷水，嘴巴还被堵上了。
  <br/>
  <br/>
  美国华盛顿聚焦中国人权状况的“改变中国”(China Change)网创办人兼主编曹雅学表示，近年来，北京当局对发布公民维权信息的平台进行了集中打压，而这些网站背后的关键人物也遭到了严惩。
  <br/>
  <br/>
  “这一两年，中国政府对维权网站进行了系统性地围剿。甄江华办的‘权利运动’网站被关闭，‘民生观察’网负责人刘飞跃被判刑，‘六四天网’创办人黄琦和给他做工的人也被抓了。”
  <br/>
  <br/>
  王晶曾任义工的“六四天网”遭到了毁灭性打击。今年7月，“六四天网”创办人黄琦被四川法院以“故意泄露国家秘密罪”以及“为境外非法提供国家秘密罪”被判处12年徒刑。黄琦于1999年6月4日创办“六四天网”，多年来披露了当局镇压法轮功学员、四川震区豆腐渣工程等众多维权事件。
  <br/>
  <br/>
  曹雅学说，这些人权网站之所以遭到当局的严酷打压，是为了起到杀鸡儆猴的作用。
  <br/>
  <br/>
  “当局就是要把这些信息全部清除掉，然后对相关人士判刑，以此给其他人造成一个恐惧效应。所以，这个效果已经达到了，现在（这些信息）的确处于一个低潮。”
  <br/>
  <br/>
  王晶周四在一封致吉林市政府、市公安局的公开信中写道，现在她家里只剩下三个体弱病重的妇女。她的母亲目前处于“在家等死”的状态。她自己的脑瘤复发，正等待手术，而她的大姐身体也欠佳。
  <br/>
  <br/>
  王晶在这封公开信中恳求政府解决家人的信访问题，给她们一个说法，并给予书面答复。她还写道：“如果说公平、正义的新时代已经来临，那么可不可以说：我和我家人的悲惨遭遇就是那毫无公平、正义可言的旧时代的产物呢？”
  <br/>
  <br/>
  自由亚洲电台记者家傲华盛顿报道。 责编：申铧 网编：郭度
