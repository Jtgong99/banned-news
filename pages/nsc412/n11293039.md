<hr/><p>
 【大纪元2019年06月01日讯】位于纽约的大纪元媒体集团属下的数字网路媒体经营部门EDN（Epoch Digital Network）诚聘“EDN数字广告项目执行经理”，负责将目前的数字媒体资源转化成资金。同时，大纪元媒体集团诚聘6名“销售人员”和6名“销售助理”。
</p>
<h4>
 <a href="http://www.epochtimes.com/gb/tag/%E6%8B%9B%E8%81%98.html">
  招聘
 </a>
 职位：EDN数字广告项目执行经理
</h4>
<p>
 EDN（Epoch Digital Network）是属于大纪元媒体集团旗下的数字网路媒体经营部门，位于纽约总部。旗下整合多语种大纪元网站、多语种新唐人电视台网站、及优美客视频网站等大型专业网络媒体及社交媒体平台。EDN经营部门是目前及未来媒体发展重要的资金来源之一，主要负责将目前的数字媒体资源转化成资金。
</p>
<p>
 职位描述：
</p>
<ol>
 <li>
  管理数字广告项目执行，包括媒体自身产品的数字广告推广（中英文媒体付费订阅、中英文媒体捐款，产品销售、大赛等） 以及全球销售的广告客户的数字广告执行；
 </li>
 <li>
  根据不同的项目需求，在多个数字广告平台上完成数字广告的运行，跟踪执行进度，达到项目的KPI， 数据分析并出结案报告，计划下一步如何优化广告表现，用最少的钱达到最好的效果。
 </li>
 <li>
  内外沟通：对外与公司合作伙伴、广告平台、行业会议用英文沟通，对内与不同部门沟通（销售、数字广告设计、技术人员、财务、各项目经理、市场部、EDN团队），确保广告的准确执行。
 </li>
 <li>
  能及时发现并解决广告项目运营中的问题，并尽可能优化广告表现，节省媒体资源
 </li>
 <li>
  学习和掌握最新的数字媒体广告技术，涉及的广告种类：展示广告、视频广告、搜索广告、Facebook广告，原生广告，赞助内容 等。
 </li>
 <li>
  有机会参与数字广告产品的设计与策划，如EDN的media kit／对外网站的设计与制作，客户的landing page，rich media creative (html5, css, js.)等。
 </li>
 <li>
  有机会参加各种数字广告行业的活动，学习最先进的数字广告行业知识，与同行交流。
 </li>
 <li>
  修炼必须正向、积极、热情，因为会遇到主管、其他部门同事、客户、业务、履约的考验和压力。
 </li>
</ol>
<p>
 工作要求：
</p>
<ol>
 <li>
  具备中英文双语书写、阅读及交流能力 ，在西方大学毕业为佳。媒体发展多语种网站和社交媒体，有多语种能力更佳！
 </li>
 <li>
  对工作认真负责，仔细耐心，不怕吃苦，愿意为他人服务。
 </li>
 <li>
  会使用Word，Excel，PowerPoint，熟悉数字媒体、社交媒体。
 </li>
 <li>
  有市场营销、广告相关的经验为佳。
 </li>
 <li>
  对数据敏感，基本的计算能力，或用excel等软件计算的能力。能解读数字、图表。
 </li>
 <li>
  会html5, css, js为佳。
 </li>
 <li>
  自主学习能力强，主动性强，有很强的自我管理能力。
 </li>
 <li>
  有团队合作意识，可与多个部门沟通和跨地区、跨时区、跨语言沟通。具备良好的理解和沟通能力。
 </li>
 <li>
  能在有压力的环境下工作，能随时发现问题并及时改进。
 </li>
</ol>
<ul>
 <li>
  有扎实的修炼基础，有强烈的救人使命感， 有热情。
 </li>
</ul>
<p>
 地址：纽约曼哈顿
 <br/>
 全职工作，薪资面议。有经验更佳，会提供3个月的专业培训。
