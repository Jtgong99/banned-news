 二手货，价被抬。黄盖假，板真挨。
 <br/>
 豆市场，逗乐坏。求改谈，重新来。
 <br/>
 谈九轮，诺排排。说重新，都旧债。
 <br/>
 入世贸，有记载。要兑现，重新来。
 <br/>
 又不干，吞肚埋。谈判再，重新来。
 <br/>
 美不应，拒无赖。关税增，调又改。
 <br/>
 拖刀计，挨脚踹。物价涨，贼洞塞。

 打土豪，本不该。那土地，正当来。
 <br/>
 反腐败，铡应开。但铡口，开太窄。

 <br/>
 储量大，乱开采。污染重，实烂牌。

  末日
 </a>
 来。

 地利无，无处踩。人心失，没人睬。
 <br/>
 魔把党，地狱拽。重新来，新何来？
 <br/>
 昔日景，回不来。党黑锅，重血债。
 <br/>
 纵万刮，还不来。牵驴的，溜计乖。
 <br/>
 把橛者，全担待。善恶报，一起来。
 <br/>
 地狱门，冲党开。马克思，在等待③。
 <br/>
 除红魔，天安排。
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
  三退
 </a>
 ④潮，灭顶来。
 <br/>
 自相斗，放血脉。路自选，谁别怪。
 <br/>
 唯好人，有未来。办
 <a href="http://www.epochtimes.com/gb/tag/%E4%B8%89%E9%80%80.html">
  三退
 </a>
 ，从新来。
