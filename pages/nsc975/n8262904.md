 <strong>
  遨游仙界

<p>
 在《古风》其七中
 <a href="http://www.epochtimes.com/gb/tag/%E6%9D%8E%E7%99%BD.html">
  李白
 </a>
 写出其与名道“千岁翁”安期公相见场景。《寄王屋山人孟大融》则描述李白在崂山东海亲尝安期公所赠之枣。近千年前，千古一帝秦始皇东巡琅邪之中，在崂山曾经召见过这位比彭祖还寿长200年的安期公，密谈了三天三宿。安期公师从河上公。当年，安期公离开时，给秦始皇留书并留言，“千年之后，求我于蓬莱山下。”（汉刘向《列仙传》，晋皇甫谧《高士传》）但千年以后，却是李白亲尝安期公所赠之枣，并和安期公一同畅游天庭。莫非历史深邃的时空中藏有更深的谜底？
</p>
<p>
 《古风》其七
</p>
<p>
 五鹤西北来，飞飞凌太清。
 <br/>
 仙人绿云上，自道安期名。
 <br/>
 两两白玉童，双吹紫鸾笙。
 <br/>
 去影忽不见，回风送天声。
 <br/>
 我欲一问之，飘然若流星。
 <br/>
 愿餐金光草，寿与天齐倾。
</p>
<p>
 《寄王屋山人孟大融》
</p>
<p>
 我昔东海上，劳山餐紫霞。
 <br/>
 亲见安期公，食枣大如瓜。
 <br/>
 中年谒汉主，不惬还归家。
 <br/>
 朱颜谢春晖，白发见生涯。
 <br/>
 所期就金液，飞步登云车。
 <br/>
 愿随夫子天坛上，闲与仙人扫落花。
</p>
<p>
 唐玄宗虽然让
 <a href="http://www.epochtimes.com/gb/tag/%E6%9D%8E%E7%99%BD.html">
  李白
 </a>
 还山，但他仍然通过各种渠道打听李白信息及其诗赋。这两首崂山诗遂让唐玄宗对崂山产生浓厚兴趣。之后，天宝七年（748年）他派遣几名道士前往崂山采药，并将崂山命名为“辅唐山”。崂山因此而更加出名。
</p>
<p>
 李白“幸遇王子晋”。王子晋，又名王子乔，周灵王太子晋也，好吹笙作凤凰鸣。游伊、洛之间，遇道士浮丘公，上嵩高山，修炼飞升成仙而去。这是李白组诗《古风》五十九首中第四十首。诗人遇仙人王子晋，结交于仙境，依依惜别。
</p>

<p>
 《古风》其四十
</p>
<p>
 凤饥不啄粟，所食唯琅玕。
 <br/>
 焉能与群鸡，刺蹙争一餐。
 <br/>
 朝鸣昆丘树，夕饮砥柱湍。
 <br/>
 归飞海路远，独宿天霜寒。
 <br/>
 幸遇王子晋，结交青云端。
 <br/>
 怀恩未得报，感别空长叹。
</p>
<p>
 再看李白同“东方朔”游天堂、见上皇仙游之旅。
</p>
<p>
 《古风》其四十一
</p>
<p>
 朝弄紫泥海，夕披丹霞裳。
 <br/>
 挥手折若木，拂此西日光。
 <br/>
 云卧游八极，玉颜已千霜。
 <br/>
 飘飘入无倪，稽首祈上皇。
 <br/>
 呼我游太素，玉杯赐琼浆。
 <br/>
 一餐历万岁，何用还故乡。
 <br/>
 永随长风去，天外恣飘扬。
</p>
<p>
 紫泥海，指东方朔成仙事。《汉武洞冥记》：“东方朔去，经年乃归。母曰：‘汝行经年一归，何以慰我耶？’朔曰：‘儿至紫泥海，有紫水污衣，仍过虞渊（日入之处）湔洗（洗涤），早发中返，何云经年乎？’”这是李白组诗《古风》五十九首中第四十一首，描述诗人受东方朔之邀，喝其玉杯所赐琼浆，一起遨游仙界。
</p>
<figure class="wp-caption aligncenter" id="attachment_8266758" style="width: 300px">
 <a href="http://i.epochtimes.com/assets/uploads/2016/09/1609040850361456.jpg">
  <img alt="" class="wp-image-8266758" src="http://i.epochtimes.com/assets/uploads/2016/09/1609040850361456-600x1244.jpg"/>
 </a>
 <br/><figcaption class="wp-caption-text">
  《晩笑堂竹庄画传》东方朔画像（公有领域）
 </figcaption><br/>
</figure><br/>
<p>
 当年秦始皇身边博士卢敖辞官隐居，遇见真人，授以仙诀，遂远遁深山修炼，道成飞升。汉武帝元鼎四年，为纪念卢敖，在河南设卢氏县。李太白在《庐山谣寄卢侍御虚舟》中表明“早服还丹无世情，琴心三叠道初成”，此时金丹服下，大道初成；和仙人九天之上已经约会，要去元始天尊之玉京，并要接卢敖（这里借指卢虚舟）一起遨游太清天界：
</p>
<p>
 我本楚狂人，凤歌笑孔丘。
 <br/>
 手持绿玉杖，朝别黄鹤楼。
 <br/>
 五岳寻仙不辞远，一生好入名山游。
 <br/>
 庐山秀出南斗傍，屏风九叠云锦张，影落明湖青黛光。
 <br/>
 金阙前开二峰长，银河倒挂三石梁。
 <br/>
 香炉瀑布遥相望，迥崖沓障凌苍苍。
 <br/>
 翠影红霞映朝日，鸟飞不到吴天长。
 <br/>
 登高壮观天地间，大江茫茫去不还。
 <br/>
 黄云万里动风色，白波九道流雪山。
 <br/>
 好为庐山谣，兴因庐山发。
 <br/>
 闲窥石镜清我心，谢公行处苍苔没。
 <br/>
 早服还丹无世情，琴心三叠道初成。
 <br/>
 遥见仙人彩云里，手把芙蓉朝玉京。
 <br/>
 先期汗漫九垓上，愿接卢敖游太清。
</p>

 <a href="http://i.epochtimes.com/assets/uploads/2016/09/1609031252301456.jpg">
  <img alt="" class="wp-image-8265206" src="http://i.epochtimes.com/assets/uploads/2016/09/1609031252301456-600x317.jpg"/>

 </figcaption><br/>
</figure><br/>
<p>
 再看下面两首诗，第一首讲诗人受秘诀于仙人韩众（又名韩终，当年秦始皇派其出海去寻仙药），玉女千余人相随，又去当年黄帝及浮丘公炼丹之地，遇见浮丘公后，又有王子乔加入，“吹笙舞松风”。第二首讲仙人玉女骑彩凤而至，赠诗人绿玉杯和紫琼琴，此二仙物，皆非世间所有。
</p>
<p>
 《至陵阳山登天柱石酬韩侍御见招隐黄山》
</p>
<p>
 韩众骑白鹿，西往华山中。
 <br/>
 玉女千余人，相随在云空。
 <br/>
 见我传秘诀，精诚与天通。
 <br/>
 何意到陵阳，游目送飞鸿。
 <br/>
 天子昔避狄，与君亦乘骢。
 <br/>
 拥兵五陵下，长策遏胡戎。
 <br/>
 时泰解绣衣，脱身若飞蓬。
 <br/>
 鸾凤翻羽翼，啄粟坐樊笼。
 <br/>
 海鹤一笑之，思归向辽东。
 <br/>
 黄山过石柱，巘崿上攒丛。
 <br/>
 因巢翠玉树，忽见浮丘公。
 <br/>
 又引王子乔，吹笙舞松风。
 <br/>
 朗咏紫霞篇，请开蕊珠宫。
 <br/>
 步纲绕碧落，倚树招青童。
 <br/>
 何日可携手，遗形入无穷。
</p>
<p>
 《拟古十二首》其十
</p>
<p>
 仙人骑彩凤，昨下阆风岑。
 <br/>
 海水三清浅，桃源一见寻。
 <br/>
 遗我绿玉杯，兼之紫琼琴。
 <br/>
 杯以倾美酒，琴以闲素心。
 <br/>
 二物非世有，何论珠与金。
 <br/>
 琴弹松里风，杯劝天上月。
 <br/>
 风月长相知，世人何倏忽。
</p>
<p>
 赏读至此，可以看出李白天宝年进宫之前，寻道、访仙并修炼。由于其来源非比寻常，佛、道、神诸家都和其渊源很深，并一再邀其修炼；而且由于其修炼有成，可以看到很多常人看不到之景象，如访泰山、道童劝其修炼、和得道高僧攀谈佛法、打坐静修，等等。从他离开皇宫后的诗文可看出，他与神佛聚会，来往于各个空间，仙山神苑，心随愿往，超然世间。
</p>
<p>
 诗仙在其诗词中品评历史人物不下二百余人，亦对很多风流人物留下嘉评。
</p>
<p>
 《公无渡河》中，李白感述大禹无私治理滔天洪水之丰功伟绩：
</p>
<p>
 黄河西来决昆仑，咆哮万里触龙门。
 <br/>
 波滔天，尧咨嗟。
 <br/>
 大禹理百川，儿啼不窥家。
 <br/>
 杀湍湮洪水，九州始蚕麻。
</p>

<p>
 《古风三》中诗仙赞佩千古一帝秦始皇：
</p>
<p>
 秦皇扫六合，虎视何雄哉！
 <br/>
 飞剑决浮云，诸侯尽西来。
 <br/>
 明断自天启，大略驾群才。
 <br/>
 收兵铸金人，函谷正东开。
</p>
<p>
 《赠新平少年》中，李白如是评论兵仙韩信：
</p>
<p>
 韩信在淮阴，少年相欺凌。
 <br/>
 屈体若无骨，壮心有所凭。
 <br/>
 一遭龙颜君，啸吒从此兴。
 <br/>
 千金答漂母，万古共嗟称。
</p>

<p>
 李白对项王的勇武十分欣赏。
</p>
<p>
 《登广武古战场怀古》
</p>
<p>
 秦鹿奔野草，逐之若飞蓬。
 <br/>
 项王气盖世，紫电明双瞳。
 <br/>
 呼吸八千人，横行起江东。
</p>
<p>
 赤壁之旁，遥想当年周瑜联手诸葛亮，孙、刘两家以少胜多，大破曹军，留下千古美谈，诗仙挥毫赞叹。
</p>
<p>
 《赤壁歌送别》
</p>
<p>
 二龙争战决雌雄，赤壁楼船扫地空。
 <br/>
 烈火张天照云海，周瑜于此破曹公。
</p>
<p>
 塞上漠北曾引无数诗人留下笔墨。诗仙在其《塞上曲》中描述当年唐太宗皇帝大破匈奴之举。
</p>
<p>
 《塞上曲》
</p>
<p>
 大汉无中策，匈奴犯渭桥。
 <br/>
 五原秋草绿，胡马一何骄。
 <br/>
 命将征西极，横行阴山侧。
 <br/>
 燕支落汉家，妇女无华色。
 <br/>
 转战渡黄河，休兵乐事多。
</p>
<p>
 <a href="http://www.epochtimes.com/gb/tag/%E8%AF%97%E4%BB%99%E6%9D%8E%E5%A4%AA%E7%99%BD.html">
  诗仙李太白
 </a>
 所留诗篇不下千首，洋洋大观，横跨东西，纵贯千古，非一两篇文章所能概括。献此拙文，唯再现诗仙脍炙人口大作中有数几篇，及古今世人很少评论、却揭示丰富神传文化、修炼正理数篇珍品而已。但笔者毕竟智浅才疏，难于理解诗仙真意之万一，只作引玉之砖耳。
</p>
<p>
 <strong>
  结语 诗风成就烁千古 神传文明宝中瑰
 </strong>
</p>
<p>
 亘古大穹，走过了无数时光，创造了无限辉煌。“成、住、坏、灭”规律掌控着大穹以及不同天体体系宿命。创世主为大穹之永远创造了圆融不灭机制，并在这五千年中华历史大戏中，安排
 <a href="http://www.epochtimes.com/gb/tag/%E5%8D%83%E5%8F%A4%E8%8B%B1%E9%9B%84%E4%BA%BA%E7%89%A9.html">
  千古英雄人物
 </a>
 来世间与众生结缘、演绎，奠定人类得法修炼所需文化、思想及内涵。
</p>

<p>
 李白作为诗人，来在世间，开创正统文风、诗风，引领文学艺术领域风骚，开示什么是文人应走之路，什么是正统神传文化艺术；特别是他的高贵人品、人格，更为后世树立千古风范。
</p>
<p>
 <a href="http://www.epochtimes.com/gb/tag/%E8%AF%97%E4%BB%99%E6%9D%8E%E5%A4%AA%E7%99%BD.html">
  诗仙李太白
 </a>
 的诗词歌赋，及常人很难理解的梦言、醉语、游仙之诗，融作者身世、语言、文风、诗意、人格、心性、道德、情操、境界、修炼层次、仙界所见、空间切换、出世入凡等等于自然完美之一体，成为中华神传文化、文学艺术之巅峰绝仞，无人可企及；更让读者受益无穷，也使社会道德提高回升。他把正道修炼注入诗、词、歌、赋，用神传文化、修炼正理主导创作，讴歌佛、道、神。其伟大人格、正统诗风、文风，这一切一切展现了创世主及神让人类文明、中华文明所应具有内涵，因此流传百世，留下千古绝唱!
</p>
<p>
 五千年中华文明历史中，一场场惊心动魄的大戏，一幕幕刻骨铭心的剧情，铸就了人类历史辉煌，在亘古大穹中熠熠生辉，永远闪耀。诗仙李太白在中华大唐盛世中璀璨峻拔、高远旷世的贡献，更是中华五千年文明瑰宝中之瑰宝，辉煌中之辉煌。
</p>
