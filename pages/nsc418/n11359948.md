 【大纪元2019年07月02日讯】（大纪元记者徐简综合报导）在香港民众
 <span href="http://www.epochtimes.com/gb/tag/%E4%B8%83%E4%B8%80.html">

 </span>

 <span href="http://www.epochtimes.com/gb/tag/%E8%8B%B1%E5%9B%BD.html">
  英国
 </span>
 外长再次警告中共，如果违反保护香港自由的《中英联合声明》，将会产生“严重后果”。他还说英国“坚定支持”香港人民。
</p>
<h4 class="p1">
 <span class="s1">
  <span href="http://www.epochtimes.com/gb/tag/%E8%8B%B1%E5%9B%BD.html">
   英国
  </span>
  “坚定的支持香港人民”
 </span>
</h4>
<p class="p1">
 <span class="s1">
  英国外交部长杰里米‧亨特（Jeremy Hunt，侯俊伟）对媒体表示，英国谴责“各方面”的暴力行为，但强调“深切关注香港人的基本自由受到攻击”。
 </span>
</p>
<p class="p1">
 <span class="s1">
  6月份，香港爆发了两次超过百万人走上街头，反对“
  <span href="http://www.epochtimes.com/gb/tag/%E9%80%83%E7%8A%AF.html">
   逃犯
  </span>
  条例”（又称送中条例）的修订。7月1日再有55万人上街，当晚数以千计年轻示威者冲击并闯入立法会大楼，喷写抗议标语，还挂上了英国统治时期的香港旗。警方施放催泪瓦斯驱离人群。
 </span>
</p>
<p class="p1">
 <span class="s1">
  亨特说，任何镇压都将会加剧民众的忧虑。许多支持者看到周一晚间的电视画面后，深感不安。但是，英国敦促港府不要利用事件作为镇压的借口，而是要理解其背后的原因，即香港人民对他们的基本自由受到侵害而深感焦虑。“我们坚定的支持香港人民。”他说。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  如果违反中英联合声明 “后果严重”
 </span>
</h4>
<p class="p1">
 <span class="s1">
  亨特说，根据1984年签订的《中英联合声明》规定，香港自1997年以后享有“一国两制”的治理模式，该模式允许香港人拥有中国大陆民众所不享有自由，包括抗议自由和司法独立。亨特警告，港府对民众的镇压，违背了中英联合声明。
 </span>
</p>
<p class="p1">
 <span class="s1">
  “如果不尊重这个具有国际约束力的法律协议，将会产生严重后果，”亨特先生警告说，“它保障了香港人的基本自由，我们坚定支持联合声明，坚定支持香港人。”
 </span>
</p>
<p class="p1">
 <span class="s1">
  在香港人民反抗中共送中条例的过程中，英国外长频频为香港人民发声，他在7月1日接受SKY新闻采访时说，“当我看到香港那些可怕的场景时，我为那些必须为自由而战的人表示同情。”当天，英国外交部发表英国对香港主权移交22周年的声明。
 </span>
</p>
