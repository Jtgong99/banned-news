 <div>
  <br/>
  据自由西藏的消息，中国政府拆迁亚青邬金禅林僧舍的计划始于今年8月，是政府削减这个地方居民数量的长期努力的一部分，上千名僧侣尼姑被逐出家门和朝拜场所，600个武装人员被派遣到当地监视居民。
  <br/>
  <br/>
  卫星图像显示出，原本人口稠密的河流西岸现在被夷为平地，修建了新的旅店和停车场。这意味着拆迁僧舍的关键原因之一是发展旅游业。
  <br/>
  <br/>
  亚青邬金禅林位于西藏甘孜州白玉县昌台乡，原本约有1万名居民，绝大多数是藏人。该地区是西藏第二大僧侣聚居区，也是世界上最大的尼姑聚居区之一。
  <br/>
  <br/>
  援藏组织“西藏观察”（Tibet Watch）确认有70个尼姑被逐出亚青邬金禅林，但是据当地消息来源说，实际数字起码有3,500个。

  自由西藏发布的新闻稿说，这70个尼姑被送回到她们在300公里之外的家乡江达县，并被拘留长达两三个月，接受爱国主义再教育，被迫赞美中国并谴责西藏流亡精神领袖达赖喇嘛。
  <br/>
  <br/>
  自由西藏组织的负责人约翰·琼斯（John Jones）认为，这些最近的图像再度证明了中国共产党对西藏人民的蔑视，一个人民甚至都不承认的政府强迫他们忍受背井离乡的厄运。
  <br/>
  <br/>
  约翰·琼斯呼吁国际社会向中国施压，迫使他们停止这个毁灭性的计划，允许亚青邬金禅林的居民生活在平静之中。
  <br/>
  <br/>
