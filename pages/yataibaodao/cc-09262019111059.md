 <p>
  香港民主党派“香港众志”主席罗冠聪9月26日在美国国会喊话，呼吁国际社会联合起来对抗极权，支持香港追求自由、民主、自治的斗争，尽快推进《香港人权与民主法案》。
  <br/>
  <br/>
  9月26日星期四，美国参议院外交关系委员会下属的东亚和太平洋事务及国际网络安全政策小组委员会举行听证会，就香港争取自由、自治和人权的运动进行了讨论。
  <br/>
  <br/>
  罗冠聪在开场发言中指出，香港警察的丧失人性和卢旺达大屠杀有相似之处，“在（对示威者的）肉体虐待之外，在警察中存在普遍的、危险的人性沦丧，他们把抗议者比作蟑螂和物品。”
  <br/>
  <br/>
  他提到，香港政府躲在警察背后，纵容暴徒任意攻击抗议人士甚至过路人。警察射击抗议者头部，造成三个人的眼睛受到永久性伤害。大批香港示威者被警方送到港深边境的新屋岭扣留中心。
  <br/>
  <br/>
  罗冠聪认为香港人不能独自对抗二十一世纪最大的极权力量，他呼吁美国参议院积极推进《香港人权与民主法案》的立法，希望国际社会敦促北京遵守《中英联合声明》中的一国两制原则。
  <br/>
  <br/>
  十一国庆即将到来，罗冠聪提醒北京：“人的心灵和头脑不能简单地被精心编排的仪式收买。香港人会继续争取自治和民主。你可以通过兑现诺言来证明你的勇敢，或者通过镇压人民继续展现你的懦弱。自由世界在看着你。”
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  华盛顿国际咨询公司总裁、前美国副总统切尼的国安顾问叶望辉（Stephen J. Yates）认为，中国对待香港的方式至关重要，这将告诉我们中国将成为一个什么样的国家。
  <br/>
  <br/>
  他说，“我们目前有两个关键性的考验。第一，在履行国际义务方面，北京能不能被信任。第二，中国人能不能和自由社会和平共存。”
  <br/>
  <br/>
  叶望辉也回顾了美国对华策略的失误。他认为中国与日俱新，而美国的政策在过去的五十年却保持“可悲的怠惰”，美国应该全面地、根本地重新审视对华战略。
 </p>

 <p>
  2047年将是香港前途的分水岭。在《中英联合声明》和《香港特別行政区基本法》中规定，“香港特别行政区不实行社会主义制度和政策，保持原有的资本主义制度和生活方式，五十年不变”。
  <br/>
  <br/>
  民主党籍参议员爱德华·马基（Edward Markey）就2047年之后，香港向何处去的问题向证人们提问。
  <br/>
  <br/>
  叶望辉预计，2047年之前香港会成为共产党控制下中国体系的一部分，和北京上海毫无区别。中国共产党届时会全面控制中国：一个国家，一个体制。
  <br/>
  <br/>
  罗冠聪认为，2047年在香港有里程碑的意义，只要人们联合起来，仍然有二十多年的时间可以改变中国。
