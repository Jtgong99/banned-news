 <p>
  美国国务院主管宗教信仰自由的无任所大使萨姆·布朗巴克表示，中国应该停止正在进行的所有宗教迫害。他在接受自由亚洲电台维语部采访时，呼吁世界主要穆斯林国家站出来对中国施加压力。
  <br/>
  <br/>
  布朗巴克大使过去几个月在全世界访问多个国家，呼吁世界各国对宗教迫害问题予以重视。10月17日，他在接受本台维语部采访时，证实美国驻印度尼西亚大使，曾与当地规模最大的两个穆斯林组织会谈，讨论正在中国新疆发生的事件。
 </p>
 <div>
 </div>
 <div>
  他表示，虽然西方多个主要国家非常关注中国当局在新疆的集中营中关押超过一百万维吾尔人，并给他们洗脑，清除所谓极端宗教思想的问题，但世界上的主要穆斯林国家至今仍然维持沉默，因为他们担心失去来自中国的投资，或者与中国的贸易受到影响。他提醒说，历史将会记住这些国家在穆斯林信徒遭到迫害时的沉默。他希望国际上民间的穆斯林团体对事件提高关注，并敦促本国政府对此发声。
 </div>

 <div>
  他认为，新疆维吾尔人和其他信仰伊斯兰教的少数民族，被中国政府强迫洗脑，以接受中国的意识形态，这违反了联合国人权宪章，也违反了中国宪法中有关宗教信仰自由的条款。
  <br/>
  <br/>
  上个星期，美国国务院宣布对参与迫害穆斯林的中国新疆官方机构和个人，以及某些协助当局进行人权迫害的企业进行制裁。
  <br/>
  <br/>
  布朗巴克大使表示，过去几年，中国对穆斯林少数民族、基督教地下教会、藏传佛教人士以及法轮功成员的迫害日益严重。美国会继续向中国施加压力，包括对某些参与迫害的企业及个人，进行制裁，如禁止与这些企业发生业务往来、禁发签证给相关个人等。他并明确呼吁，中国应该停止所有的宗教和信仰迫害。

  <br/>
