 <p>
  正在狱中服刑的维吾尔学者伊力哈木·土赫提近日被欧洲议会提名，角逐欧洲最高人权奖。
  <br/>
  <br/>
  欧洲的最高权力机关欧洲议会日前提名了著名的维吾尔学者，身在狱中的伊力哈木·土赫提作为候选人，角逐2019年度欧洲最高人权奖哈维尔人权奖。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  伊力哈木·土赫提曾经是中央民族大学的经济学副教授，2014年因分裂国家罪被中国判处无期徒刑。欧洲议会周一的一份声明中，评价伊力哈木“20多年来一直致力于改善维吾尔人的处境，促进不同民族之间的对话和理解”。
  <br/>
  <br/>
  旅居加拿大的时政评论人士盛雪在2017年担任国际笔会和平委员会副主席时，曾提名伊力哈木参与竞逐这一奖项。她在接受本台采访时说，“伊力哈木作为一个维吾尔学者，可以说他的大半生都在致力于民族和解、和平、理解、沟通这样的工作。但是，他却不幸被判处了终身监禁。所以说像他这样的人在监狱里边，应该说是一个极大的讽刺。”
  <br/>
  <br/>
  今年年初，美国多名来自不同党派的国会议员提名伊力哈木角逐今年的诺贝尔和平奖。议员们致函诺贝奖委员会，强调没有人比和平争取人权的伊力哈木更值得获奖。
  <br/>
  <br/>
  本次伊力哈木被提名欧洲人权奖，正值中国对维吾尔人的压迫逐步为国际社会所了解。人权组织估计，已有超过一百万维吾尔人被关进再教育营。中国一开始否认这些营地的存在，后来说这些营地是“职业教育中心”，目的是为了去极端化和培训就业技能。上个月，北京说大部分再教育营里的人已经回家，并没有提供更多细节。
  <br/>
  <br/>
  盛雪还表示，伊力哈木此次被提名，得奖的可能性很大。因为维吾尔人在新疆集中营受迫害的状况已经为世人广泛地了解，这种情况会极大地提高伊力哈木的知名度，评选委员会也能够更多地关注到伊力哈木做出的贡献。
  <br/>
  <br/>
  本次哈维尔人权奖共有三名候选者，除伊力哈木以外，塔吉克斯坦的人权律师布泽梅尔·约洛夫和巴尔干的青年战后和解组织也都被提名角逐这一奖项。评选结果将于9月30日在欧洲议会的总部斯特拉斯堡揭晓，获奖者将得到6万欧元的奖金。
  <br/>
  <br/>
  哈维尔人权奖以捷克前总统，苏联时期持不同政见者瓦茨拉夫·哈维尔的名字命名。过去的获奖者包括2018年诺贝尔和平奖获得者，来自伊拉克的雅兹迪人权活动家，曾受到伊斯兰过酷刑和性虐待的纳迪娅·穆拉德。
  <br/>
  <br/>
  自由亚洲电台实习记者曲亦歌华盛顿综合报道。 责编：申铧 网编：郭度
