### 二十年电子游戏狂 因修炼脱胎换骨

<p>
 【大纪元2019年05月15日讯】我出生于一九八六年，生活在一个无神论肆虐、人类道德迅猛下滑、金钱物欲极度膨胀的社会中。与大法结缘后使我在这个乱世中能够明辨是非，分清正邪，守住良善，不断回归先天本性。
</p>
<div class="ar_AuthorDate">
 <div class="ar_articleAuthor">
  得法前，我是个玩了近
  <a href="http://www.epochtimes.com/gb/tag/%E4%BA%8C%E5%8D%81%E5%B9%B4.html">
   二十年
  </a>
  电子游戏的游戏狂。得法修炼后，游戏虽然走出了我的生活，但是后天养成的负面观念与坏习惯却像大山一样牢牢的固守在那里。而在这一年中，通过坚持不懈的学法，时刻用真、善、忍的法理要求自己，我有了巨大的收获：
 </div>
</div>
<p class="p21">
 <span class="s1">
  首先，我思考问题的方式和角度有了
  <a href="http://www.epochtimes.com/gb/tag/%E5%A4%A9%E7%BF%BB%E5%9C%B0%E8%A6%86.html">
   天翻地覆
  </a>
  的
  <a href="http://www.epochtimes.com/gb/tag/%E5%8F%98%E5%8C%96.html">
   变化
  </a>
  。
 </span>
</p>
<p class="p21">
 <span class="s1">
  人的思想是看不见摸不到的，很多时候是无法控制的，过去看别人，有时思维中会带着一种类似诅咒的想法，而现在我不管看到什么人，即使他对我表现的很恶，我都会真心微笑的看着他，心里祝福着他；脑中出现不好的念头，我就会去排斥，告诉自己那不是我。那个被动、疯狂、极端、满脑子负面思维的我渐渐的消失了，积极、乐观、为他的思想一点一滴的填补着我的思维。
 </span>
</p>
<p class="p21">
 <span class="s1">
  就像师尊在《转法轮
  <span class="s13">
   》
  </span>
  中讲的业力的转化：“白色物质和黑色物质之间有一种转化过程。人与人之间发生矛盾之后，它有一个转化过程。做了好事得到白色物质――德；做了坏事得到黑色物质――业力。”我悟到业消掉了就由德来填补，坏思想没了就由好的来补充。
 </span>
</p>
<p class="p21">
 <span class="s1">
  慢慢的我发现随着思想发生
  <a href="http://www.epochtimes.com/gb/tag/%E5%8F%98%E5%8C%96.html">
   变化
  </a>
  ，我的审美观、爱好、说话方式与生活习惯都在变。在得法前，每到过大年时我妈都会强拉着我看新唐人电视台播放的神韵晚会。那时的我不喜欢看，还很排斥，甚至感觉看不懂。而今年这场神韵晚会我看了三遍都没看够，自己完全融入其中，真心为神韵的整场编排而赞叹，觉的音乐、舞蹈、服装等一切元素只应天上有，为此还感动的流下了眼泪。
 </span>
</p>
<p class="p21">
 <span class="s1">
  之前我所热衷的流行音乐和电视剧都不听不看了，因为一接触就觉的脑袋疼，反而非常喜爱中西方的古典乐曲，每每听到都感觉心神安详、内心宁静。
 </span>
</p>
<p class="p21">
 <span class="s1">
  以前我喜欢穿嘻哈风格的衣服或运动服，而且经常邋里邋遢，自己的内衣、袜子等结婚前由母亲来洗，结婚后就成了妻子的任务，还喜欢将脏衣服乱扔乱放，搞得家人处处为我操心；
 </span>
</p>
<p class="p21">
 <span class="s1">
  而现在我喜欢穿衬衫、西裤和皮鞋，有时还会穿汉服，因为衣着变的简洁、得体，同事们都说我的穿衣品味变高了，即使穿运动鞋也是白色、淡色居多，同事调侃我说，这么白的鞋，是不是你媳妇给你刷的，我告诉他们现在都是我自己在打理啦。
 </span>
</p>
<p class="p21">
 <span class="s1">
  以前上学时我喜欢写连笔字，因为可以偷工减料，还认为自己的草书写的很有风格。现在翻看以前的字，我觉的就像蜘蛛爬一样难看。
 </span>
</p>
<p class="p21">
 <span class="s1">
  后来看到大法弟子制作的动画片《悠游字在》后，我才明白文字是神传给人的，从此开始规范的使用楷书，字体也比以前清秀多了。
 </span>
</p>
<p class="p21">
 <span class="s1">
  讲话则是我最难控制的一方面，因为我家人说话都大嗓门，加上爷爷九十多岁了，有点耳背，小声说话怕他听不清，长此以往我养成了大声说话的习惯。自己又有个爱闲聊的毛病，总喜欢炫耀自己的所知所得，特别是说起社会道德下滑、政权腐败等话题，经常停不下来，时间一长或过于激动就会大脑发热或头痛。
 </span>
</p>
<p class="p21">
 <span class="s1">
  后来我对照大法向内找，明白自己说话时不能把控情绪的问题其实是执着心在作祟，比如显示心、怨恨心、为了达到某个目地而刻意说某些话的心。当明白了问题所在后，我便开始按照《转法轮》中的要求修口，并时刻警示自己注意说话方式，力求做到温文尔雅、和风细雨。
 </span>
</p>
<h4 class="p21">
 <span class="s1">
  <b>
   逆流而上 看淡钱财
  </b>
 </span>
</h4>
<p class="p21">
 <span class="s1">
  二零一八年年末，公司会议室要做设备更新与格局改建，领导安排我负责整个工程。因为是第一次独自承接这样的任务，所以在整个过程中，我的内心出现很多波动：面对金钱时动摇过、面对成就时膨胀过、面对权力高压时也考虑过妥协，但因为内心里时时告诫自己是个大法弟子，我最终通过了这一次次的人心考验。
 </span>
</p>
<p class="p21">
 <span class="s1">
  工程开始时，从整体规划到设备的选择我都一丝不苟的去认真对待，在购买设备前我做了多方面的调研，小到一个地插，都去考虑实地情况与它的后期使用问题。我不但全程和施工方一起干活，还多次主动加班工作，给公司节省了大量资金与时间，领导们都非常满意。
 </span>
</p>
<p class="p21">
 <span class="s1">
  一天，领导突然找到我，说让我找人施工，这件事使我直面利益的诱惑。若在修炼前，我一定一口答应，那时我曾想通过给人介绍活儿从中吃回扣，可现在修炼大法了，立刻意识到不能再那样做了。于是我找到领导说：“这种涉及到钱的事情，我是不会动一分的，但古人言，君子不立危墙之下，我希望有公正透明的方法，不要让我独揽权力。”领导听后，决定让我负责技术和施工，资金方面走招标流程。
 </span>
</p>
<p class="p21">
 <span class="s1">
  在招标时，另一位领导授意，让我直接提供设备做个报价，他自己找两家陪标的走走流程就行了。我听后拒绝了，陪标其实就是造假，这种内定的方式我无法接受。
 </span>
</p>
<p class="p21">
 <span class="s1">
  后来在与施工方接触时，两个供应商都主动要给我回扣，都被我拒绝，我善意的和他们说：“钱我不要，降低一下设备的成本就行了。”可能是因为我的真诚打动了他们，施工方直接把底价给了我，使得最终采买的设备多数是平价或低于市场价的。后来我与施工方成为了很好的朋友，他们也通过我了解了一些大法真相。
 </span>
</p>
<p class="p21">
 <span class="s1">
  类似的金钱诱惑还发生过不少次，比如：给我所发的工资与工龄不符，每个月会多拿几百块钱。我找到领导私下谈了这事，他说：“我知道你工作付出的多，工资却最低，我找大领导说过给你涨工资的事，他没给答复，我就私下把你的工龄增加了五年作为补偿。”
 </span>
</p>
<p class="p21">
 <span class="s1">
  我站在他的角度上善意的拒绝了，我说：“如果公司领导看到我的工龄与实际情况不符，会影响到您的工作。您给我改回来吧，而且我会把之前多发的钱退回给公司。”他感受到我的善意，微笑着说：“之前的钱不用退了，工龄回头给你改回来。”当他笑着送我出门时，我从他的眼神中看到了一种信任，那眼神我无法形容，但我知道自己做对了！
 </span>
</p>
<h4 class="p21">
 <span class="s1">
  <b>
   善待家人得称赞 感化世人明真相
  </b>
 </span>
</h4>
<p class="p21">
 <span class="s1">
  我们每个大法弟子都有自己的家人朋友。我家是个四世同堂的大家庭，如今我修炼大法了，知道要面对这样复杂的家庭环境，必须随时用大法修正自己。正所谓 “中土难生”，复杂的环境让我修去了更多的人心，也让更多的世人看到了大法弟子的与众不同。
 </span>
</p>
<p class="p21">
 <span class="s1">
  前不久，八十九岁的奶奶突然感冒发烧，姑姑和大爷们都来看望。这群亲戚中除了我二姑家以外，都已经明白了大法真相。因二姑父以前是邪党的村书记，被邪党严重洗脑，他的家人也受到了影响，都排斥大法。
 </span>
</p>
<p class="p21">
 <span class="s1">
  这次我二姑来我家看望奶奶，看到我这个当孙子的一直陪着奶奶上厕所、帮她洗漱，又听到奶奶感叹说自己命好，有孙子和孙媳妇伺候着，说我们还经常给她和爷爷洗澡洗脚等。看得出二姑受了触动。吃饭时，奶奶想将吃不了的饭拿给爷爷，但又犹豫了，因她的感冒还没痊愈，饭又有些埋汰（东北话，弄脏了）。作为修炼人不怕这些，我就把饭拿过来吃了。二姑看到后说：“我从来都不吃别人的剩饭，你能做到这样真是难得。”奶奶听后又说我平时也是这样的。
 </span>
</p>
<p class="p21">
 <span class="s1">
  师尊在《转法轮》中讲：“当然，我们在常人社会中修炼，孝敬父母、管教孩子都是应该的，在各种环境中都得对别人好，与人为善，何况你的亲人。对谁也一样，对父母、对儿女都好，处处考虑别人，这个心就不是自私的了，都是慈善之心，是慈悲。”
 </span>
</p>
<p class="p21">
 <span class="s1">
  二姑看到我们善待老人的种种，联想起自己儿媳因为钱对她无理谩骂的事，心中很是不平。我就用师尊所讲的法理与她交流，告诉她说：“您的儿媳妇要是修炼大法就不会发生这样的事情了，因为我修炼之前也是因为钱、房子和父亲闹得特别僵，曾经把父亲气得睡不着觉，自己也为此哭了好多次，甚至想到离家出走。我修法轮大法后，我和父亲间的矛盾全化解了，我自己的钱也全交给父亲帮忙打理。”
 </span>
</p>
<p class="p21">
 <span class="s1">
  二姑听后，语气中对这种和睦的家庭氛围充满了期盼。后来她观看了法轮功真相视频，我又把去年投给明慧网的文章亲口读给她听。她听后竟然哭了。我明白那是宇宙特性真、善、忍触动了她的本性，让她对大法彻底认可了。二姑回家时带走了多本真相资料。
 </span>
</p>
<p class="p21">
 <span class="s1">
  再一个故事是上周发生的。我经常去同一个理发店理发，因为店主很善良也很健谈，我就在每次理发时，根据她的接受能力讲一些中国传统文化中善恶有报的故事。她接触过我和我的家人后，感受到了大法弟子身上展现出的善良、纯真与忍让，看到我们四世同堂还能如此和睦，而且全家人都健康、祖辈长寿，她从心底认可了大法，由原来排斥真相到后来和她女儿一起真心做了“三退”。
 </span>
</p>
<p class="p21">
 <span class="s1">
  最近她的丈夫和她的娘家人因为钱财问题发生了纠纷，使得他们俩人闹得要离婚。每当想起丈夫说要与她“同归于尽”的话，这位店主就直淌眼泪。我之前通过与她接触得知，她的命很苦，走到今天实属不易，能看得出迷在难中的她已经达到了承受能力的极限。我知道只有大法才能为她破迷，改变她的人生，就把师尊的《广州讲法》录音送给她，并告诉她一定不要带着任何观念去听。
 </span>
</p>
<p class="p21">
 <span class="s1">
  十多天以后，我专程去理发店看看她听法的情况，结果一进屋就看到她满脸笑开了花。
 </span>
</p>
<p class="p21">
 <span class="s1">
  我问：“听讲法了吗？”她说已经听了一遍了，并笑着说，刚开始的时候根本没听懂师尊讲的是什么，但是后来越听越明白，发现自己在生活中遇到的很多问题在师尊的法中都讲到了。说她能用师尊所讲的法去衡量遇到的矛盾了。我听后就知道她与丈夫一定和好了，她笑逐颜开的模样与之前凄切的面孔简直判若两人。
 </span>
</p>
<p class="p21">
 <span class="s1">
  是师尊给她改变了人生道路。那天我们聊的特开心，我叮嘱她一定要多听几遍讲法，按法中讲的去做，生活中的一切问题自会迎刃而解。
 </span>
</p>
<p class="p21">
 <span class="s1">
  借世界法轮大法日这个光辉的日子，愿更多的世人明白大法的美好！#
 </span>
</p>
