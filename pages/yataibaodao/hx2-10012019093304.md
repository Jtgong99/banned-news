 <p>
  中国国家主席习近平在中共建政七十周年的庆典上再提“和平统一、一国两制”。台湾主管两岸事务的大陆委员会强调，“中华民国”从来不是中华人民共和国的一部分，且台湾已实践民主体制，奉劝对岸及早还政于民。
  <br/>
  <br/>
  中国政府10月1号上午在北京举行建政70周年阅兵式，习近平在阅兵前讲话时提到，“在前进征程上，我们要坚持‘和平统一、一国两制’的方针，保持香港、澳门长期繁荣稳定，推动海峡两岸关系和平发展，团结全体中华儿女，继续为实现祖国完全统一而奋斗。”
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  对习近平重提“和平统一、一国两制”方针，台湾朝野政党都表示无法接受。
  <br/>
  <br/>
  <b>
   陆委会正告习近平：台湾作为主权国家超过一世纪
   <br/>
  </b>
  <br/>
  台湾的陆委会发言人邱垂正说：“中华民国作为主权国家，迄今已经108年，并在台湾实践民主体制，繁荣发展。中共当局必须认清国际现实，台湾从来不是中华人民共和国成立以来，任何时刻的一部分；一国两制不是两岸关系的处理方案，更不适用于台湾，台湾绝对不会接受。”
 </p>
 <p>
 </p>

 <p>
  <br/>
  陆委会指出，中国共产党坚持一党专政70年，治理理念违反民主、自由与人权价值，造成中国大陆发展充满风险与挑战。邱垂正还奉劝对岸在此时此刻，应反躬自省，及早推动民主政治改革、还政于民。
  <br/>
  <br/>
  <b>
   民进党：香港回归中国22年就被国家暴力对待向外求救
  </b>
  <br/>
  <br/>
  民主进步党发声明指出，习主席一再片面要台湾接受一国两制、和平统一，却未曾顾及台湾2300万人民的意愿，台湾已经建立自己的民主制度，台湾人民无法接受。
  <br/>
  <br/>
  民进党发言人李明俐强调：“香港主权移交中国22周年，一国两制已尽失民心，香港大量人民走上街头争取权利，却被国家还以暴力相向，被迫奔走国际寻求支援，不知这是什么样的一国两制执政成果？呼吁中国不要再一厢情愿提出一国两制的药单，过去70年台湾人不需要，未来也不会需要。”
 </p>

  <b>
   国民党：坚决不接受一国两制 习近平未体察台湾民意
  </b>
  <br/>
  <br/>
  在野的国民党发言人洪于茜说：“针对习近平近日发表谈话宣称坚持‘和平统一，一国两制’的方针，国民党感到遗憾的是，习近平似乎没有体察到这九个月来两岸形势的变化 ，国民党坚决不接受一国两制，我们始终坚持一中各表的九二共识。”
  <br/>
  <br/>
  <b>
   韩国瑜：捍卫中华民国主权 坚定台湾自由民主制度
  </b>
  <br/>
  <br/>
  国民党2020总统参选人韩国瑜竞选办公室，就习近平谈话的回应表示，韩市长的两岸主张，一直以来都没有变过，就是捍卫中华民国主权、坚定台湾的自由民主制度，以及维护台湾人民的生活方式。大多民众对于两岸的态度其实很简单，就是希望能两岸维持和平稳定。
 </p>

  在台湾朝野政党同声表态捍卫主权，不接受一国两制的同时，台湾的统派团体中华统一促进党却申请举办庆祝中华人民共和国国庆游行，遭警方驳回后，仍试图号召约500人，以“庙会”民俗活动，在台北车站广场搭建舞台，采取舞龙舞狮、昆曲、国乐等表演形式“遮掩”庆祝。警方出动包括霹雳小组共约千名警力，严阵以待。
  <br/>
  <br/>
  <b>
   统促党违法自办“十一国庆”遭警强制驱离
  </b>
  <br/>
  <br/>
  统促党总裁“白狼”张安乐抵达时，突然有五星红旗跟着大进场，张安乐对五星旗致敬、率众高唱义勇军进行曲。
 </p>
 <p>
  张安乐对党员致词说，为什么要在台湾庆祝中华人民共和国的国庆？因为中华人民共和国要带领中华民族走向万世太平的时代。自从一九七八年第十一届三中全会，共产党拉开改革开放的序幕，从此中国人站起来了。
 </p>

  媒体询问张安乐，对香港歌手何韵诗泼漆的胡姓男子，是否为统促党员？张安乐起先没有正面回应，最后才说胡男确实是党员，但已经很久没有参加统促党活动。张安乐还说，他不赞成泼漆行为，并指统促党人多势众，不需要用泼漆表达立场。胡姓男子涉嫌对何韵诗的泼漆案，警方正在朝组织暴力犯罪方向侦办。
  <br/>
  <br/>
  警方认定统促党的集会属违法政治性集会，中正一分局长沈炳信接连四次警告驱离，命令解散，张安乐与党员停留十余分钟后，下台走人，被台湾媒体形容为“落荒而逃”。
