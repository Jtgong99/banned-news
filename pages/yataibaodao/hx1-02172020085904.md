 <p>
  台湾15日惊传新冠肺炎（武汉肺炎）首例死亡个案。死者为61岁的出租车司机，无境外旅游史，20多天前曾载过浙江台商，可能是感染源。死亡个案的弟弟、妈妈、外甥女婿共3名家人也都陆续确诊。目前台湾新型冠状病毒肺炎案例总共有22例。
  <br/>
  <br/>
  台湾中央流行疫情中心16日晚间宣布台湾新增2例武汉肺炎个案，为一对兄弟，其中一人（第19例个案）15日死亡，为台湾首例死亡个案。17日晚间台湾疫情指挥官宣布新增2例确诊，为该名死亡个案同住的母亲，以及曾在1月27日一同聚餐的外甥女婿。一家共4口确诊。
  <br/>
  <br/>
  中央流行疫情中心指挥官陈时中16日说明台湾首例本土死亡个案：“61岁男性，有B型肝炎、糖尿病史，无出国旅游史，1月27日出现咳嗽症状，2月3日因呼吸急促走医院，诊断为肺炎，当日收治于负压加护病房治疗，2月15日晚间因肺炎合并败血症死亡，遗体依家属同意，依法定传染病规定火化，并保留部分检体供进一步检验。”
  <br/>
  <br/>
  该名死亡个案从出现咳嗽症状到死亡仅短短20天，也是中国境外罕见数例的死亡个案。台湾武汉肺炎确诊个案至2月17日累积至22例。

  陈时中说，该死亡个案没有旅游史，原本不在采检范围内，疾管署考量2月看到新加坡很多个案，50%也没有旅游史，认为以旅游史判定是否采检可能有疏漏，因此2月7日进行采检后发现。
  <br/>
  <br/>
  陈时中说：“2月3号案19（死亡个案）就入院，在诊所做快筛的时候，是流感的阳性，进院虽然身体状况非常不好，但是医院也非常地注意，所以把他送到负压隔离病房里面，在那里采验痰，都是针对流感，不过那时得到流感都是阴性。”

   <br/>
   阴性带毒？防疫中心：将进一步验血

  追踪该名死亡个案的感染源，陈时中表示，锁定一名从浙江返台的台商，1月22日时，他搭乘死亡个案驾驶的白牌计程车回家，搭车当天咳得非常厉害。至于为什么这名台商返台有咳嗽却没有被通报？因为当时浙江并不包含在台湾的疫情通报定义里。

  陈时中：“为什么抓到这样一个（浙江）台商，最主要他的主述，有跟他近距离，坐他的车子，回来的时候在车上咳得很厉害，追踪他的健保医疗史，发现他连续做呼吸道治疗，所以我们认为他的可能性高非常高，但是他初步检查为阴性，没有办法做直接证明。”
  <br/>
  <br/>
  防疫中心进行直播到这段时，有网友听到疑似死亡个案的感染源采检却为阴性，有人留言惊呼：“阴性带毒？”
  <br/>
  <br/>
  不过陈时中解释，如果1月22日到2月17号仍维持有病毒，那整体发病程序、到感染的程序就非常地长，不太可能。当局将进一步对这位台商进行验血检测，看是否有抗体？
  <br/>
  <br/>
  陈时中表示，现掌握到的79名密切接触者中，已完成73名采检，其中60名检验皆为阴性、3名阳性（家人），其余尚在检验中。目前采检医护人员皆为阴性。
  <br/>
  <b>
   <br/>
   包括副总统在内的台湾防疫专家认为非“社区感染”

 <div>
  16日陈时中被问及是否为“社区感染”首例时，曾一度表示“算是”，震惊台湾社会，学者何美乡呼吁陈时中收回“社区感染”的说法。稍晚陈时中改口称仅是社区内零星个案，并指台湾只符合四项“社区传播定义”中的一项。
  <br/>
  <br/>
  流行病学研究出身的台湾副总统陈建仁稍后则在脸书强调台湾并未进入“社区传播”的状况，防疫等级不必升级。
  <br/>
  <br/>
  “社区传播”4征兆包括“确诊病例无法找到传染来源”、“本地感染个案数已远超过境外移入感染个案数”、“已经出现持续性的传播链”、“有广泛发生的群聚感染事件”。陈建仁说，疫情中心正检视第一征兆，积极调查第19例的传染来源，至于其他三征兆在台湾都不存在。
  <br/>
  <br/>
  台湾大学公共卫生学院院长詹长权17日在该校主持新型冠状病毒疫情记者会上也表示：“就目前资讯来看，尚未达社区感染，民众不必恐慌，给政府时间厘清。”
  <br/>
  <br/>
  台大公卫学院副院长陈秀熙分析，该例不幸的死亡案例，本身是多重慢性病患，且早在1月27日就发生症状，但直到2月3日才就医，如果是社区感染，应该早就爆发，目前仅查出传染给亲人，仍在家户感染范围。
  <br/>
  <br/>
  这名死亡个案入院被诊断“一般肺炎”，外界担忧会否造成院内感染？陈秀熙指出，死亡个案接触的医护检验结果目前全部都呈现阴性，代表院方做了防疫的全面准备。不然依照这样的途径，死亡个案应该是个“超级传染者”。
  <br/>
  <br/>
  <b>
   台学者预警 台湾潜在7千感染数 防疫焦点应转到日韩
  </b>
  <br/>
  <br/>
  陈秀熙另指出，约500万人在中国武汉封城前离开，统计有7,515人到台湾，推估潜在感染武汉肺炎者为505人，但根据文献分析，台湾现行发烧检疫可排除78%的感染者，所以估计有111人会是无法侦测的潜在感染者。不过武汉封城已有3个星期，这些在台的潜在感染者或许都已脱离风险，对台湾的威胁性减弱，但这段期间是否有将病毒传给其他人，将关系到台湾的疫情是否能持续稳定控制。
  <br/>
  <br/>
  陈秀熙提醒，如果将相关统计转化到亚洲一到三级疫区赴台及旅游人数，只用现在目前的发烧筛检，可能会有7千个个案的感染人数，潜伏在台湾的社区。如果把防疫的警戒线拉高，这些感染的个案自然就会减少。
  <br/>
  <br/>
  陈秀熙强调，他提出7千人，不在引起恐慌，而是预警台湾要将防疫焦点转到日本、韩国、新加坡、东南亚国家，尤其台湾人每月到日本有14万游客，游量相当大，占了前往东南亚人数几乎一半，目前台湾已将日本列为传染病疫区，对自日本入境旅客扩大筛检条件，问题在台湾人赴日本的风险提高。
  <br/>
  <br/>
  陈秀熙：“日本因为在他们警戒线上，不如台湾有中港澳的入境限制，所以日本相对而言，中国大陆从武汉以及其他县市封城之前，到日本的游客很多，造成日本非常危险，也呼吁台湾民众到日本要特别的小心，如果有可能牺牲大家到日本观光的机会，特别这一两个月不要到日本，特别几个城市，东京、横滨、北海道，都是目前有很多社区流行爆发的事实，希望大家在日本旅游要多小心。”
  <br/>
  <br/>
  台大医院医师苏大城呼吁，计程车司机载客最好摇下车窗，避免空气密闭循环，且加强擦拭车内把手、座椅等，降低传染风险。
  <br/>
  <br/>
  台学者：面对防疫不能温情主义 应跟着美国疾管署政策走
  <br/>
  <br/>
  对于武汉台商、陆配子女撤回台湾，以及在日本搭乘钻石公主号返台到底要不要再隔离检疫14天等争议，台湾大学公共卫生学院院长詹长权强调，面对防疫应该以科学、理性讨论，不能感情用事、不能温情主义，“没有病的不能让他感染、已经病发的留在当地治疗”，且过程中的隔离检疫都至少得比照美国的班机。
  <br/>
  <br/>
  （自由亚洲电台记者：夏小华 /  责编：许书婷、安克）
