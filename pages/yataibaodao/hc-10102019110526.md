 <p>
  美国篮球职业联赛(NBA)火箭队总经理莫雷上周发表挺港言论而引发的美中两国纷争又有新的发展。有迹象表明，中国政府正试图对此事降温，而美国朝野则在持续发声，在声援香港民众的同时，呼吁NBA继续坚守价值观。
  <br/>
  <br/>
  火箭队总经理达雷尔•莫雷(Daryl Morey)上周发推支持香港示威者，再加上NBA总裁亚当•萧华(Adam Silver)出面挺莫雷和言论自由，迅速煽动了中国公众的爱国情绪。

  就在大家纷纷猜测这场风波究竟会闹多大时，中方近日却释出了一些收敛信号，试图给此事降温。
  <br/>
  <br/>
  《纽约时报》周四引述三名记者表示，中国官媒的编辑已经告知他们不要再炒作NBA事件，以免造成过于激烈的后果。随后，这起事件就淡出了微博热搜榜前十。
  <br/>
  <br/>
  报道还引述《环球时报》总编胡锡进说，他认为NBA 事件会逐渐平息，而他的报纸不会再对此进行炒作。同时，他希望美方不要作出让事态继续升级的动作。
  <br/>
  <br/>
  美国独立政治学者顾为群认为，当局放任此事迅速扩大后、又突然念起了紧箍咒，很可能与周四开始的新 一轮美中经贸高级别磋商有关。
  <br/>
  <br/>
  “我觉得一大因素就是美中贸易谈判。在此期间，中方还是希望制造一个稍微和缓些的环境，以便促进双方达成协议。”
  <br/>
  <br/>
  让不少中国球迷欣慰的是，NBA湖人队和篮网队的一场季前赛周四在上海如期举行。不过，两位NBA美国直播解说员在比赛期间并没有提到包括香港在内的政治话题。但在整场比赛期间，直播屏幕下方一直在滚动播放NBA总裁萧华捍卫言论自由的声明。
  <br/>
  <br/>
  路透社报道，这场比赛全场坐满。几百名中国球迷在场内挥舞着五星红旗，还有一些群众聚集在体育馆外，抗议莫雷和萧华的言论。此前，中国央视和腾讯宣布将立即暂停NBA季前赛(中国赛)的转播安排。
  <br/>
  <br/>
  <b>

  尽管北京当局似乎在刻意淡化此事，美国朝野却并没有就此罢休的迹象。
  <br/>
  <br/>
  美国国会两党的8名议员周三致信萧华，敦促NBA暂停在中国的所有活动，直到中国媒体和赞助商停止抵制NBA和火箭队。
  <br/>
  <br/>
  这封公开信中写道:“当捍卫言论自由对你来说毫发无伤时，这是件易事。但当你的利润受到威胁时，含糊其辞就是背叛美国的基本价值观。”
  <br/>
  <br/>
  同日，美国总统特朗普也就此事首次发表了看法。他说，NBA需要自行解决这起事件，他们知道自己在做什么。但他点名批评了NBA勇士队主教练科尔(Steve Kerr)和马刺队主教练波波维奇(Gregg Popovich)，指责他们只顾讨好中国，却经常批评美国。特朗普说， 这让他非常难过。但他并没有对中国就香港问题向 NBA施压一事直接表态。
  <br/>
  <br/>
  除此之外，NBA事件还引发了美国民间的不满。华盛顿人权组织“共产主义受难者基金会”( Victims of Communism Memorial Foundation)周三在当地的一座体育场入口处发放了几百件印有“光复香港，时代革 命”的T恤。当晚，NBA奇才队对阵了CBA广州龙狮队。

  基金会执行董事马里恩•史密斯(Marion Smith)周四对本台记者透露，他们当场送出了500多件T恤。他还说 ，全美各地的不少球迷表示他们也想穿上它去看其他比赛，而基金会当天已经开始寄送这些T恤了。
  <br/>
  <br/>
  “一些到场球迷领取并穿上了我们的T恤，还有些计划在场内或场外举牌的抗议者也欣然拿走了这些T恤。”
  <br/>
  <br/>
  无独有偶，香港《明报》报道，美国有网民周二发起众筹活动，希望定做1万多件印有“支持香港”(Stand With Hong Kong)的T恤，计划在本月22日NBA常规赛首战湖人队对阵快艇队时在场外派发。他们最终筹得4万多美元，远远超出了2万美元的筹款目标。
  <br/>
  <br/>
  史密斯表示，萧华应该引咎辞职。他解释说，NBA起初对莫雷施压，让他删除了挺港推文，还为此道歉。尽管萧华随后看似守住了底线，却已无济于事。
  <br/>
  <br/>
  “这来得太少太迟了。NBA当前的处境并不是一条推文造成的，而是NBA决定对中国采取的立场所致，这让他们赚了很多钱。”
  <br/>
  <br/>
  据悉，NBA目前在中国的市场估值超过40亿美元。
