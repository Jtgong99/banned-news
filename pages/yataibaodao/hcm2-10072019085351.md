 <p>
  台湾第一学府台湾大学7日上午有陆客闯进校园，毁损声援香港的连侬墙海报，台大学生会随即报警并提告。负责的大安分局依法将现行犯逮捕，并追查幕后是否有人策划及教唆。
  <br/>
  <br/>
  一对穿着休闲服的男女周一(7日)闯入台大第一活动中心，戴着眼镜的男子伸手撕毁连侬墙上展示的海报，整面墙瞬间完全清空。一旁的女伴插着手在旁边观看并未动手，但是，当两人抱着撕毁的海报准备离开时，遭到台大学生制止。
  <br/>
  <br/>
  台大学生：“诶，你怎么可以这样？”
  <br/>
  撕连侬墙陆客：“不要拍了！不要拍了！”
  <br/>
  台大学生：“如果你觉得自己做的事情是对的，就不要怕被拍。”
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  台大学生会指出，学生会设置的连侬墙，遭到两位中国观光客破坏。中国观光客将连侬墙展板上展示的海报纸张撕毁之后，本欲携带逃离现场。所幸现场目击同学将整段破坏连侬墙的过程以影片纪录，同时也协助校方报警处理。
 </p>
 <p>
 </p>
 <p>

  <br/>
  基于连侬墙的空间系由学生会借用并与校内学生一同展设，学生会确认会以法律诉讼追究中国观光客的行为，决不息讼宁人。
  <br/>
  <br/>
  台大学生会会长凃峻清：“台大学生会经过活动场地申请核可，申请放置香港连侬墙，时间是九月中到十月中，今天大约十一点钟时，我接到活动中心通知表示，现场连侬墙遭到破坏，我到的时候，现行犯已经被带到派出所，到派出所之后，确定由台大学生会提出毁损罪告诉。”
  <br/>
  <br/>
  两位中国大陆观光客被带至台大附近的罗斯福路派出所侦讯，所属的大安分局副分局长陈少旭7日受访时表示，据调查，涉案的 2 名陆客是来台自由行，据涉案的 30 岁李姓男子供称，是来台观光刚好路过看到连侬墙，才临时随机起意撕毁，但警方对此说法仍持保留态度。
  <br/>
  <br/>
  <b>
   台警方追查幕后教唆者
  </b>
 </p>
 <p>
  大安分局副分局长陈少旭：“现行犯依法逮捕究办，对于中国观光客说词予以保留，本分局会持续追查幕后是否有教唆的行为。”
 </p>

 </p>

  <br/>
  据台媒自由时报报导，毁损台大连侬墙的李姓男子经台北市大安警分局初讯后，被依现行犯身分移送台北地检署复讯，移民署也派出3名官员至北检等候；据报，这名陆客已违反“大陆地区人民来台从事观光活动许可办法”，停留许可将被移民署撤销并驱逐出境，列为“不受欢迎人物”，管制5年内不得再来台。
  <br/>
  <br/>
  台大校方则发表声明表示，尊重言论自由、依法行政，因涉案人员为校外人士，依法委请警方处理。
 </p>
 <p>
  校方非常重视此议题，校长也很关心，除加强校安通报工作，驻警队和活动中心也早已共同加强连侬墙的巡查。同时学校非常关切港生及陆生的状况，加强维护校区安宁勤务，并保障学生的安全。
 </p>
 <p>
 </p>

  <br/>
  除了台大之外，台湾艺术大学2日也发生两名陆生在校内破坏撕毁连侬墙事件。香港学生张俊豪6日受访时表示，一群港生及台生透过学校程序尝试跟两位涉事陆生沟通不果，因此决定赴警察局提告毁损罪。
  <br/>
  <br/>
  张俊豪说，连侬墙是每个人都能发表意见，甚至陆生也能表达，不希望台湾言论自由被破坏。“不仅仅是在香港有打压言论自由的白色恐怖，我们也能看到在无形中，也把暴力行为从香港移转到台湾。我们不希望在台湾拥有自由民主的国家，发生这样侵犯言论自由的举动。”
