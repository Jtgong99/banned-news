 <p>
  【新唐人北京时间2019年05月10日讯】下面我们来关注一下中国大陆各地民众维权的情况。
  <br/>
  上海王扣玛被截访 染病入院
  <br/>
  上海残疾
  <a href="https://www.ntdtv.com/gb/维权人士.htm">
   维权人士
  </a>
  王扣玛，4月25号应医生的复查通知，与妻子一起进京复查，结果被截访人员截回上海。之后，妻子姚敏华被强制带走，留下生活不能自理的王扣玛独自在家。因无人照顾，王扣玛受寒感冒、持续发烧，5月5号被送到医院时，医生诊断为“细菌性肺炎”，病情严重，需住院治疗。
 </p>
 <p>
  福建政府征迁 数百黑衣
  <a href="https://www.ntdtv.com/gb/保安.htm">
   保安
  </a>
  闯村破坏
  <br/>
  福建福州市晋安区鼓山镇政府以河道改造、拆除违建的名义在埠兴村与牛山村征迁，在未达成补偿协议的情况下，4月28号，派出二三百名黑衣
  <a href="https://www.ntdtv.com/gb/保安.htm">
   保安
  </a>
  闯入村庄进行破坏，把村民还在居住的房屋砸毁。
 </p>
 <p>
  湖北
  <a href="https://www.ntdtv.com/gb/维权人士.htm">
   维权人士
  </a>
  尹旭安北京看病 被拦截
  <br/>
  湖北大冶市维权人士尹旭安5月8号前往北京治病，但刚到北京西站就被大冶警察拦截，不准他进北京城，之后他被带到派出所问讯，又被威胁要送
  <a href="https://www.ntdtv.com/gb/马家楼.htm">
   马家楼
  </a>
  黑监狱关押。目前，仍然遭大冶警方贴身跟踪。尹旭安担心自己随时会被绑架遣返。
 </p>
 <p>
  四川维权人士刘正有声援良心犯 被传唤
  <br/>
  四川省自贡市维权人士刘正有，因近期在网上发表声援良心犯陈云飞、唐荆陵、
  <a href="https://www.ntdtv.com/gb/王全璋.htm">
   王全璋
  </a>
  的言论，特别是向山东临沂监狱打电话声援王全璋，5月6号遭到辖区国保传唤，警方警告他如果继续声援这些良心犯，就要对他严肃处理。
 </p>
