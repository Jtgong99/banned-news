 <p>
  香港反送中运动持续将近100天之际，香港歌手何韵诗和台湾独立音乐人在台北召开记者会，呼吁台湾人参加“撑港、反极权—9/29港台大游行”。何韵诗提到，香港人申请9月15日游行再被警方禁止，香港人失去上街表达诉求的自由，台湾人一定要非常珍惜还享有的自由。
  <br/>
  <br/>
  “撑港、反极权”、“光复香港、时代革命”
  <br/>
  <br/>
  港台音乐人12号齐声呼吁参加929台港大游行。
 </p>
 <p>
 </p>
 <p>

 <p>
 </p>
 <p>
  <br/>
  <br/>
  香港歌手何韵诗说：“快要进入第一百天的抗争，我们面对的是极权政府，香港政府后面的中共政府一直打压香港人，而且这种手法已经走到香港人无路可退的地步。香港年轻人每一天面对警暴的状况，不断被打、不断地被逮捕，没有任何申诉的门路。”
 </p>
 <p>
 </p>
 <p>
  <iframe frameborder="0" height="476" scrolling="no" src="https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRFAChinese%2Fvideos%2F420837758536692%2F&amp;show_text=0&amp;width=622" width="622">
  </iframe>
 </p>
 <p>
  <br/>
  <br/>
  何韵诗认为，这不只是香港人独自面对一场抗争，中国政府这种打压，已经走到世界各地，到台湾、澳大利亚、美国，各种各样的自我审查，“香港只是走在这条路上的前沿”。
  <br/>
  <br/>
  何韵诗爆料，她在澳大利亚一场座谈，没来由“被取消”。她没有想到离开中国这么远，民主自由的国度也受影响。
  <br/>
  <br/>
  对于香港妇女会代表近日在联合国人权理事会议发言，抹黑香港抗争者的示威行动，何韵诗痛斥“两个权贵”替“中共机器”到联合国里散播“假讯息”。她认为，显然中共这个维稳机器，已经全力在世界各地发动起来。
  <br/>
  <br/>
  何韵诗说，她到澳大利亚争取国际声援，被亲中媒体指责“潜逃”，另被抹黑说她有外国势力资助、说她是越南人等各式奇怪指控，中共惯于将各种罪名安在他想打压的人上。
 </p>
 <p>
 </p>

  <b>
   何韵诗：畏惧发声的人
  </b>
  <b>
  </b>
  <b>
   失去最重要的自由
  </b>
  <br/>
  <br/>
  提到“自我审查”和“白色恐怖”，何韵诗说：“现在演艺圈绝大部份人不敢发声，大家都活在一个非常大的恐惧里面。很多的艺人已经失去他们的自由，很多人觉得我们失去了很多，我觉得他们才是真正失去了最重要的东西的一群人。”
 </p>
 <p>
  至于怎么样去对抗极权政府的压迫和白色恐怖？何韵诗认为，要靠人民一起站出来，她会争取在9月29日到台湾参加撑香港游行。她认为，港台人民之间可以有更密切的，为争取自由民主人权的相互声援：“台湾，在你们还享有自由人权状况下，一定要非常珍惜这种自由。”
  <br/>
  <br/>
  曾创作太阳花运动神曲《岛屿天光》的灭火器乐团主唱杨大正说：“我这几个月一直在关注香港的行动，其实，很多的震撼、很多的学习、很多的心痛，包括已经有愈来愈多的生命殒落。我们总是希望香港可以好好的、平平安安的。”
 </p>

  <br/>
  <br/>
  台湾原住民歌手巴奈听何韵诗谈香港反送中运动的近况，她对比台湾原住民人权也是一步步抗争得来，激动落泪。
  <br/>
  <br/>
  巴奈哽咽地说：“我一点一点地把那个人权活在我的身上，可是现在香港的朋友，是一天一天在失去他们的尊严。”

 <p>
  何韵诗看到巴奈因为香港泪流不止，反过来握住巴奈的手，鼓舞“不要绝望”。
  <br/>
  <br/>
  何韵诗说：“我觉得我们不用绝望，香港人其实到目前，我们还是非常坚定，虽然每一天确实非常困难，看着这么多孩子被打到头破血流，在这个时候，才是我们真的要连结在一起，台湾、香港，甚至中国大陆里面有很多民众，其实是明白事理的。”
  <br/>
  <br/>
  <b>
   “
  </b>
  <b>
   香港国歌
  </b>
  <b>
   ”

  <b>
   何韵诗：脑袋的创意不能被删除打压
  </b>
  <br/>
  <br/>
  何韵诗说，尽管像她这个唱歌的，可能擅长的就是唱歌，但有很多不同类型的人，像学生、妈妈、社工、医师、护士，都站出来。“香港这次最美丽的地方，是有很多你没有想像 的人都走出来了，虽然我们要经历22年才觉醒，希望香港人坚定的能量，真的可以影响台湾。”
  <br/>
  <br/>
  最近香港年轻人传唱各种版本得《愿荣光归香港》，有网友还在何韵诗脸书上留言，希望她接下来在伦敦、纽约的演唱会，也能唱这首香港人心中的“国歌”—《愿荣光归香港》。
 </p>
 <p>
 </p>

 <p>
  对此，何韵诗说：“那可能就看看怎么样演绎吧！这个当然我是最支持艺术上的表达，因为在这么严峻的一个状况，当他们可以抹黑、去打压、去删除大家的声音的时候，我们脑袋的创意他们是不可能删除、阻挡的。尤其在互联网的时代，一首歌、一个艺术品出来，他们无法阻挡他们的散播，他们很聪明，新一代用各种方法把声音纪录下来。”
  <br/>
  <br/>
  何韵诗认为，音乐和艺术在抗争中占有非常非常重要的地位，这三个月的反送中行动，出现很多不同创意，年轻人非常厉害，她也是被感动、被打动的一个，在过去，没有见过香港人想创作属于香港人自己的所谓的“国歌”。
  <br/>
  <br/>
  何韵诗提到，6月中旬她也在街上一起“吃”过催泪瓦斯，她观察，7月1号抗争者为何会走进香港立法会，那是因为绝望，因为觉得如果坐在那里枯等，政府永远不会有回应的，已经有100万、200万人上街，仍看不到未来。
 </p>

  何韵诗说，7月1号那天开始，是改变运动的分水岭，之前坚持不要有任何暴力、激烈行为的抗争者，在那一天终于理解，面对极权政府，以温和及进取的方法，混在一起进行，能发挥大效用，当时柱子上就有个标语写着：“是你教会我和平游行是无用的！”
  <br/>
  <br/>
  何韵诗以此认为，香港年轻抗争者讲的“揽炒”，是一种策略，不是真的要把香港推到极端的境地，也不是悲观地要自杀，而是一种积极、不断进化的心态。

  提到9月17日将与香港众志秘书长黄之锋在美国国会出席《香港人权与民主法案》的听证会，何韵诗说，她要控诉这三个月在香港发生的事，以及香港主权移交中国22年来，曝露出“一国两制”在香港没有被中共和港府尊重、落实的真相。
  <br/>
  <br/>
  一名在台港生、香港边城青年代表，在记者会上提到，很多台湾同学看到香港特首林郑月娥宣布“撤回”《逃犯条例》草案，不理解为何香港人还要抗争？事实上，林郑对另四大诉求至今没有回应，港人坚持“五大诉求”缺一不可，希望透过929游行，让外界更了解，今日反送中运动的目标，已经提升到“光复香港、时代革命”。
