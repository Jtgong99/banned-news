 <p>
  美国白宫网页出现有民众发起“购买台湾”的提案连署，倡议美国买下台湾比买格陵兰岛更符合美国利益。台湾外交部强调台湾“非卖品”。那么，台湾老百姓又如何看待这个提案呢？
  <br/>
  <br/>
  美国总统特朗普日前想买下丹麦在大西洋的自治领土格陵兰（Greenland），遭到丹麦总理拒绝。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  “美国买下格陵兰”的议题还未退烧，8月27日，另有民众在美国白宫政策请愿网站“我们人民”（We the People）发出一则“购买台湾”的连署书。
 </p>

  </b>
  <br/>
  <br/>
  提案网友“C.C.”指出，台湾具有优良的战略地位，被称为“永不沉没的航空母舰”，何况台美共享自由、民主、人权等普世价值。台湾如果成为美国的领土，可以提升美国在印太地区的战略利益和安全，同时也能保护台湾民众免受共产主义的侵扰，将创造双赢的局面。
  <br/>
  <br/>
  “美国买下台湾”这项提案在网路引发热议，被转贴在台湾各社群媒体平台，有网友抢当美国51州，笑称“免费送”，也有人批评这不就是“卖台”？
  <br/>
  <br/>
  自由亚洲电台29号也就“美国买下台湾”这项民众对白宫的倡议连署，在台北街头随机访问了10名台湾人。受访者年龄层跨越18岁到60岁，有学生、上班族、家庭主妇。不过，都没有人同意“贩卖台湾”。

  至于台湾政府部门又怎么看？外交部发言人欧江安29号在例行记者会上被问到此案，她说，所谓“购买台湾”这样的提案，当然是伤害台湾主权，无法认同。

  <br/>
  <b>
   外交部：台湾非卖品
  </b>
  <br/>
  <br/>
  欧江安强调：“台湾当然是非卖品！台湾当然是非卖品！而且我们已经很自豪，我们从1912年以来一直存在，而且台湾很自豪我们是民主的国家、民主蓬勃发展、自由开放的国家。”

  <br/>

  欧江安说：“中华民国台湾是主权独立的国家，我们有自己的政府、有我们自己的人民、有我们的外交部、有我们的军队，有我们处理对外关系的能力。我们现在跟17国建立正式的邦交关系，还有跟世界上超过100个国家，建立实质的往来。我国是主权独立国家，所以不可能被任何国家或是其他国家的人民来所谓的‘购买’。身为一个自由民主的国家，而且是自由民主的灯塔，台湾是非卖品。”
