 <p>
  民进党任用一位曾经是国民党员、立场亲中的原住民，担任原住民族事务部主任，被批评是“绿皮红骨”。对此，民进党党主席亲赴火线召开记者会捍卫这项人事案，还为此自曝他自己也曾是国民党员。
  <br/>
  <br/>
  台湾媒体《镜周刊》踢爆民进党新任命的原住民族事务部主任曾志湘，曾是国民党员。担任桃园县复兴区区长时，曾发表“蓝绿政党如果通通都做不好，共产党更有办法的话，当然也欢迎共产党来领导”、“两岸一家亲比较务实”等亲中言论。
 </p>

  《镜周刊》报导称，有民进党内不具名主管直言，曾志湘曾与对岸统战官员互动频繁，若党内主管会议讨论到国安、选战机密，难保不会泄漏给对岸。
  <br/>
  <br/>
  民进党党主席卓荣泰28号大动作召开记者会捍卫这项人事任命。
  <br/>
  <br/>
  <b>
   捍卫部属    民进党主席自曝曾是国民党员
  </b>
  <br/>
  <br/>
  卓荣泰说：“这个报导上有说，他过去是国民党党员，现在是民进党党员，未来是共产党党员，我想这个逻辑太可怕了。如果用过去是国民党党员，现在是民进党党员，这样来检验我们的党员，好像我们会‘死伤过半’，连我自己都曾经是国民党党员。曾经是国民党员的民进党员，何其之多？何罪之有？到现在我还欢迎国民党员成为民进党党员，欢迎其他党的党员成为民进党员，我们应该觉得骄傲。”
  <br/>
  <br/>
  卓荣泰要求幕僚将他的手机拿给他，直接在记者会上，秀出自己过去国民党员证的照片。
  <br/>
  <br/>
  卓荣泰说：“这就是我的国民党党证，我从来不以为意，我一直带在身上，在我手机的资料里面。这不是一件丢脸的事情，而且这个时间是民国66年（西元1977）4月6号，我还是高中生的时候，那个时候党国时代，谁不是国民党员？”
  <br/>
  <br/>
  被质疑“绿皮红骨”的原住民族事务部主任曾志湘受访表示，当时发表亲中的言论，是“时空环境不同”。

  曾志湘说：“当初我能够讲那种话，我也是一个以我们原住民的未来，是不是能让所有政党重视我们原住民。可是综观这三年多来，我觉得民进党对原住民的政策很实在、很尊重，所以我毅然决然加入民进党。”
  <br/>
  <br/>
  曾志湘说，他担任桃园县复兴区区长任内四年，走遍原住民部落，他会以自己这种身分的转变感化所有原住民，争取原住民生存权，在明年大选前大力宣扬蔡英文在原住民部落的施政。
  <br/>
  <br/>
  至于现在是否同意过去所发表的“两岸一家亲比较务实”等亲中言论？曾志湘说：“那怎可能嘛，对不对，人都有转变的时候”，谁对我们原住民事务最照顾，我就站在那一方，现在会听从民进党的旨意和国家政策走。
  <br/>
  <br/>
  <b>
   民进党立委质疑党“小圈圈任命” 道不同不相为谋
  </b>
  <br/>
  <br/>
  曾质疑民进党秘书长罗文嘉上台后，强势主导党内人事，并公开批评党中央人事“小圈圈任命”怒辞中常委的民进党籍立委王定宇，28号接受自由亚洲电台采访指出，原民部主任曾志湘过去说错了，做错了，就是错。
  <br/>
  <br/>
  王定宇说：“比方说认为不管谁领导都一样，那看看香港，有没有都一样？比方说他曾经去参加海西特区相关会议，进入到统战体系里，那个时候的言行，对台湾来讲，是损害台湾的国家利益的。”
  <br/>
  <br/>
  王定宇认为，曾志湘既然现在要担任民进党原民部主任，对过去的言行想法必须诚实面对，如果他现在还是同样的想法，政党是相同理念的结合，道不同不相为谋，有所转变必须讲清楚，否则如果理念想法和民进党南辕北辙，怎么共同打一场国会和总统的选战？
  <br/>
  <br/>
  曾志湘现阶段工作主要是辅选民进党的山地和平民原住民族立法委员进入国会，以及在蔡英文争取连任的总统大选重要一役上，在台湾各原住民部落冲出更高选票。
  <br/>
  <br/>
  民进党立委陈莹将争取连任，曾志湘任务之一是为陈莹辅选，不过陈莹接受自由亚洲电台采访表示对曾不了解。
  <br/>
  <br/>
  陈莹说：“不熟啦，可能约略有听过这号人物，不熟。”
  <br/>
  <br/>
  RFA记者：“这次民进党为何找他？”
  <br/>
  <br/>
  陈莹说：“完全不晓得，就没有人知道，也很意外 。”

 <div>
  陈莹说，民进党有基本的台湾价值， 希望曾志湘能以台湾原住民主体性价值观，打从心里认同民进党的中心价值，在他这个新职务上，可以为总统、为立委候选人、为党加分，不要横生枝节。
  <br/>
  <br/>
  陈莹受访时提到，中国大陆对台湾的渗透无所不在，特别在锁定原住民上，中国大陆很早就有这样的远见，因为原住民是台湾最早的主人。陈莹还透露，曾有国安单位成员感叹她是最后一根稻草，千万别倒了。
  <br/>
  <br/>
  <b>
   学者：两岸交流退潮 亲中背景渗透？转投靠绿营？
  </b>
  <br/>
  <br/>
  一名不愿具名的原住民族研究领域的学者对本台表示，曾志湘说“时空环境不同了”而有不同看法立场，在某种角度上并没有说错，国民党马英九总统执政那八年，两岸几乎天天在交流，民意代表率团到大陆参与交流，可以见到国台办官员，甚至习近平、胡锦涛。2016年国民党下台，很多原住民部落很紧张，担心资源断掉，没想到民进党桃园县长郑文灿执政，蓝绿通吃，政绩亮眼，去年九合一选举，郑文灿连任，居然在深蓝的复兴乡原住民部落，拿下七成选票，远远超过过去民进党能在原住民部落拿三成就不错的成绩。这促使原住民开始转向民进党。
  <br/>
  <br/>
  这位学者就说，过去在马英九时代，可说达到统战高潮，红色上升的高潮，现在潮流退了的时候，到底还有哪些还处于被洗红的状态，变成一个渗透的棋子，哪些是真的再倒回绿营，卓荣泰的作法，有机会把被染红的“毒素”洗过来。
  <br/>
  <br/>
  台湾原住民族政策协会理事长、阿里山邹族人亚博思涌· 阿古亚纳说，没办法接受曾志湘的亲中言论，但比起其他原住民族民意代表或地方首长的言论，曾志湘程度还算是轻微的。
  <br/>
  <br/>
  亚博思涌说，有原住民代表在中国参访时直接说：“台湾原住民有三个妈妈，中共统战部、什么部、什么部，就是已经认祖归宗的，什么血浓于水，切不断的这种关系。”
  <br/>
  <br/>
  亚博思涌说，不希望原住民族成为两岸拉锯的讨论筹码，到底曾志湘是否中共派来民进党的“卧底”？他个人观察不是这样，因为要比民意代表率团到大陆参访或说“朝贡”、“输诚”，在原住民部落，会多到令人感到“触目惊心”。
  <br/>
  <br/>
  亚博思涌说，原住民族比较在意的是，民进党在原乡经营的方式，都挑些“失意政客”。过去的国民党员，在国民党失去执政之后，他们没有位子，在组织上还有一定人脉的，民进党喜欢挖这种角色。这会让族人反感，换个党籍，同样的人又可以有位子，那民进党跟国民党有什么不一样？新气象在那里？
  <br/>
