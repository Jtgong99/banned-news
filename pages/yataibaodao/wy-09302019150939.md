 <p>
  <br/>
  中共建政70周年之际，中国媒体加大了拥护政府的宣传力度。这也促使人们反思中共的统治模式，民族主义是否是中共有力的意识形态工具，中共对“中国模式”的信心何在？本台记者王允邀请到《北京之春》荣誉主编胡平和普林斯顿高等研究院历史所访问学者裴毅然对这些问题进行了深入的讨论。
 </p>
 <p>
  <b>
   民族主义是中共重要的统治工具之一
  </b>
  <br/>
  <br/>
  记者：最近国内媒体对国庆的宣传铺天盖地，歌曲《我和我的祖国》到处传唱，有关电影也在上映。有人认为，整个态势体现出中国共产党治国的主要工具是民族主义。请问裴先生，您同意这种说法吗？
  <br/>
  <br/>
  裴毅然：准确地说，是中共借助民族主义抵御西方、欧美的普世价值，因为中共发现，没有比民族主义更有实效的思想工具。
  <br/>
  <br/>
  它在意识形态领域，已经没有武器，没有子弹了。所以，它就把1840年鸦片战争以来，我们民族一直挨打、受辱的耻辱，从情绪上来引导群众仇洋、仇美，这是很低的、义和团的层次，但在中国，现在还有大批有这种义和团思想的民众。所以，它这个工具还一时能够用。
  <br/>
  <br/>
  记者：民族主义足以概括中国共产党对中国的统治方式吗？
  <br/>
  <br/>
  胡平：民族主义在中国是长期存在的东西，民族主义是一个基础，但它本身并不能告诉我们，这个国家或者这个统治集团，它到底要做什么事情？
  <br/>
  <br/>
  中国共产党相对于同时代的很多其他政党而言，在意识形态上，民族主义东西反而最少。因为它原来就是苏联的附庸，它所强调的一直是共产主义，而共产主义是国际性的运动。现在，共产主义的那套东西已经破产了，所以，中国共产党对那套东西讲得比较少了。
  <br/>
  <br/>
  但现在中国共产党依然有一个问题，就是它的民族主义的内涵到底是什么的问题。
  <br/>
  <b>
   <br/>
   如何解读习近平瞻仰毛？
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>

  <br/>
  记者：习近平30号去瞻仰毛泽东遗容，有人说这是代表他要回到毛泽东的道路上，您同意这种判断吗？
  <br/>
  <br/>
  裴：我的理解是,习近平去瞻仰毛泽东的遗容,这当然是一种政治上的表态，向老一代，或者是说向左派表明自己不忘初心的政治立场。他所要坚持的是中国共运的合法性，也就是此前胡锦涛所说的，绝不走改旗易帜的邪路。这是他主要的政治底牌。
  <br/>
  <br/>
  但是，他也想有限地回到毛时代，比如终身制、搞个人崇拜等等，这些都是毛时代很显著的特点。
  <br/>
  <br/>
  记者：按照您的说法，习近平对毛泽东是一种有限的继承，那么习近平和毛泽东最主要的不同是什么呢？
  <br/>
  <br/>
  胡：在十一前夕，习近平去瞻仰毛泽东的遗容，这主要是因为他把毛泽东看作是中华人民共和国的创始人。至于习近平本人，他很清楚，今天的中国不可能完全回到毛时代，而且他本人也未必愿意回到毛时代。
  <br/>
  <br/>
  实际上，今天的中国更多是带有邓小平时代的色彩。因为这两个时代之间存在很大的张力和矛盾，他就需要缩短这两个时代之间的差距。所以，习近平看起来要比他的两个前任，江和胡，好像更强调毛泽东。当然，他个人肯定是更崇尚毛泽东那种带有个人独裁式的统治方式。
  <br/>
  <br/>
  <b>
   习近平的中国模式是要取代西方吗？
  </b>
  <br/>
  <br/>
  记者：在习近平治下，中国政府要为全球提供一种新的治理模式，有人认为，这代表中国共产党是想代替西方。您同意这种看法吗？
  <br/>
  <br/>
  裴：我的理解，这并不是代表他们想全面取代西方模式。为什么？因为他知道他取代不了。他们想要的只是要当第三世界的头。用经济实际，也就是有限的市场经济，来证明一党专政在落后国家的相对合理性。
  <br/>
  <br/>
  他们想论证一种在第三世界的所谓的发展模式，也就是经济上相对开放一些，但在政治上拧紧阀门，保持政治恐怖，保持所谓的一党专政和政治稳定，同时用市场跟西方做生意，也可以富起来。这就是它所谓的中国模式。
  <br/>
  <br/>
  记者：中国现在的情况是经济相对自由，但政治上还在沿袭高度管控，而且是自上而下的治理模式，老百姓心里也有这样的问题，它到底是不是一种新的东西，西方也比较关注这个问题。请问胡先生您的看法？
  <br/>
  <br/>
  胡：其实中国在经济上的成功，和历史上的很多偶然事件是分不开的。在这一点上，其他国家根本不可能复制。所以，这并不意味着它推进了一种新的模式，而是它靠着自身取得的一些成功，对普世价值形成了一种严峻挑战。就这一点来说，它是值得重视的问题。
  <br/>
  <br/>
  <b>
   国庆“普天同庆”是信心爆棚？
  </b>
 </p>

 <p>
  记者：现在国庆的态势体现出“普天同庆”，而且中国共产党也似乎有极大的信心，裴先生，您怎么看待中共领导层的这种信心？
  <br/>
  <br/>
  裴：中国共产党当前这一代常委、政治局委员等，其实他们并没有什么信心。他们也只是摆摆架子。他们也知道国际共运是西风残照、汉家陵阙了。
  <br/>
  <br/>
  记者：海外对中国的这种模式有很多抨击，但从国内媒体看来，似乎中国政府还有很强烈的信心。请问胡先生，您怎么看待这种信心？
  <br/>
  <br/>
  胡：现行中共领导人看似对这种专制统治抱有很大的信心，这种信心是对专制的信心，实际也就是对压制的信心，反过来，也就是对人心不够有信心。它就觉得，如果不是它的高压，那可能就翻篇了。如果高压不存在了，第二天人心就会显示出不同。
  <br/>
  <br/>
  记者：感谢两位先生接受我的采访。
