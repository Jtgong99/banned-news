 <p>
  所罗门内阁会议决议和台湾断交。台湾同时宣布与所罗门中断36年邦谊。外交部长吴钊燮证实，北京以5亿美元金援夺取所罗门，并指控所国官员被收买。蔡英文总统谴责北京霸凌台湾，公然破坏国际秩序，意图转移香港反送中抗争焦点。蔡英文上台已失掉6个友邦，台湾友邦数量降至16国。
  <br/>
  <br/>
  尽管台湾外交部次长徐斯俭16号抵达所罗门，受到所国人民支持，但仍无法挽回台湾在南太平洋最大友邦的断交之举。所罗门总理苏嘉瓦瑞16号召开内阁会议，会中27票支持所国邦交转向和中国大陆建交，0票反对，6票弃权，从而确定所罗门结束和台湾36年的邦谊。

  “中华民国”外交部第一时间召开记者会，宣布和所罗门断交。所罗门的国旗也同步从台湾外交部大厅悄然撤离。

 <p>
  <br/>
  外交部长吴钊燮说：“所罗门群岛政府今天在内阁会议中通过决议，转与中华人民共和国建立外交关系。中华民国台湾政府对此一决定至为遗憾，并予以强烈谴责，政府并宣布自即日起终止与所罗门群岛的外交关系。”
  <br/>
  <br/>
  吴钊燮表示，全面停止双边合作计划，立即撤离大使馆、技术团、台湾卫生中心，也要求索国驻台人员立即撤离。
  <br/>
  <br/>
  外传中国大陆以金援美金5亿夺取所罗门，吴钊燮证实了这项说法：“这个数字和我们掌握的一样。”

  吴钊燮抨击北京说：“中国政府再次透过金钱外交手段，承诺华而不实的巨额援助，收买少数政客，并让所国政府赶在中国国庆前通过决策、决议，和我断交，目的就是在打击台湾，伤害台湾人民，一点一滴压缩、消灭台湾主权。这次更刻意在大选前进行攻击，影响选举的企图昭然若揭，政府予以强力谴责，呼吁台湾人民坚持主权，自由民主理念，坚持走向国际。”
  <br/>

  至于指控所国官员被北京“收买”证据是什么，吴钊燮强调，情资来自友邦，相当可靠，但不便多谈。

  总统蔡英文也随即在总统府发表声明：“对于所国政府跟中国建交的决定，我们要表达强烈的遗憾跟谴责。过去几年，中国一再透过金钱、政治上的压力，打压台湾人的国际空间，我们也要给予最严正的谴责，这不仅是对台湾的威胁，更是对国际秩序的公然挑战和破坏。”
  <br/>
  <br/>
  蔡英文说，多年来，台湾对邦交国的态度，就是当一个真诚的朋友。竭尽心力，诚意待人。但是，面对中国的介入打压，台湾绝不受威胁，不会任人予取予求。
  <br/>
  <br/>
  蔡英文强调，台湾不会和中国竞逐金钱外交，应付不合理的要求，这不是台湾做外交的方式。更何况中国的金援承诺，经常只是空头支票。而台湾为所国人民所做的，特别是医疗、农业、教育、文化上的协助，也从来不是金钱可以衡量的。

  蔡英文说，未完成的合作计划无法持续，确实很遗憾，也是所国人民的损失，这是所国政府的选择。
  <br/>
  <br/>
  蔡英文表示，感谢国际上协助斡旋此事的盟友。邦交生变的确是一个挑战，但台湾在全世界还是有许多好朋友，与台湾并肩作战，台湾并不孤单。
  <br/>
  <br/>
  蔡英文说，中国正透过断台湾友邦的方式，和各种加压手段，打击台湾的民心士气，迫使台湾接受“一国两制”。相信两千三百万人民的答案都是：不可能。台湾朝野政党，对两岸关系有不同的看法，但不接受“一国两制”，这是台湾最大的共识。

 <p>
  <b>
   蔡英文：中国欲转移香港反送中焦点
  </b>
  <br/>
  <br/>
  蔡英文说：“挖走台湾的邦交国，就算能够制造一时的舆论，对于两岸关系的发展，终究是百害而无一利。中国的举措，无论是针对军购F-16展开报复，或者是为了香港“反送中”事件转移焦点，都不会影响台湾人民的信念和团结。台湾不会放弃提升国防战力，更不会放弃对民主的坚持、以及对香港的声援。”
  <br/>
  <br/>
  蔡英文说，来自中国的压力只会更大，不会更小，但是台湾人民不会屈服，中国影响台湾政治和选举的企图，也不会得逞。面对中国霸凌，台湾必须要勇敢团结，走向世界，没有退路，不能畏缩妥协，必须一致对外，走出自己的路。
  <br/>
  <br/>
  台湾最大在野党中国国民党也发声明表达遗憾。国民党文传会代理主委程美华指出：“这已经是民进党执政短短三年多来第6个断交国家，我国邦交国面对国际现实，多处于两岸之间的抉择难题，邦交国选择与我断交的连锁效应，恐形成各界所担忧的“雪崩式断交”。但民进党政府迟迟无法面对断交潮的根本原因，仍采取义和团式的冲撞心态处理外交事务，相信我邦交国的数字恐将持续下探。”

  国民党强调，民进党执政以来，从圣多美和普林西比、巴拿马、多米尼加、布吉纳法索、萨尔瓦多，到索罗门群岛，已有6国和台湾断交，目前邦交国剩16国。民进党政府始终将责任推给断交国家、大陆当局，甚至在野党，就是不见民进党自我省思当前外交问题的根本症结。
  <br/>
  <br/>
  国民党认为，除强化台湾和国际间的竞争力与影响力外，化解两岸敌意、增进彼此善意，减少两岸花费过多资源与精力于国际间厮杀与仇视，才是正本清源之道。

   所国倒向北京显示美澳日影响力减
  </b>
  <br/>
  <br/>
  政治大学国际事务学院副院长黄奎博16号接受自由亚洲电台采访感叹：“中华民国政府一直强调什么理念相同、价值同盟等等，但是在现实残酷的外交场域上，可能还是不敌中国大陆的金钱外交。”
