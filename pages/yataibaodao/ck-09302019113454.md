 <p>
  9月29日下午，数十位华人来到旧金山中国领事馆前，向中共献花圈，向国徽泼墨，以此纪念“十一”中共建政70周年。
  <br/>
  <br/>
  下午4点钟，人们陆续来到中领馆大门前，开始举行纪念中共建政70周年的活动。此时，有人带头高呼口号：“打倒共产党！不做马列子孙，做真正的中国人！香港加油！……”
 </p>

 </p>

 <p>
  活动的组织者、在89六四被解放军坦克碾压致残的方政说：“中共建政70周年，今天我们到这里来是给中共送终的，希望它早日垮台，还中国人真正的自由。”
  <br/>
  <br/>
  旧金山华人有一辆民主战车，车上贴着要求结束中共一党专政，实现民主宪政的标语，不定期在旧金山大街上巡游。车长刘福开着战车带来一只花圈，献给建政70年的中共。花圈上写着“篡政前二十八年本为苏联走狗，窃国已七十載早已恶贯满盈。”刘福在发言中列数中共70年的罪行：“文革之后，本来改革开放让中国人看到一丝曙光，但是到了1989年，学生的民主诉求被中共无情镇压，一直到现在，习近平上台开历史倒车，实行独裁。最近湖南一位公民王美余，因为要求习近平下台，实现普选，在看守所离奇死亡，这些事，一桩桩、一件件，都表明中共的罪恶，它应该到结束的时候了。”
 </p>

  <br/>
  旧金山“五洲洪门致公总堂”盟长赵炳贤也来参加这场“纪念”活动。他特别提到香港，他说：“香港一定要推行时代革命，因为港共政权已经成为中共暴力镇压的工具，已经损害了香港的核心价值。我们作为一个正义的中国人要勇敢的面对，就算是一息尚存，都要反对港共政权，争取自由民主。”
  <br/>
  <br/>
  参加中领馆前活动年龄最小的是13岁的孙誉朗，他刚从香港来，他说他也参加了香港人“反送中”的游行：“我们去游行，就猜到会被警察抓，有个跟我差不多大的朋友去了，被抓了，最后被送到监狱里。我自己也吃过催泪弹，催泪弹是用香港市民的钱买的，现在三个多月，香港警察打出了2000发催泪弹。”

  <br/>
  参加旧金山华人纪念“十一”中共建政70周年活动的，还有美国音乐家瓦莱丽·萨姆森（Valerie Samson）女士，她的中文名字叫赛颂菲。1989年她在中国留学，亲历“六四”屠杀，并被解放军逮捕。赛颂菲把目前的香港比作1989年的“六四”，她说：“我不知道为什么事情怎么会变得这么糟，当局应该从过去的错误中吸取教训。现在香港人快要什么都没有了，他们只有香港，我希望香港人继续为他们的人权和民主而抗争。”
  <br/>
  <br/>
  在纪念活动将要结束时，人们将一面面五星红旗踏在脚下，对这个代表中共政权的图腾表达蔑视。此时，一位华人手持墨汁，奋力泼向中领馆大门顶上的中华人民共和国国徽，旧金山华人纪念中共建政70周年活动就在这惊人一泼中结束。
