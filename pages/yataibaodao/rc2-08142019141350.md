 <p>
  香港已经持续两个月有数百万人参与的反送中运动，在中国却被官方媒体描绘成完全不同、缺乏民众支持的小型暴力团伙行为。这场信息战如何展开的呢？
  <br/>
  <br/>
  反送中行动延烧至占领香港机场，抗议者诉求“警察还眼”，这是源自一名抗议女子遭击伤眼睛，示威者指控，香港警方过度执法造成她可能失明。不过，在中国中央电视台网站与微博上，除了有未核实的讯息，还有观点偏颇的评论。
 </p>

  央视网站报导抗议女子遭击伤眼睛事件时，迅速引用网民提供的照片，指称这名抗议女子在香港街头发钱给示威者。《纽约时报》在8月14日的一篇题为《中共如何对香港抗议者展开信息战》的报导中就指出，中国官方媒体报导意在暗示抗议民众是花钱雇来的闹事者。

 <div>
  央视的官方微博上则指称，这名女子是被其他示威人士击伤眼睛，并指抗议者污蔑香港警方，短文最后还标注上“丢人现眼”一词。前香港媒体人闾丘露薇在个人推特上批评，这不只是国家级媒体传播谣言的问题，而是散播支持仇恨与暴力。
  <br/>
  <br/>
  仇恨与暴力的传播速度惊人。纽时报导认为，虚假信息让人们不再同情抗议者的诉求，而有能力删文的中国审查员也未阻止网路上要打死抗议者的言论。一个现实在中国与海外有不同版本。纽时引述《这不是宣传》 (This is Not Propaganda)一书作者波莫兰采夫(Peter Pomerantsev) 的话说，虚假信息的目的是为了使人心怀疑虑，助长阴谋论。
  <br/>
  <br/>
  纽时指出，让人担忧的是，中国官方媒体的叙事方式形同中国国家主席习近平在内的国家领导人的说法。这加剧中国大众的误解与愤怒，也可能导致政府承受更大压力，在有限或不准确的信息上，做出过度反应或误判。
