<hr/><div class="post_content" itemprop="articleBody">
 <p>
  【明慧网二零一九年五月三十一日】在美国的一些宗教及信仰团体日前被告知，美国政府意在更加严格地审核签证申请、对人权及宗教迫害者拒发签证，包括移民签证和非移民签证（如旅游、探亲、商务等），已发签证者（包括“绿卡”持有者）也可能被拒绝入境。
  <a href="https://www.ntdtv.com/gb/美国国务院.htm">
   美国国务院
  </a>
  官员并告知美国
  <a href="https://www.ntdtv.com/gb/法轮功.htm">
   法轮功
  </a>
  学员可以提交迫害者名单。请海内外大法弟子立即行动起来，更完整的收集、整理和向明慧网提交迫害者名单，包括迫害者本人及其亲属、子女、资产的信息，以便定位迫害者。
 </p>
 <p>
  同时，请美国大法弟子顺便收集和提交那些在美国参与中共海外迫害者的个人信息，比如在神韵艺术团和神韵交响乐团演出剧场外、在法轮大法修炼心得交流会场外，在法拉盛、真相点、邪恶网站等场所参与中共海外迫害者。台湾、香港、韩国、西班牙等其它国家或地区的
  <a href="https://www.ntdtv.com/gb/法轮功.htm">
   法轮功
  </a>
  学员也请提供当地参与中共海外迫害者、其亲属、子女、资产信息。
 </p>
 <p>
  收件信箱： ReportFugitive@minghui.org，ErRenBang@minghui.org
 </p>
 <p>
  明慧编辑部
  <br/>
  二零一九年五月三十一日
 </p>
 <hr/>
 <p>
  <strong>
   恶人基本信息：
  </strong>
  <br/>
  • 姓名（中文和中文拼音，护照姓名的英文拼写）
  <br/>
  • 性别
  <br/>
  • 出生日期（至少有年份，减少和避免重名重姓者）
  <br/>
  • 近照
  <br/>
  • 中国大陆的工作单位、职务
  <br/>
  • 中国大陆的家庭住址（省、市、县）
  <br/>
  • 迫害者本人、亲属、子女的美国住址
  <br/>
  • 迫害者本人、亲属、子女在美国的工作、学校名称和地址
  <br/>
  • 迫害恶行的描述和证据（恶行的类型见下）
  <br/>
  • 国外资产情况
 </p>
 <p>
  <strong>
   注意事项：
  </strong>
  <br/>
  A. 暂时缺乏信息的项目栏，请据实填写“不详”或“待查”。
  <br/>
  B. 基本信息栏目未包括的信息请填写在“更多信息”栏目。
  <br/>
  C. 表格中写不下的信息，请在表格中填写关键词，将完整的信息填写在相应名称的附件中。
  <br/>
  D. 提交文件，请寄给明慧编辑部专用收件信箱：ReportFugitive@minghui.org, 或者ErRenBang@minghui.org
  <br/>
  E. “迫害者”包括、但不仅局限于直接实施迫害者，也包括制定具体政策、下达命令以及协同者。
  <br/>
  F. 根据美国的相关法律及总统公告，符合拒发签证要求的恶行包括：
  <br/>
  E-1. 未经法院判决为“死刑”而故意杀人；
  <br/>
  E-2. 酷刑和其它残忍、不人道或有辱人格的待遇或处罚；
  <br/>
  E-3. 没有刑事指控的拘押；
  <br/>
  E-4. 绑架或秘密拘押而致人失踪；
  <br/>
  E-5. 其它对生命权、自由权或人身安全权的公然剥夺；
  <br/>
  E-6. 下令、煽动、协助或以其它方式参与群体灭绝。
  <br/>
  F.附件名称：
  <br/>
  附件一：迫害事实
  <br/>
  附件二：迫害证据
  <br/>
  附件三：更多信息
  <br/>
  附件四：更多图片
  <br/>
  更多附件：
