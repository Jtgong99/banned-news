 <p>
  9月13日，台湾处理两岸事务的民间组织海基会证实，再有台湾人士在大陆失联。陆委会呼吁台湾民众，评估前往中国大陆的风险。
 </p>
 <p>
  台湾屏东枋寮乡政顾问李孟居上个月赴香港关心反送中运动后，入境中国大陆失联20多天，国台办才公布他涉犯危害国安被捕，引发台湾社会哗然。未料，9月12日晚上，又有蓝营名嘴江岷钦在TVBS政论节目爆料，他的大学学弟蔡金树入境中国也失联。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   蓝营名嘴电视爆料：
  </b>
  <b>
   “
  </b>
  <b>
   蓝到底
  </b>
  <b>
   ”
  </b>
  <b>
   的学者赴陆一样被逮
  </b>
  <br/>
  <br/>
  江岷钦12号，在《少康战情室》政论节目爆料说：“蔡金树，对两岸一向友好。他出钱出力，还担任南台湾两岸关系协会联合会主席，跟郑立中、跟一些来来往往都有很密切的关系，他大概半年多前也一样到大陆去，也因为用国安理由，把他关到现在还没有出来。他的家人也不知道他现在情况怎么样。所以我觉得以这种情况来看，其实跟蓝绿没有关系。蔡金树一定是蓝，蓝到底的嘛，各位上网看一下中评社，很多评论两岸关系、对蔡英文政策有所质疑。它（陆方）一样用国安的理由关起来。”
 </p>
 <p>
  <div class="image-inline captioned" style="width:1500px;">
   <div style="width:1500px;">

 </p>
 <p>
  海基会发言人蔡孟君13号接受自由亚洲电台采访证实，蔡金树已失联超过400天。
  <br/>
  <b>
  </b>
 </p>
 <p>
  <b>
   海基会：蔡金树去年
  </b>
  <b>
   7
  </b>
  <b>
   月出席泉州两岸交流后失联
  </b>
  <br/>
  <br/>
  蔡孟君说：“海基会指出，蔡金树的亲友曾于去年8月底向海基会陈情表示，蔡金树于7月20日赴大陆福建省泉州市参加两岸食品交易会，入住厦门某饭店，翌日退房后即失联。海基会立即去函海协会协寻，并透过管道多方查询，迄今未获具体回覆。”
  <br/>
  <br/>
  对于名嘴爆料蔡孟君被控“国安理由”羁押等说法，海基会表示，没有进一步消息。
  <br/>
  <br/>
  RFA记者：海基会这一年去函过几次？
  <br/>
  <br/>
  蔡孟君：目前我们就是去函一次。
  <br/>
  <br/>
  RFA记者：家属还有没有什么说法？有台媒报道说，人是平安，被关在福州？
  <br/>
  <br/>
  蔡孟君：我目前看到一些讯息是这样，不过因为家属并没有再跟我们做一些联系，所以我也不便去代为说什么。
 </p>
 <p>
 </p>
 <p>
  <div class="image-inline captioned" style="width:1500px;">
   <div style="width:1500px;">

  <br/>
  <b>
   陆委会呼吁陆方：说明蔡金树犯什么法？人在那？
  </b>
  <br/>
  <br/>
  主管两岸事务的大陆委员会发言人邱垂正表示，因为家属是向海基会反应，陆委会目前得知有关蔡金树的消息也是来自海基会。
  <br/>
  <br/>
  邱垂正说：“疑似触犯什么样的法律，人身自由被限制的时间、地点，我觉得中国大陆都要说明清楚。”
  <br/>
  <br/>
  相较2017年3月台湾NGO工作者李明哲，及最近李孟居赴陆失联遭羁押案，家属亲友急欲透过政府、媒体关注协寻。蔡金树“被失踪”却在一年后才曝光，陆委会对此案也不清楚。
  <br/>
  <br/>
  邱垂正强调：“不是不关心，有些家属要用低调的，你必须要尊重。也许家属有他们的方法、有他们的救援途径，政府基本上要尊重。但是我们对于国人在大陆的人身自由安全非常重视，如果有请求，政府要竭尽所能提供协助。”
  <br/>
  <br/>
  <b>
   人权团体：中国刑事逮捕程序违反国际人权标准
  </b>
  <b>
   <br/>
  </b>
 </p>
 <p>
  台湾人权促进会曾大力声援两年前的李明哲案，李明哲最后被中国法庭以“颠覆国家政权罪”判刑5年。台湾人权促进会13号在台北奥斯陆自由论坛会场也摆设一个摊位，展示营救李明哲的纪录影片。
 </p>
 <p>
 </p>
 <p>
  <div class="image-inline captioned" style="width:929px;">
   <div style="width:929px;">
    <img alt="蔡金树赴陆失联逾一年。（摘自蔡金树脸书）" src="https://www.rfa.org/mandarin/yataibaodao/gangtai/hx2-09132019104438.html/4.jpg" title="蔡金树赴陆失联逾一年。（摘自蔡金树脸书）"/>
   </div>
   <div class="image-caption">
    <span style="width:929px;">

    </span>
    <span class="copyright">

   </div>
  </div>
  <br/>
  台湾人权促进会秘书长邱伊翎13号接受自由亚洲电台采访，得知在李孟居失联不到一个月后，又传出蔡金树在中国大陆失联案，她惊讶地说：“又一个？”
  <br/>
  <br/>
  邱伊翎说：“看到李明哲绝不是单一一个个案，两岸早已签署司法互助协议，但事实上，每次当中国政府逮捕人的时候，其实都不会通知我国政府，也不会让家属有探视当事人的机会。当事人通常在中国被抓，处于失联状态，也不知道被关押在什么地方，甚至大概连律师辩护的可能性都是微乎其微。”
  <br/>
  <br/>
  邱伊翎指出，中国目前在刑事人权的逮捕程序上非常有问题，也违反国际人权标准，所有可能入境中国的台湾人当然也会被人权恶化的情况所波及，呼吁政府应该积极要求中国政府保障所有在中国的台湾人基本人身自由，即使被逮捕也应尊重相关司法程序。
  <br/>
  <br/>
  <b>
   蔡金树曾取得厦门大学学位
  </b>
  <b>
  </b>
  <b>
   活跃于两岸交流事务
  </b>
  <br/>
  <br/>
  60岁的蔡金树是台湾中山大学政研所硕士、厦门大学台湾研究院区域经济学博士，现任高雄市两岸关系研究学会理事长、南台湾两岸关系协会联合会主席，曾在多所大专院校兼课。
  <br/>
  <br/>
  蔡金树个人脸书最后发文是2018年7月11日，他分享《鹰传媒电子报》报道的《中国社会科学院大学教授岳西宽来台分享“一带一路”商机》文章，不过这篇文章现已无法点阅。另有友人在2018年7月20日，分享数张蔡金树参加“两岸食品交易会开幕会”的照片。
 </p>
 <p>
 </p>

 <p>
  蔡金树个人脸书有许多他在中国大陆参访、交流的照片，他脸书自述“来自穷乡僻壤的东石乡型厝寮村，1959年出生时正逢端午节早上卯时，家父正愁小孩名字无从命取，愁绪中家门眼前一堆海蛎壳，之前八棵破布子片叶无留，结实累累，黄澄澄如金树般灿烂夺目，就叫金树吧！”
 </p>
 <p>
  对于蓝营名嘴江岷钦在政论节目上呼吁关注蔡金树时提到，如果两岸要愈走愈近，这是开倒车。如果台湾人觉得到大陆做事心不安，那民进党现在要推动《中共代理人法》，台湾人也会有压力，会不会一不小心被自己台湾人关起来，这都是开民主倒车。
  <br/>
  <br/>
  <b>
   绿委呼吁郭台铭、韩国瑜跟国台办关系好应伸手救人
  </b>
  <br/>
  <br/>
  对此，高雄出身的民进党立法委员管碧玲13号接受自由亚洲电台采访说：“胳臂要往里弯，不能把中国的蛮横当作我们的过错，如果他们跟中国的关系好，这种时候应该要伸手救援，而不是不理不睬以后，回过头骂我们的政府。郭台铭、韩国瑜跟国台办关系都那么好，赶快出手救人不是应该这样吗？”
  <br/>
  <br/>
  管碧玲表示，蔡金树是她念大学时的学弟，十多年前曾带中国访问团到她的服务处拜访，十多年没有连络，知道蔡金树在中国做生意一段时间。管碧玲说，她会透过海基会了解此案。
  <br/>
  <br/>
  民进党籍立法委员罗致政12号，在与江岷钦同台的政论节目上回应蔡金树案，提醒香港反送中运动这段时间很敏感，台湾人到大陆要非常小心，大陆如果要找理由对付你，国安是最好理由，随时可以抓，不要说绿营的，蓝营也有机会被抓。
  <br/>
  <br/>
  <br/>
  自由亚洲电台记者夏小华 台北报道 责编: 陈美华、何平    网编：瑞哲
