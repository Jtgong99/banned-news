 <p>
  中国女子田径400米获奖选手在社交媒体上被人说“看起来像男人”。中国体育中的性别问题，已经成为社交媒体的热门话题。
 </p>
 <p>
  最近在黑龙江省大庆举行的全国田径锦标赛总决赛中，女子400米比赛金牌和银牌得主廖梦雪及童曾欢，分别以53.25秒和53.35秒的成绩取胜，比第三名快出好几秒。但是她们的性别因为外表而受到质疑。在中国社交媒体上他们被网友认为“看起来像男人”。还有网友说，她们接受采访时的声音根本就是男人。”
 </p>
 <div>
 </div>
 <div>
  廖梦雪及童曾欢不是第一次受到社交媒体的质疑。2019年7月她们在沈阳举办的全国田径锦标赛上的成功也引起了评论。当时她们所在的湖南女子接力队获得女子400米接力赛金牌，廖梦雪还得到女子400米比赛的金牌，童曾欢获得铜牌。从那以后，廖梦雪和童曾欢遭中国社交媒体用户暗示“他们看起来像男人”。
  <br/>
  <br/>
  中国田径协会宣布这两位选手都是女性，但是网民们并没有放过这个问题。
  <br/>
  <br/>
  据《人民日报》报道，很多网友都说从她们的外表或者说话方式来看，很难说她们是女性。现在这个话题已经传播到英语社交媒体。
  <br/>
  <br/>
  中国体育中的性别问题，如今已成为社交媒体的一个热门话题。
  <br/>
  <br/>
  自由亚洲电台记者陈爱祯报道 责编：申铧 网编：郭度
