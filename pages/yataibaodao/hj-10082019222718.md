 <p>
  2019年已经过去一大半，全球经济增长依旧疲软。与此同时，在美中贸易战白热化以及香港抗议不断的情况下，中国未来的经济走向似乎面临不少挑战。
  <br/>
  <br/>
  美国财政部前首席经济学家卡伦·迪南（Karen Dynan）星期二华盛顿智库彼得森国际经济研究所举行的一场研讨会上介绍说，由于受包括美国财政刺激措施减少、贸易壁垒增加、还有贸易政策不确定性等因素影响，全球经济增长正在放缓。
  <br/>
  <br/>
  “其中，美国的经济增长慢了下来，同时，中国经济也在经历持续的结构性放缓。”
 </p>
 <div>
 </div>
 <p>
 </p>
 <p>
  彼得森国际经济研究所高级研究员尼古拉斯·拉迪（Nicholas Lardy）在会议休息时接受本台记者采访时指出，美中两国去年7月爆发贸易战以来，层层加码的关税不但对双方经济造成影响，同时也使全球经贸受到一定冲击。
  <br/>
  <br/>
  “例如，我们看到美国、中国、甚至欧洲的制造业增长都在放缓。如果新的关税生效的话，这将给全球经济带来巨大的压力。”
  <br/>
  <br/>
  但拉迪认为，造成中国经济增长放缓的原因不是美中贸易战。
  <br/>
  <br/>
  “中国制造业在贸易战开始之前就开始下行。这反映出，中国经济正进行结构性改变，朝着以服务业为导向的经济模式发展。服务业指标成为中国经济增长的主要动力。”
  <br/>
  <br/>
  值得注意的是，拉迪今年8月就曾发文提到，造成中国经济放缓的主要原因是“中国下调整体信贷增速，以降低金融风险”，还有“中国国家主席习近平主席错误地、有选择性地将信贷和其他资源分配国有企业，而不是民营企业”。
  <br/>
  <br/>
  美国智库高地研究所（High Point Institute）研究员秦伟平告诉记者：“中国的服务业可以说是先天不足，后续乏力。只是说中国的人口基数大，所以才表现出在某些行业上消费力比较强，市场好像很大。例如，我们明显看到乘用车的数据在大幅下滑，还有一些在华企业撤资的情况。中国现在的经济困境是政府过去十多年采取非正常手段刺激经济造成的。官方一直在加杠杆、发债，导致整个国家陷入资不抵债的状态。”
 </p>
 <p>
  <br/>

 </p>
 <p>
  中国除了要应对各种国内经济疲软状态外，香港不断升级的抗议行动以及美国或取消香港特殊关税区的决定也令北京政府十分头疼。
  <br/>
  <br/>
  秦伟平补充说，如果香港的政治危机不断恶化，香港经济遭受灭顶之灾的同时，中国经济也未能幸免。
  <br/>
  <br/>
  “香港作为一个可以让西方世界观察中国，中国吸引国际资本的地方，其经济如果受到冲击，对中国打击会更大，因为中国现在不是在经济上升期，也就是说他无法用很好的状态去抵御香港带来的冲击。”
  <br/>
  <br/>
  拉迪则表示：“香港对中国的重要程度随着时间的推移，已经逐渐减弱。我认为在融资层面来说，香港对大陆依然重要。”
  <br/>
  <br/>
  他认为，就算香港失去作为国际金融中心的地位，中国受到的影响也相对较小。
  <br/>
  <br/>
  （记者：韩洁 责编：申铧 网编：郭度）
