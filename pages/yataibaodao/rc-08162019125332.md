 <p>
  香港“反送中”运动让迪士尼跟着“躺枪”，定于明年上映的电影《花木兰》，因女主角刘亦菲表态支持香港警察后，网路上已掀起“抵制木兰”行动。全球企业和中国往来，在面对政治事件时，越来越难以避免陷入两难困境。
  <br/>
  <br/>
  中国女演员刘亦菲在《花木兰》电影中说着为国奋战，她有责任挺身而出的台词；现实世界里，面对香港 反送中 抗议示威，刘亦菲表达支持香港警察的立场，在推特上“#抵制木兰（#boycottmulan）”成为热门关键词。拥有美国国籍的刘亦菲也遭网民批评，她是享受着美国的自由，却支持中国政府的极权专制。
 </p>
 <div>
 </div>
 <div>
 </div>
 <div>
  电影《花木兰》的制片商迪士尼面对这股抵制风潮，选择低调，至今不愿发表看法。美国华盛顿智库传统基金会亚洲研究中心主任沃尔特·罗曼（Walter Lohman）告诉自由亚洲电台，国际企业为了能在中国市场发展生存，必须是市场导向，但中国的政治影响力全面介入经济活动，这不是正常商业经营，他对此感到忧虑沮丧。但他认为，企业面对重大事件表态，不是没有弹性空间，需经过慎重评估。
  <br/>
  <br/>
  罗曼表示：“一些美国企业有时候未能展现它能有的创意，而这次事件显示，包括企业及其中的个人都必须认识到，（一旦表态）对企业本身及演员自身可能影响为何，尤其是像对香港目前如此受到国际社会关切的事件，（表态后）当然会得到国际社会注意。”
 </div>

  他举例，美国航空业去年在中国官方限期压力下必须遵守北京的一中原则，不允许航空企业在网站上将台湾名列为国家，而包括联合航空及美国航空干脆全面更改为航程飞行来往的城市，不涉及北京介意的主权国家认定，就是最好例子。
  <br/>
  <br/>
  除了刘亦菲以外，还有艺人李冰冰，黄晓明及杨幂多人也就香港表态，支持港府和香港警察。港星成龙更接受中国央视访问，他除了痛心香港的发展还要誓言成为拥护五星旗的护旗手。

  中国打国际舆论战，在国际上有名气的影视艺人，不论是否是被迫，都难避免卷入政治斗争，要选边站。有网友点评道：这些艺人没有政治权利也不能有政治目的，但他们必须要有政治原则。
