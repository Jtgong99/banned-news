 <p>
  美国国防部印太事务助理部长薛瑞福（Randall Schriver）28号证实，中国取消美国军舰原订访问青岛的行程。亚太局势近来正逢多事之秋，面对日韩关系的紧张升级，薛瑞福也表达美国的沮丧和失望，他呼吁两国对话，否则中国与俄国这两大竞争对手将渔翁得利。
  <br/>
  <br/>
  探讨美日韩三方国防事务合作的重要性，华府智库战略暨国际研究中心28日上午举行的研讨会恰逢其时，时间点正是日本与韩国双方政治和经济纷争不断加深之际。
 </p>
 <p>
  除了东北亚，美国与中国从经贸到军事关系，双方歧见也不断加深，薛瑞福在回答自由亚洲电台询问时证实，中国解放军的确取消了美国海军军舰原订访问青岛的行程。
  <br/>
  <br/>
  薛瑞福说：“美国军舰原定访问青岛。我们不久前确实收到中方告知，他们用词是说现在“不方便接待”美国军舰访问青岛。我们会再看看是否要重新安排时间，或以其他特定军舰、访问其他港口来弥补。”
  <br/>
  <br/>
  军舰互访是美中两军交往的其中一环，而这已是中国本月第二次拒绝美军请求。在取消美军军舰访问青岛前，中国刚刚拒绝了两艘美军船舰访问香港。
 </p>

 <p>
  尽管如此，薛瑞福对于中国在南海的军事化作为及香港局势，再度重申美方的关切。
  <br/>
  <br/>
  薛瑞福说：“我们希望美中的两军关系，在整体双边关系中扮演稳定的脚色。但是，我们现在对许多议题都感到关切，像是中国国家主席习近平就破坏了自己的承诺，就是不会将南海军事化。我们仍对此表示关切。另外，美国也密切关注香港局势的发展。”
  <br/>
  <br/>
  在南海，中国近来进入越南专属经济海域展开地震勘测作业，引发美国包括国务院及国防部的接连关切与批评；而在东北亚，日本28日将韩国从俗称“白名单”的出口审查优待国名单中剔除，而韩国则决定，终止与日本的军事情报保护协定（GSOMIA）。日本与韩国的摩擦不断，薛瑞福也难掩美国的沮丧失望。
  <br/>
  <br/>
  薛瑞福说；“我们呼吁韩国，恢复与日本的《军事情报保护协定》，让协定重新生效；我们也呼吁双方展开‘有意义的对话’，解决分歧。所谓有意义的对话是要抱持着解决问题的心态上谈判桌，而不是以扩大分歧的态度对话。”
 </p>

  <br/>

  他更说得清楚明白，美国在亚太的两大盟友吵架，中国和俄罗斯最开心。
  <br/>
  <br/>
  要维系自由与开放的印太地区，这个多事的秋天，美国除了要关注中国如何处理南海与香港问题，还得安抚日本与韩国这两个重要的印太盟友。
