  据路透社报道，香港行政长官林郑月娥曾向北京提交一份报告，撤回有争议的引渡法案以缓解该地区日益严重的政治危机，但是遭到北京的否决。
  <br/>
  <br/>
  路透社在报道中援引三个知情人说， 香港行政长官林郑月娥8月7日在国务院港澳办及香港中联办于深圳共同举办香港局势座谈会时，向北京提交了一份报告。报告中分析了抗议者提出的五个诉求：撤回对《逃犯条例》的修订；对警方滥用暴力的行为进行独立调查;进行双真普选;撤回对抗议活动的“暴动”定性;并对目前被捕的人撤回控罪。港府在报告中说，对这些诉求进行评估后，他们认为撤回修法可以缓解该地区日益严重的政治危机。
  <br/>
  <br/>
  不过，报道说，中国政府拒绝了香港政府安抚抗议者的计划，并命令她不要对抗议者做出任何妥协。这是首次有证据表明，中国中央政府控制香港政府对反送中运动的反应。

  <br/>
  路透社的报道还说，引渡法案导致数百万人涌入香港街头抗议，北京的影响力触及了香港“一国两制”模式的核心。两个多月的抗争活动使得香港面临严重的政治危机，这场运动也演变成一场更广泛的争取更大权利和民主的运动。
  <br/>
  在深圳会议上，国务院港澳办主任张晓明曾表示，如果动荡持续，“中央政府必须进行干预”。自此以后，有迹象显示北京采取了更强硬的立场。
  <br/>
  <br/>
  在回应路透社的一份声明中，香港行政长官办公室表示，香港政府已经努力解决抗议者的担忧，但没有直接评论它是否已向北京提出这样的建议，或接到指示。
