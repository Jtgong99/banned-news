<div class="post_content" itemprop="articleBody">
 <p>
  【新唐人北京时间2019年05月31日讯】中共迫害法轮功学员的残暴行为遭到国际社会的谴责。根据
  <a href="https://www.ntdtv.com/gb/美国.htm">
   美国
  </a>
  国务院的消息，将严格审查签证，对在中国迫害法轮功的人
  <a href="https://www.ntdtv.com/gb/拒发签证.htm">
   拒发签证
  </a>
  ，甚至已得到
  <a href="https://www.ntdtv.com/gb/绿卡.htm">
   绿卡
  </a>
  的人，也会
  <a href="https://www.ntdtv.com/gb/拒绝入境.htm">
   拒绝入境
  </a>
  。
 </p>
 <p>
  中共已是穷途末路，很多官员想移居海外寻退路。法轮功学员将根据掌握的情况提交材料，酷刑、绑架或秘密拘押而致人失踪等恶行，都在
  <a href="https://www.ntdtv.com/gb/拒发签证.htm">
   拒发签证
  </a>
  之列。
 </p>
 <p>
  符合拒发签证要求的恶行包括：
  <br/>
  •未经法院判决为“死刑”而故意杀人
  <br/>
  •酷刑和其它残忍 不人道或有辱人格的待遇或处罚
  <br/>
  •没有刑事指控的拘押
  <br/>
  •绑架或秘密拘押而致人失踪
  <br/>
  •其它对生命权 自由权或人身安全权的公然剥夺
 </p>
