 <p>
  针对外国情报机构在社交媒体领英进行招募活动，西方情报机构发出了警讯。
  <br/>
  <br/>
  《纽约时报》报道，西方反间谍官员警告说，外国特工们正在利用社交媒体试图招募有价值的人，微软旗下的领英是他们的一个主要猎场。针对外国特工在领英上跟许多用户进行接触，美国、英国、德国和法国的情报机构已发出了警告。官员表示，中国间谍在这个社交媒体上最为活跃。
  <br/>
  <br/>
  领英除了社交功能之外，同时具有徵才、求职的功能，这款职场社交平台，既是潜在的虚假信息的又一个载体，也是招募间谍的理想渠道。领英还是唯一未被中国屏蔽的美国主要社交媒体平台。
  <br/>
  <br/>
  美国国家反谍报与安全中心主任威廉·埃瓦尼纳透露，中国特工每次都在领英上同时联系数千人。他说，外国情报机构在寻找任何能够接触到他们想要的信息的人，包括企业的商业秘密、知识产权和其他研究成果。中国特工经常利用包括领英在内的各种渠道，主动提出让潜在的招募对象去中国访问，有时伪装成企业招聘公司，提出请招募对象做有偿演讲、进行有偿咨询活动，或者提出为他们的研究提供帮助。特工们以此为出发点，与招募对象建立关系。
  <br/>
  <br/>
  《纽约时报》报道提及2017年12月，德国情报官员公开点名中国特工使用领英，试图与一万名德国人建立联系。2018年10月，法国情报机构告诉政府，中国特工使用社交网络，特别是领英，试图与4000名法国人建立联系。联系的对象包括政府雇员、科学家和公司高管。美国官员则表示，中国有“线人”， 那些表面上没有被情报机构雇用但与他们合作的人，设立假账户在网上“钓鱼”。他们的目标包括超级计算、核能、纳米技术、半导体、隐形技术、医疗保健、杂交谷物、种子和绿色能源等领域的海外专家。
  <br/>
  <br/>
  美国一位高级情报官员指出，一些设立虚假账户的人与中国情报机构的IP地址有关，而其他人则是由虚假公司设立的，其中包括一些声称属于高管猎头公司。
  <br/>
  <br/>
  领英发言人妮科尔·莱弗里奇表示，公司会积极主动地发现虚假账户并予以删除。她说， 我们严格执行我们的政策。这些政策非常明确：建立虚假账户，意图误导或欺骗会员的欺诈活动，都违反我们的服务条款。领英公司信息和安全部门负责人保罗·洛克威尔证实，该公司一直针对中国的间谍活动与美国执法机构展开合作。
  <br/>
  <br/>
  根据《纽约时报》报道，对中国利用领英招募海外间谍一事，中国外交部没有回复置评请求。
  <br/>
  <br/>
  自由亚洲电台记者陈爱祯华盛顿报道 责编：申铧 网编：郭度
