 <p>
  本周三（5月29日）晚8点，福克斯电视女主播崔西‧里根（Trish Regan）与中共党媒（CGTN）英文女主播刘欣进行辩论。（视频截图）
 </p>

<hr/><p>
 【大纪元2019年06月01日讯】（大纪元记者穆清报导）美国福克斯商业频道主播
 <a href="http://www.epochtimes.com/gb/tag/%E7%BF%A0%E8%A5%BF%C2%B7%E9%87%8C%E6%A0%B9.html">
  翠西·里根
 </a>
 <b>
  （
 </b>
 Trish Regan
 <b>
  ）
 </b>
 和中共大外宣CGTN英文主播
 <a href="http://www.epochtimes.com/gb/tag/%E5%88%98%E6%AC%A3.html">
  刘欣
 </a>
 就中美贸易
 <span class="s1">
  问题
 </span>
 一问一答，从最初的辩论变为访谈，过程中，双方没有唇枪舌剑，但却有诸多看点。
</p>
<p class="p4">
 <span class="s1">
  访谈时间16分钟，里根就知识产权、华为、关税、国家资本主义等议题发问，
  <a href="http://www.epochtimes.com/gb/tag/%E5%88%98%E6%AC%A3.html">
   刘欣
  </a>
  的回应颇具中共官方语调，多以绕圈子，不正面回答问题而展开。
 </span>
</p>
<p class="p4">
 <span class="s1">
  而里根针对刘欣所犯下的明显或严重错误，并没有去追问和反驳，只是一开始给出她提问的依据和具体案例，并欣然让刘发表她的看法。
 </span>
</p>
<p class="p6">
 <span class="s1">
  此次访谈受到海内外各方关注，主流媒体、自媒体以及时政专家等多有点评。
 </span>
</p>
<p class="p6">
 <span class="s1">
  自媒体
 </span>
 <span class="s4">
  “
 </span>
 <span class="s1">
  磨镜片的学霸新保守主义
 </span>
 <span class="s4">
  ”
 </span>
 <span class="s1">
  基于一些中文媒体对访谈的误导性报导及分析，在
 </span>
 <span class="s4">
  youtube
 </span>
 <span class="s1">
  频道上解析了刘欣出现的一些主要错误和漏洞，以下是选取的部分内容。
 </span>
</p>
<h4 class="p4">
 <span class="s1">
  漏洞一：
 </span>
</h4>
<p class="p4">
 <span class="s1">
  在里根提出美中知识产权问题时，屏幕上列出了一串有关美国和中国之间发生的知识产权纠纷所提起的司法指控案例。
 </span>
</p>
<p class="p3">
 <span class="s1">
  里根表示，中国是世界贸易组织（WTO）的成员国，偷窃知识产权对美国公司面临风险，FBI有很多证据证明中国从美国窃取了很多机密，涉及金额达到上亿美元。无论如何，偷窃别人的东西都是不对的，问刘欣如何看待这些问题。
 </span>
</p>
<p class="p4">
 <span class="s1">
  刘欣避开了正面回答这个问题，而是说应该去问美国公司是否愿意来中国做生意，很多美国企业愿意在中国建立产业和业务，他们在中国赚取丰厚利润…… 我不否认有侵权，有版权问题或存在盗版甚至盗窃商业机密。那是需要处理的事情。
 </span>
</p>
<p class="p8">
 <span class="s1">
  刘欣接着表示，关于盗窃知识产权，她认为世界上每一个国家都会这样做，这是一个很普遍的现象，但政府很支持保护知识产权，这是一个国家或企业发展所必须的。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “这是一个相当客道的官话。”“磨镜片的学霸 ” 在视频点评中表示，首先刘欣是已承认，中国（中共）在这些方面确实存在知识产权盗窃，“但是她把它普遍化了，认为这是每个国家都在做的事，而且是一个普遍的商业恶行。”她犯下了一个很大的漏洞。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “对于盗窃知识产权，中国（中共）这边是以一个国家、政府行为，且政府资助，但对其他国家的公司来说，这些行为属于公司的个人行为，是一个公司为谋取个人公司利益的行为，所以国家对这些行为是要打击的，如果要上升到政府层面，政府一定会制止公司做这样的欺诈行为，但在中国，（中共）政府支持企业去偷窃其他国家的知识产权，这是根本差别。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  “在国外这属于一个企业的商业诈骗行为，在中国，这属于政府行为，”所以在国外很早就有了保护知识产权的一些法律，但是到中国，到现在为止都还没有完善的这类法律，即便是最近，也是北京刚刚提起要建立这样的法案，而且还非常的不完善，也没有正确的进行落实，这也是一路一带峰会上其他国家要求中共去做的。
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  漏洞二
 </span>
</h4>
<p class="p3">
 <span class="s1">
  美国公司在中国挣取很多利润，的确中国有很大的市场，所以美国公司赚取利润。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “赚取利润是公司的责任，对商人来说是他们的责任，对国家来说，替那些公司在政治上立法，保护知识产权这是国家的责任，这是两种差别。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  他表示，在访谈中，刘欣的很多话都是官方表述，里根虽然没有去反驳刘欣、追击刘欣的漏洞，但是里根却提供了很多信息，且非常准确和有参考价值。
 </span>
</p>
<p class="p3">
 <span class="s1">
  在中美贸易冲突过程中，很多报告都共同指出，中共在很多方面都违反了WTO的规则。对中国和美国之间的争端，WTO提供了很多专业报告、包括具体案例（Case），并非刘欣所说只有美方的一个声明（Statements）。
 </span>
</p>
<p class="p3">
 <span class="s1">
  此外，在过去的16、17年期间，美国对中国在23个领域中提出诉讼，一共有23个案例 ， 已经结案的19个案例以美国完胜，中方是零胜利（19比0 ），还有4个案例还处于审理过程中。
 </span>
</p>
<p class="p3">
 <span class="s1">
  在访谈中，刘欣也表示，她不否认中共存在盗窃知识产权问题，也承认中共的确触犯了大量的WTO规则。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “磨镜片的学霸”提醒，一些愤青、小粉红，不要盲目的相信中共官方、大外宣以及外交部单方指责美国的官话。
 </span>
</p>
<p class="p3">
 <span class="s1">
  再如，里根提到中国（中共）在2017年推出的《国家情报法案》。
 </span>
 <span class="s1">
  这个法案迫使中国的公司必须和中共政府合作、为政府提供情报，法案同时对国外的公司也有效，只要它们在中国做生意。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “这种情况在美国是绝对不允许的，这是侵犯了私有权和公司权利，甚至是政府对公司行为的直接干预，在整个国际贸易中是绝对不允许的。”“磨镜片的学霸”说。
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  漏洞三
 </span>
</h4>
<p class="p3">
 <span class="s1">
  里根接下来的问题是中共强制技术转让，并举例说，如果华为到美国来做生意，美国政府要求华为把技术交出来，行不行？
 </span>
</p>
<p class="p3">
 <span class="s1">
  刘欣则绕了一个圈子、偷换了概念来回答。“你给我技术，我给你市场没有问题，花钱用你的技术；就如我也有英文老师，我花钱她教我英文一样。”她说。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “磨镜片的学霸”解读说，如果是一个单纯的市场行为——我有技术卖给你，你出钱买我的技术——没问题，明买明卖的关系。
 </span>
</p>
<p class="p3">
 <span class="s1">
  但中共政府的做法则完全不是这样，明要暗拿统统用上。“中国（中共）政府目前的行为是国家直接通过法令介入、告诉人们，你绝对不可以占用我的市场，除非你把技术转让给我，这叫强制技术转让，是通过政府进行直接干预的行为。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  对于公司来说，目的当然是赢取利润。所以它们即便面对不公道、不合理的行为，为了谋求短期利益，仍可能选择出卖自己的技术。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “但如果没有政府干预，让市场自己去做决定的话，没有公司会愿意出卖自己的技术产权，或者就算明买明卖，也绝对不可以通过政府强制行政力的干预来完成。
 </span>
</p>
<p class="p3">
 <span class="s1">
  而中共政府现在就是采用强制行政令的干预，我制定这个决策，你不想玩就别来啊。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  问题是中共还想让他国接受它的这套规则。当中国参与国际贸易，就得按照国际规则玩游戏，但参与了，并仗着世界最大的市场以及世界第二大经济体，强行要他国接受中共的规则，当然会引起众怒。
 </span>
</p>
<p class="p3">
 <span class="s1">
  这样看刘欣的回答，用“花钱请英语老师”的例子无法为中共强制技术转让开脱。
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  漏洞四
 </span>
</h4>
<p class="p3">
 <span class="s1">
  从访谈开始，里根几乎每段问话都会给出非常明确的案例、时间点和数据。
 </span>
</p>
<p class="p3">
 <span class="s1">
  谈到中美之间的贸易关税非常不平衡时，里根做了一个统计数据，给出中国对美国的关税平均下来是9.9%，美国对中国的关税只有三分之一都不到，并问到这如何叫关税对等？要不要把所有的关税都取消，大家都是零关税如何？
 </span>
</p>
<p class="p3">
 <span class="s1">
  刘欣回答说，基本上我们谈论的关税不只是中国和美国之间的关税。如果清零中美之间的关税，那欧盟、日本、委内瑞拉都会这样要求。这就是对其他国家的歧视，不符合贸易多边协议。 应该用一个多边机制，来进行谈判把关税降下来。
 </span>
</p>
<p class="p3">
 中共官方向来希望把对美贸易问题扯到WTO去解决，刘欣只是变了花样重复了一遍官方说辞。
</p>
<p class="p3">
 <span class="s1">
  “磨镜片的学霸” 说，实际上刘欣的说法有明显错误，WTO是一个多边的机制，在WTO的规则之上，其他国家完全可以制定双边机制，只要这个双边机制或多边机制，不破坏WTO的规则就可以。
 </span>
</p>
<p class="p3">
 <span class="s1">
  比如说，在WTO 的基础上，美国、 加拿大和墨西哥可以有USMCA（美墨加贸易协议）， 这个协议可以比WTO规则要求更高，但不能比WTO要求低，就如WTO是60分，你有一个70、90甚至100分的规则就没问题。
 </span>
</p>
<p class="p3">
 <span class="s1">
  这就是为什么有WTO 以后，依然可以有USMCA协议，也可以有美国和欧盟之间将来可能谈成零关税协议， 也可以有美国和日本之间谈成的贸易协议。
 </span>
</p>
<p class="p3">
 <span class="s1">
  所以里根提出可以把关税清零，完完全全可以是一个双边机制，“刘欣说必须要有一个多边机制，否则对其他国家是一种歧视，是一种非常勉强的回答。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  “要改变规则，就必须是要一个多边机制，没有这样的必须。只要你在WTO之上，规则更好更严就行。”
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  漏洞五
 </span>
</h4>
<p class="p4">
 <span class="s1">
  里根称中国是国家资本主义。刘欣则以80%的中国员工受雇于私营企业，称市场在资源配置中发挥主导作用，大多数对民众生活有影响的公司，如网路公司都是私营企业。
 </span>
</p>
<p class="p8">
 <span class="s1">
  刘欣这里的回答可以说错得离谱。稍微了解中国经济的都知道，中共从来讲的都是国有企业为主导，对民众生活有影响的公司，电力、自来水、天然气、交通、银行、矿产以及电信，有哪个领域是私营企业领头。而私营企业跟中共高层同样有千丝万缕的联系。
 </span>
</p>
<p class="p8">
 <span class="s1">
  <span class="s6">
   “
  </span>
  磨镜片的学霸”说，
 </span>
 <span class="s1">
  中国的私营企业背后权利的执掌人都是红二代、
 </span>
 <span class="s1">
  官二代，中国很多大型上市公司背后的持股企业实际上都是国资，有些是地方国资，有些是中央国资委持股，甚至之前的联想公司也是有国资委出资控股。
 </span>
</p>
<p class="p8">
 <span class="s1">
  他表示，决定一个国家是不是市场经济国家，不是他有多少人口受雇于私营企业，而是在于整个国家的经济命脉，
 </span>
 <span class="s1">
  影响整个国家行业资源
 </span>
 <span class="s1">
  ，影响整个国家经济发展的核心行业都掌握在哪些人的手里？
 </span>
</p>
<p class="p8">
 <span class="s6">
  “
 </span>
 <span class="s1">
  且不说
 </span>
 <span class="s6">
  80%
 </span>
 <span class="s1">
  的此数据是否对，就是
 </span>
 <span class="s6">
  90%
 </span>
 <span class="s1">
  的人都是私营企业，只要是这些行业掌握在政府手里，你就不能算成是一个市场经济国家。这个问题和人数多寡没有关系。”
 </span>
</p>
