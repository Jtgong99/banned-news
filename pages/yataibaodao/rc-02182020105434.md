 <p>
  中国疾控中心的专家日前发表论文说，武汉当地感染新冠肺炎的病人，从1月11日开始暴增。谁该为疫情爆发负责？中国国家主席习近平发表在《求是》杂志上的文章说，早在1月7日他就对防疫工作“提出要求”。这是否显示习近平与湖北官场的甩锅比赛正在进行中？
  <br/>
  <br/>
  中国疾病预防控制中心新型冠状病毒肺炎应急响应机制流行病学组的专家们，在国内的医学学术期刊《中华流行病学杂志》发表论文，把新冠病毒疫情分为五个阶段。从去年12月31日前的104个病例，到今年1月11日到20日的第三阶段，总共有新冠肺炎感染者超过6000例，到第四阶段的1月21日到31日，则是彻底大爆发，新增2万6468人发病，其中，湖北就占74.7 %。
  <br/>
  <br/>
  这篇文章也指出，新冠病毒肺炎的粗估病死率为2.3%，其中，湖北的粗估病死率为2.9%，其他省市则为0.4%，湖北的粗估病死率是中国其他省市的7.3倍。
 </p>
 <div>
 </div>
 <div>
  中国的专家与官方机构，现在频频发声，而官方《环球时报》则解读认为，这凸显湖北第一线医护人员早意识到问题的严重性，地方政府却未采取有效防控措施，直到1月22日才启动突发公共卫生事件二级应急响应。
  <br/>
  <br/>
  美国新泽西州卫生部公共健康实验室主任范智华从事环境污染和暴露科学研究多年。18日她以个人身分接受自由亚洲电台访问时指出，回溯新型疾病的流行周期统计与研究，目的是在找出最强传染周期与特征，以做后续防疫准备工作，她认为，从湖北的致死率比其他省市要高出7倍之多来看，湖北的医护人员与医疗设备，明显无法承担，亟需外省市支援。
  <br/>
  <br/>
  范智华：“我自己个人的建议就是，应该最大限度去救治重疫区，同时，如果需要，是可以‘往外输送病人’的。因为，你现在派部队、派医生（去武汉）也好，你建不出这么多的隔离病房，而大陆的情况大家都知道，很多医院都是（政府）控制的，其他省份的医院，很多都不上班，因为是控制的。”
  <br/>
  <br/>
  其他省市空着的病房与医疗器材，现在该是支援湖北的时候，但范智华很清楚，要想把湖北轻症或无症状患者外送，这样的调配一定会引起其他省市非议，但她指出，防疫要从科学且宏观的角度出发，人性的自私与无知惊恐，是此时此刻最不需要的。
  <br/>
  <br/>
  她举例说，从公卫专业来看，无症状的感染者要传染给其他人的机率非常小，如果能及早得到医疗资源治疗，这也是给湖北的第一线医护人员缓解压力，这就是科学防疫。
  <br/>
  <br/>
  她看到国内甚至出现根本没确诊的民众家门被上锁、上封条，不让出入，实在痛心。范智华说，没有必要的限制人身自由，绝对不是该有的防疫措施。
  <br/>
  <br/>
  从政治与法律的角度来看防疫工作，也同样触及对自由的追求。

  <br/>
  <br/>
  贺卫方指出，因为没有新闻自由，不仅民生多难、且政府亦无信，更谈不上现代化的治理能力与体系。
  <br/>
  <br/>
  他也特别提到中国共产党中央委员会机关刊物《求是》杂志，近来刊发习近平在2月3日于中共政治局常委会上的讲话。
  <br/>
  <br/>
  这篇文章特意透露，习近平早在1月7日就于政治局会议上对防疫工作“做出要求”。但习近平提出了什么要求？没人知道。
  <br/>
  <br/>
  贺卫方说，最让人震惊的是，习近平1月7日的讲话，似乎被中国官媒“留中不发”。习近平的讲话也会遭中国官媒“屏蔽”，归根结底在于，如果湖北地方媒体能就疫情有自由且负责的报导，现在何须出现这种相互推诿卸责的官僚体系？
  <br/>
  <br/>
  “甩锅大赛”赤裸裸上演，地方官员不背的锅，习近平作为定于一尊的领导人也不愿背。这看在纽约城市大学政治系教授夏明眼里，不论锅怎么砸，付出代价的是中国人民的性命，习近平没有不扛的道理。

  夏明：“你有权力，权力配套的就是责任，你的权力无限大，责任也就无限大，在这种情况下，他如果要甩锅，甩给谁？习近平代表着中国共产党处于‘武大郎开店’的格局，他上来，确实是党内处在‘逆向淘汰’、‘劣币驱逐良币’。经历这场大疫后，我不认为习近平还能够坐稳。”
  <br/>
  <br/>
  夏明告诉自由亚洲电台，已故领导人邓小平是集体领导；到前国家主席江泽民在“工程师治国”的大帽子下，还算维持尊重地方与科学的形象；胡锦涛时代是“九龙治水”格局；习近平上任，凡事强调亲力亲为、身兼多个小组长，却将防疫工作交给了国务院总理李克强，这也凸显习近平现在是没有底气与定力。
  <br/>
  <br/>
  中共治国究竟是以人为本？还是以权为上？夏明说，相信中国共产党内菁英也看得很清楚，习近平在必要时，也可能牺牲李克强，而党内所有人，都可能是镰刀下的韭菜，中南海内的波涛汹涌恐怕才刚刚开始。
