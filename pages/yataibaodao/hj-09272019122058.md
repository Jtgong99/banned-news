 <p>
  近年中国对外扩张对内压制的状况，引发国际社会强烈谴责。联合国大会在美国纽约召开之际，北京发布《中国与联合国》立场文件，总结与联合国各领域的合作成就。与此同时，中国国务院也出台一份政策白皮书，强调如果没有共产党领导，中国将会解体。就此，自由亚洲电台记者韩洁邀请旅美学者吴祚来与美国圣汤姆斯大学国际研究中心教授叶耀元，针对这两份极具争议的中国官方文件进行分析、讨论。
  <br/>
  <br/>
  记者：这份《中国与联合国》的立场文件讲到3个重点，中国自称，深入参与全球人权治理，维护国际和地区和平安全，还有推动高质量“一带一路”。在涉及人权的部分，中国举的唯一例子就是“积极致力于促进和保障残疾人权利”，却没有涉及香港、新疆和西藏等问题。
 </p>
 <p>
  叶耀元：照顾残疾人权利这件事情除了未开放国家外，大部分发达国家都在做。人权必须要建构在自由的环境里面，但中国没有言论、新闻自由，我不知道中国有什么资格说自己促进人权。
  <br/>
  <br/>
  吴祚来：社会良性的东西总是会有，但拿这个作粉饰是非常无耻的做法。中国侵犯宗教自由，维权律师、公知被打压得非常严重。
 </p>
 <div>
 </div>
 <div>
  记者：关于人权这部分有个特别值得注意的地方，它说中国“成功走出了一条符合国情的中国特色人权发展道路”。我想请两位嘉宾分析一下，这个“中国特色人权发展道路”意味着什么？
  <br/>
  <br/>
  叶耀元：人权是普世价值，前面加上任何国家的名字就不是人权。所谓中国特色，就是中国共产党去曲解人民需要的东西，合理化自己的作为。例如，人民要安全，那新疆穆斯林“破坏”安全，所以政府要把他们关起来，是在合理化自己的作为。
  <br/>
  <br/>
  吴祚来：所有东西前面加上“中国特色”就是一块遮羞布，只要加上这四个字，就可以不把人当作人。中国正驱动政权的力量，在积极侵犯人权。
  <br/>
  <br/>
  记者：我们都知道，中国在不断对外扩张军力，但中国却号称，自己保卫了地区和国际的和平与安全，并且用朝鲜半岛、伊朗、阿富汗、缅甸、中东、叙利亚和南苏丹作为例子。叶教授，中国特别提及这些国家的意义在哪？
  <br/>
  <br/>
  叶耀元：中国当然是要写对自己有利的事情，为什么要特别提出自己在其他地方做的“小动作”，或造成的不稳定因素？就好像中国不会提到内蒙古、印度、尼泊尔、新疆这些边界问题，也不会提到台海之间可能会发生的冲突。

  记者：中国对周边国家，甚至世界的威胁是全方位的，中国国家主席习近平提出的“一带一路”倡议就是一个很好的例子。这份白皮书是这样形容一带一路的：“一带一路”已成为全球最大规模的国际合作平台和广受欢迎的公共产品。吴先生，您怎么样评价“一带一路”倡议？
 </div>

  吴祚来：参与“一带一路”的很多国家是没有制约机制的，只要中国买通这些国家的领导人、权贵，就可以把中国的市场模式扩张到这些国家，形成政治模式的输出，并且制造的大量的债务陷阱。这些“一带一路”国家的领导人只要自己获得利益之后，就不会考虑国家的利益，把港口和资源交易给中国，这些国家的国民就成为了受害者。
  <br/>
  <br/>
  另外，整个世界贸易体系也会受到影响，一旦这些国家经济失败后，会成为累赘和动乱之源。中国还会将获得的港口作为军事基地，为以后的布局，军事主义的扩张做准备。
  <br/>
  <br/>
  记者：过几天就是中共建政70周年，北京官方出台《新时代的中国与世界》提出，中国发展之所以成功，根本原因在于有中国共产党的领导，如果没有共产党，中国将“走向分裂和解体”。吴先生，中国官方在这个时间点，发布这样一份重申党领导地位的白皮书用意何在？
  <br/>
  <br/>
  吴祚来：这是要形成一套话语逻辑，对中国人民进行洗脑教育，同时还威胁说如果没有中国共产党，中国四分五裂的结果就会成为动乱的地方，却没有提到民主化对很多有民族或历史遗留问题的国家带来的和平和进步。
  <br/>
  <br/>
  记者：这个“中国没有共产党会崩解”的说法，在网上引起很大的讨论。叶教授，您怎么看这个问题？
  <br/>
  <br/>
  叶耀元：这些话明显不是讲给“外人”听的，因为“外人”觉得中国共产党存在与否并不重要。假设中国共产党消失了，中国会不会变成很多个区块？这种假设在目前国粹主义比较高昂的情况下可能性并不高。当然，我们看到新疆、西藏、内蒙古、香港可能会希望脱离中国共产党的体制，但这个是民族自决的问题。
  <br/>
  <br/>
  记者：非常感谢两位嘉宾的精彩评论。
