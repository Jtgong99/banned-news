 <p>
  包扩前总统马英九在内的中国国民党31名大老，12号在台湾三大报纸刊登广告，呼吁鸿海集团创办人郭台铭与高雄市长韩国瑜在大选中合作求胜。不过，郭台铭铁了心，旋即宣布退出国民党。
  <br/>
  <br/>
  “即日起！郭台铭先生退出国民党！”
  <br/>
  <br/>
  郭台铭中秋节前退党传言成真。鸿海集团创办人郭台铭12号透过发言人、永龄基金会副执行长蔡沁瑜宣布“退出国民党”。党内总统初选后分裂态势底定。
 </p>
 <p>
 </p>
 <p>
  <iframe frameborder="0" height="476" scrolling="no" src="https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRFAChinese%2Fvideos%2F503259596885507%2F&amp;show_text=0&amp;width=622" width="622">
  </iframe>
 </p>

 </p>
 <p>
 </p>
 <p>
  就在郭台铭宣布退出国民党的当天早晨，包含荣誉党主席连战和吴伯雄、前总统马英九、前副总统萧万长、现任党主席吴敦义等31名国民党大老以亲笔签名共同连署，在台湾三家日报，发表《 团结、奋斗、救中华民国》的声明，呼吁郭台铭与国民党总统候选人韩国瑜合作。
 </p>
 <p>
  国民党刊登大老的声明指出：“明年一月十一日的总统与立法委员选举，是决定中华民国命运、台湾前途与人民未来的关键选举。若重蹈分裂之路，将给台湾带入动荡不安的局面，这必然会是国家之祸。希望全党全民集中选票支持韩国瑜，支持国民党在国会过半，中华民国才可重建光明，两岸重拾和平，台湾重返安定，才是国家之幸！台湾之幸！人民之幸！”
 </p>

 <p>
  国民党大老声明强调，中国国民党经由民主初选机制，已产生总统候选人。政党的可贵，就是在于亲爱精诚、群策群力、众志成城，才能赢得选战。对于当前韩国瑜与郭台铭两位同志，切盼能合而为一，以“四步”的诚意谋求真心的合作：“‘进一步’放弃意气，相互献策”、“‘退一步’反省折衷，彼此协调”、“‘让一步’海阔天空，取得共识”、“‘跨一步’集中全力，争取胜利”。
 </p>
 <p>
  郭台铭决定投入党内总统初选时，国民党颁给他“荣誉党员证”。今日，国民党大老搬出创党领袖孙中山、蒋介石，仍唤不回荣誉党员郭台铭出走决心。
  <br/>
  <br/>
  蔡沁瑜12号代表郭台铭退回国民党党员证，并回应国民党大老登广告的声明：“郭先生可以理解他们的担忧，但这名单上的人，有多少人是‘人在曹营心在汉’，因此我们更担忧的是，如果这些30年来同一批人还在为台湾把关，还有百年世袭的中常委在把持这个政党，台湾人民不会认同如此迂腐的政党。”
  <br/>
  <br/>
  蔡沁瑜继续宣读郭台铭的声明：“这群守旧迂腐的中常委，把自己的利益放在政党利益之前，把政党利益放在国家利益之前，这和郭先生当时返回到国民党的初衷，完全背道而弛。郭先生也不会眷恋这个政党，即日起，郭台铭先生退出中国国民党。”
 </p>
 <p>
 </p>

  <br/>
  <b>
   郭台铭手写退党声明：这是翻转中华民国命运的大事
  </b>
  <br/>
  <br/>
  郭台铭亲笔写下退党声明：“我并非没有挣扎。在情感上，无法继续与国民党并肩作战，感到有些惆怅，但在理智上，我知道我正在做一件对的事，一件将翻转中华民国命运的大事。”
  <br/>
  <br/>
  郭台铭退党声明还引述一句西方谚语：“一个鸡蛋从外面被打破只是一个生命的结束，若它被内部孕育的力量打破，则是一个生命的诞生。”
  <br/>
  <br/>
  郭台铭退党声明最后写道：“我个人再多的努力也撼动不了现有国民党的分赃与腐朽文化。结束了。离开了。挥挥衣袖不带走一片云彩，但愿我郭台铭的离开，能够加速国民党的改造与重生。”
 </p>
 <p>
  郭台铭退党消息传出后，国民党文传会代理主委程美华紧急召开记者会，呼吁党内同志团结辅选经过公开、公正、透明、公开程序选出的国民党唯一总统参选人韩国瑜，最大目标是2020重返执政，拨乱反正。
 </p>
 <p>
 </p>

  程美华说：“本党对于参与总统初选所有同志非常感谢，初选之后希望大家都能摒除参选过程的心理转折，全力支持国民党。”
  <br/>
  <br/>
  被问到何时知道郭台铭要退党？程美华只说：“一切都在掌握”。
  <br/>
  <br/>
  国民党考纪会主委魏平政表示，将依照党员违反党纪处分规程处理。对郭台铭声明中对党的批评，他认为，“君子交绝，不出恶声”。
  <br/>
  <br/>
  魏平政质疑郭台铭参加党内初选时，曾在前中国国民党总理孙中山、前国民党总裁蒋介石，跟前总统马英九前承诺，如果选输一定会全力支持胜选的候选人。
  <br/>
  <br/>
  魏平政：“结果在你最爱的总理、总裁跟总统前面，你都可以背弃你这样的承诺，我们也很遗憾。如果在中国国民党内初选输了，你都不认，那我们都很难想像，如果以后中华民国选举你又输了，你会怎样？”
  <br/>
  <br/>
  前总统马英九办公室表示，党都有相关规定，没有评论；前国民党主席朱立伦办公室则说，朱立伦昨天已在脸书表示立场，“中秋是团圆的日子， 一个家庭、一个企业、一个政党，乃至国家，期盼都能月圆人团圆、家和万事兴。”
  <br/>
  <br/>
  韩国瑜竞选办公室表示，对于郭董事长的决定，感到万分遗憾与失望。韩国瑜市长身为国民党提名的总统候选人，会用尽洪荒之力，和所有党内志同道合的伙伴，一起努力改变台湾的命运，并且推动国民党的改革和转型。
 </p>

 <p>
  对郭台铭宣布退党，一名支持韩国瑜的“韩粉”崔姓计程车司机接受自由亚洲电台采访时，不客气地说：“叫他赶快滚，这种人输不起、输不起。”
  <br/>
  <br/>
  <b>
   学者：利空出尽、大事底定
  </b>
  <b>
  </b>
  <b>
   兄弟登山各自努力
  </b>
  <br/>
  <br/>
  台湾中山大学政治学研究所教授廖达琪接受自由亚洲电台采访，认为郭台铭宣布退党，对国民党是好事。
  <br/>
  <br/>
  廖达琪分析：“利空出尽很好啊！一直拖拖拖、猜郭台铭要不要选？最怕就是一直在那摇摆不定，还又有一些‘换瑜’的说法，这其实对整个国民党士气和内部凝聚力都是一个打击。既然有国民党大老一起背书说希望他能够加入合作，他选择退出，反而是大事底定，那就是兄弟登山各自努力，这是好事。”
  <br/>
  <br/>
  廖达琪说，以目前民调，韩国瑜和蔡英文如果一对一对决，韩国瑜输蔡英文。但如果郭台铭与台北市长柯文哲、前国民党立法院长王金平结盟，可以吸收浅绿和中间选民选票，反而可能对韩国瑜选情有利。
