<div class="post_content" itemprop="articleBody">
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      5月初本来接近签约的美中谈判突然生变。中方收回关键承诺，
      <a href="https://www.ntdtv.com/gb/川普.htm">
       川普
      </a>
      以重税回应，中方几天后也出台一个规模小得多的反制措施。这一变故原因究竟是什么？美中贸易战的走向究竟如何？
     </span>
    </span>
   </span>
  </span>
 </p>
 <p style="font-weight: 400;">
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      5月初之前，美中确实已经形成了接近签约的协议文本。
      <a href="https://www.ntdtv.com/gb/川普.htm">
       川普
      </a>
      对5月初之前的协议基本是满意的，甚至说很快会在白宫会见习。同时习近平也在多个场合谈到贸易战与体制改革的关系，从改革开放的角度对美方的要求作出了正面的回应。实际上刘鹤的谈判是在习近平的授权和授意下进行的。也就是说，川习对最初的协议并无实质异议。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p style="font-weight: 400;">
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      但是实际的结果和最初的协议天差地别已知事实是：中方突然在多个最核心的部分（如保护知识产权和禁止强迫技术转移等）撤回承诺，把原先通过修改法律实施结构性改革的部分删掉了。而这些部分是美方认为是已经确定和必不可少的，这等于是釜底抽薪，这导致川普恢复加税决定。川普是合理的，因为两次推延加税是为达成协议。如果签约前可以推翻承诺，那签约有什么用？目前谈判仍在继续，6月底川习可能亲自会面解决问题。川习会能达成协议吗？在我看来，这次谈判生变的原因也正是6月底能否签约的关键因素。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p style="font-weight: 400;">
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      那么，这次谈判中方变卦的原因究竟是什么？我认为有几个重要原因。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p style="font-weight: 400;">
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      第一个原因，是要用党的领导来压过法治美国这次非常明确：协议中中方必须用修改法律来保障结构性改革，这是协议的基石和主线协议文本中美国要求中方修法非常细节化，中共很难向以前那样任意解释法律，在美国巨大关税压力下，这等于是用法治来取代中共领导打开一个缺口。这对反对习的江派残余非常恐惧，江派与中共是相互依存的，为了保自己，江派极力蛊惑习手中的权力是共产党给的，用权力把习近平与中共存在捆绑在一起。其实中共人心丧尽，习顺应民意结束共产党是一夜之间的事。建立在民意之上的权力才是牢靠的。但只要习在权力来源上与中共纠缠不清，就会做出错误判断。这是导致协议流产的重要原因。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p style="font-weight: 400;">
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      中方变卦的第二个原因实际是川普道破的：中方对拜登4月底参选产生了幻觉中方一厢情愿地认为，拜登上台会让贸易战轻易熄火实际上拜登对川普很难胜选。川普上台后，迎头痛击恐怖主义和共产主义，同时振兴美国，各类经济数据尤其是就业数据，都惊人地好，支持率已超同期奥巴马至少5个百分点。拜登当副总统时就没什么业绩，现与川普竞选更不行了。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p style="font-weight: 400;">
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      。中方变卦的第三个原因是江派残余的捣乱和压力如果这种捣乱和压力切中了习的弱点
     </span>
     <span style="vertical-align: inherit;">
      ，这会造成中方变卦。拜登非常有意思。与川普竞选不行，但当时是他在习近平访美时把王立军出逃美领馆时留下的材料交给习
     </span>
     <span style="vertical-align: inherit;">
      （其中可能包括薄熙来政变和活摘法轮功学员器官内容
     </span>
     <span style="vertical-align: inherit;">
      ）。川普还说，他的前任让中共杀人走脱（让中国逃脱谋杀）此话意味深长川普的话让江派胆战心惊
     </span>
     <span style="vertical-align: inherit;">
      ，也提醒人们：。清除江泽民集团一直是中国历史演变的主轴
     </span>
     <span style="vertical-align: inherit;">
      ，包括贸易战现在还有一些帮着江派的人和习近平周围
     </span>
     <span style="vertical-align: inherit;">
      的江派特务给习近平出馊主意
     </span>
     <span style="vertical-align: inherit;">
      ，把与美谈判说成“丧权辱国”，或者歪曲解读社会经济数据，挑动与美对抗，蛊惑美中经贸脱钩等等。实际上都是要把江派造成的贸易战罪责推到习头上，对习近平落井下石。江当这种考法2蛊惑状语从句：压力与习近平的档案本人对权力地位及中共
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      错误认
     </span>
     <span style="vertical-align: inherit;">
      有符合时，就会对习的决策产生作用。这是一个极其关键的因素。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p style="font-weight: 400;">
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      中方变卦的第四个原因就是就是党文化的影响。中共窃国几十年摧毁传统文化，党文化已渗透上下，延伸各个领域。比如，按国际谈判惯例，双方在未承诺的地方出现变化是常事，但对写进协议文本的承诺是不能任意改动的，这是一个根本信用问题。但背信弃义，说谎成性是党文化的特点，中方对已承诺的条文进行全面改动。这就是党文化在国际谈判中的典型表现。再比如，中方提出用行政法规来取代法律。这明显就是把中共​​在国内用惯了的做法延申到法治社会和国际法领域。美方用加税回应这是必然的。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p style="font-weight: 400;">
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      最后，六月底的川习会会有什么样的结果？川普认为，协议有可能发生，因为美国处于强势地位，而中方非常想要达成协议。我认为，能不能达成共识和签约还取决于习近平能否厘清和克服导致这次变卦的几个因素。如果习近平能正确看待权力来源，也就是用民意而不是中共作为权力来源，不陷入拜登迷思，能成功清除江派干扰和党文化影响以至抛弃中共，协议有可能发生，否则协议不会发生，而且对习本人也极其危险，因为一旦中共垮台，中共和江泽民的罪责就可能要由习近平承担了。
     </span>
    </span>
   </span>
  </span>
 </p>
