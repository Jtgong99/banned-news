 <p>
  香港歌手何韵诗12日凌晨抵达台湾。何韵诗在机场接受自由亚洲电台访问说：“就希望大家支持香港人，支持这个对自由的抗争。”
 </p>
 <p>
  香港反送中运动持续三个月，何韵诗是极少数站在第一线与香港人民共同抗争的香港影艺圈人士。她因此遭到北京抵制，演艺事业几近全毁，但她依然不惧不悔。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  何韵诗访问台湾，受到关注。她自香港搭乘国泰航空，于12日凌晨抵达台湾，在桃园机场受到约二十名航警、维安、保全人员高规格严密保护，比上周访问台湾的香港众志秘书长黄之锋在机场受到更大阵仗的维安接待。
  <br/>
  <br/>
  身穿黑T恤，戴黑口罩，围着深蓝色围巾的何韵诗，步出机场通道，面对自由亚洲电台访问，频频遭到维安人员阻挡。
  <br/>
  <br/>
  RFA记者：“这次到台湾来主要作什么样的呼吁？”
  <br/>
  <br/>
  何韵诗：“嗯！我们先不作访问，谢谢。”
 </p>

  <br/>
  <br/>
  何韵诗：“就希望大家支持香港人，对！支持这个对于自由的抗争，对！谢谢！”
  <br/>
  <br/>
  RFA记者：“你觉得台湾支持香港有什么重要性？”
  <br/>
  <br/>
  何韵诗：“不如明天有个活动，就明天再说吧！”

  <br/>
  RFA记者：对香港警察打学生有什么看法？
  <br/>
  <br/>
  何韵诗：“小心、小心”、“有点危险”。
  <br/>
  <br/>
  访谈过程在不断被维安人员拦阻打断下，何韵诗双手合十点头，并提醒记者“小心”、“危险”。
  <br/>
  <br/>
  何韵诗此行将出席13日纽约人权基金会举办的奥斯陆自由论坛。预计和台湾闪灵乐团主唱，刚退出时代力量党的立法委员林昶佐对谈。
  <br/>
  <br/>
  何韵诗12日下午四点将与台湾独立乐团歌手共同召开记者会，呼吁台湾人在9月29日上街游行，声援香港反送中行动，响应身在世界各地香港人，在极权中共建政七十年的十一国庆前夕发出怒吼。

  <br/>
  13日在台北举行的第二届奥斯陆自由论坛，何韵诗以“香港知名社运歌手”身分受邀，讲者还有朝鲜脱北者太永浩、资讯战争专家莫莉马克乌、肯尼亚知名变性人权社运者奥德莉么布瓜、泰国教育改革者秦联丰，还有获美国普立奖的缅甸克钦族记者艾斯特突珊等。
  <br/>
  <br/>
  奥斯陆自由论坛讲者一行人，预计12日下午四点拜会立法院长苏嘉全，何韵诗因为同一时间将参加“九二九台港大游行：撑港、反极权”行动记者会，因此并没有在与立法院长会面名单中。
 </p>
 <p>
  下周何韵诗将赴美国国会参加听证会，为香港反送中运动发声，
  <br/>
  <br/>
  何韵诗今年七月曾受邀在瑞士日内瓦联合国人权理事会上，对香港反送中运动情况发表两分钟演讲，她当时呼吁联合国人权理事会将不尊重人权的中国除名。
