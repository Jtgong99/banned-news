<hr/><p>
 【大纪元2019年06月07日讯】周四（6月6日）上午，
 <a href="http://www.epochtimes.com/gb/tag/%E5%AD%9F%E6%99%9A%E8%88%9F.html">
  孟晚舟
 </a>
 <a href="http://www.epochtimes.com/gb/tag/%E5%BE%8B%E5%B8%88.html">
  律师
 </a>
 团队与
 <a href="http://www.epochtimes.com/gb/tag/%E5%8A%A0%E6%8B%BF%E5%A4%A7.html">
  加拿大
 </a>
 检方再次走上卑诗法庭，敲定孟案未来
 <a href="http://www.epochtimes.com/gb/tag/%E5%BC%95%E6%B8%A1.html">
  引渡
 </a>
 程序的各个主要时间点。
 <a href="http://www.epochtimes.com/gb/tag/%E5%AD%9F%E6%99%9A%E8%88%9F.html">
  孟晚舟
 </a>
 本人当日并未出庭。
</p>
<p>
 法官与双方
 <a href="http://www.epochtimes.com/gb/tag/%E5%BE%8B%E5%B8%88.html">
  律师
 </a>
 商讨后，最终决定采纳孟晚舟律师团队提议的方案，即在明年1月20日-24日对孟案是否符合“双重有罪”（double criminality）原则进行单独
 <a href="http://www.epochtimes.com/gb/tag/%E8%81%86%E8%AE%AF.html">
  聆讯
 </a>
 。“双重有罪”是
 <a href="http://www.epochtimes.com/gb/tag/%E5%8A%A0%E6%8B%BF%E5%A4%A7.html">
  加拿大
 </a>
 决定是否
 <a href="http://www.epochtimes.com/gb/tag/%E5%BC%95%E6%B8%A1.html">
  引渡
 </a>
 某人的核心原则，即考量一个人被控在他国犯下的罪行如果发生在加拿大，是否同样构成犯罪。
</p>
<p>
 孟晚舟律师团队在周四出庭期间表示，孟晚舟案由于涉及美国国内对伊朗制裁的法令、而加拿大并无这一法令，因此不符合“双重有罪”的原则；其律师团队将在1月
 <a href="http://www.epochtimes.com/gb/tag/%E8%81%86%E8%AE%AF.html">
  聆讯
 </a>
 期间就该论点进行辩护。而代表加拿大检方的律师则在周四庭上强调，孟晚舟受控的主要罪行是“欺诈”，即欺骗银行，因此符合“双重有罪”的原则。
</p>
<p>
 1月20日-24日期间举行“双重有罪”单独聆讯的同时，孟晚舟案的引渡聆讯也就此正式开始。明年6月15日，法院将就孟晚舟被捕过程是否触犯滥用程序（abuse of process）举行为期二周的聆讯。明年9月28日，引渡聆讯将再次进行二周。其后，孟案若仍有事务需上庭处理，将于明年11月16日-23日再次开庭。法官将于其后做出判决。
</p>
