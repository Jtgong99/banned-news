 </p>
 <p>
  针对香港特首林郑月娥宣布撤回《逃犯条例》修订，目前在台湾的香港众志秘书长黄之锋公开回应强调，五大诉求缺一不可，否则港人的抗争不会停止。
 </p>
 <p>
  对于香港特首林郑月娥4日以预录的视频宣布，将会在立法会撤回《逃犯条例》修订草案，香港众志秘书长黄之锋在推特发文形容为“太迟及太少”（too little and too late now），并指已有7人在反送中抗议风波中死亡，另有超过1200名示威者被捕。
  <br/>
  <br/>
  黄之锋4号晚间在台北受访时强调：“我们面对警察的暴力，香港的特首不是由人民选出来的时候，其实就算《逃犯条例》撤回也好，香港人的抗议活动，直到今年的10月1号我们还会持续下去。所谓五大诉求，缺一不可，这个是我们的信念和坚持。
 </p>
 <p>
 </p>
 <p>

  </iframe>

 <p>
  ”
  <br/>
  <br/>
  黄之锋认为：“北京现在要求香港政府撤回条文，其实只是小修小补而已，根本只是要香港人在10月1号之前不会上街抗议，但我们的抗议还会持续下去。”
  <br/>
  <br/>
  香港立法会议员朱凯迪形容林郑月娥只从“暂缓”变成“撤回”修例，其他一步未退，是“绵里藏针”。
  <br/>
  <br/>
  朱凯迪在记者会上呼吁：“千万不要误会以为，林郑月娥今天说要会撤回这个《逃犯条例》就等于这个运动已经胜利，或者说这个运动就会平息。”
 </p>

  <br/>
  <br/>
  朱凯迪说，香港抗争者7月1号提出五大诉求，不是随随便便提出的，而是成千上万人到街头上冒各样风险讨论的结论。
  <br/>
  <br/>
  朱凯迪指出，抗争者提出的五大诉求已经变成“全民共识”，两周前中文大学针对三个诉求进行民调，有80%受访市民坚持一定要撤回《逃犯条例》，另有80%要求要成立独立调查委员会，还有75%要求就要在这次重启政改，落实“双普选”，港人的抗争运动一定不会结束，直到五大诉求得到落实。
 </p>

 <p>
  岑敖晖呼吁台湾人能在北京庆祝十一国庆前，举办大型游行声援香港，这对港台的未来都有紧密影响。
  <br/>
  <br/>
  黄之锋也说，目前像韩国、美国、日本、澳大利亚、加拿大等世界不同地方的香港人和关心香港的朋友，准备在九月底举行大型集会或游行声援香港。
  <br/>
  <br/>
  记者会上，被问到中国大陆民众对香港反送中抗争者一片骂声，有什么话想对大陆人民说？以及香港反送中运动给大陆人民什么启示？
  <br/>
  <br/>
  黄之锋说，香港面对“一国两制”全面崩溃，中国网路审查严格、假新闻泛滥，使得中国大陆的朋友难以接触真相，希望他们能了解香港人争取的是民主、自由、人权等普世价值。
 </p>

  黄之锋：“希望有一天，不只香港有民主自由，中国大陆也有。台湾有民主选举、政党轮替选举的制度非常可贵，应珍视这样的体制。”
  <br/>
  <br/>
  黄之锋、朱凯迪和岑敖晖4号在台北出席“香港怎么了？台湾怎么办？”座谈，并与立法委员林昶佐对谈。
  <br/>
  <br/>
  黄之锋强调，港台面对共同压迫需要彼此支持。他呼吁，不分背景、党派台湾朋友跟香港人一起对抗北京的压迫，共产党是全世界最大独裁政权，“不要说一定会赢，我们一定要说，我们不会想输！”
  <br/>
  <br/>
  <b>
   黄之锋：台湾支援的面罩头盔
  </b>
  <b>
  </b>
  <b>
   保护香港街头学生的安全
  </b>
  <br/>
  <br/>
  在开放问答时有一名中年妇女提问，台湾能做什么帮助香港？
 </p>
 <p>
  黄之锋表示，香港抗争者现在抱着“揽炒”的决心，也就是“玉石具焚”的决心争取五大诉求。过去这段时间，很多台北的朋友在台北的商店买面罩、头盔支援香港，让香港很多年轻人，比我还小的中学生，有很大的安全。很现实的，在6月9号、16号百万人上街的时候，抗争者只戴医院用的那种口罩，到了现在，很多抗争者已不用那些口罩，而改用防毒面罩，比较安全。
 </p>

  <br/>
  <br/>
  温起锋还举手发言说：“一定要坚持反共！反送中持续两个多月，死了七个人，我们非常伤心，而且我们支持香港，鼎力支持香港，一定会战胜邪恶的共匪，一定要团结。香港如果溃退，台湾就很危险了，中共的野心就是想吞并台湾。”
  <br/>
  <br/>
  这场两小时的论坛最后，学联前副秘书长岑敖晖主动表示，透过主办单位光合基金会在网路直播，他看到有中国大陆网民翻墙留言，他想对此作出回应。

  这场论坛受到台湾警方高度重视，包括台北市警察局副局长高寿孙、中正一分局分局长沈炳信都现场坐镇指挥，出动数十名警力以确保3名港人在台安全。虽有零星统派人士在对街举横幅表达不同立场，不过在活动开始前已离开，未酿冲突。
  <br/>
  <br/>
  三名港人两天下来密集拜会多名民进党、时代力量和国民党籍立委。对黄之锋投书呼吁台湾通过难民法作为港人逃生后盾，台湾的蔡英文总统4日表示，依既有机制个案处理。外界质疑蔡政府睁眼说瞎话，未展现立法积极性，流于嘴巴声援香港。对此，黄之锋未作回应。
