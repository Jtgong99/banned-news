 <p>
  港独领袖、香港民族党前召集人陈浩天访问台湾之前，才遭亲中人群袭击暴打，他负伤抵台，在自由世界发声示警：“新疆集中营已在香港发生”。陈浩天接受自由亚洲电台专访强调，香港情况危急，呼吁国际加快制裁香港，这是捍卫自己的自由，防范纳粹中国势力渗透蔓延到全世界。
  <br/>
  <br/>
  倡议港独遭港府禁止运作的香港民族党，前召集人陈浩天受独派邀请访问台湾，他29日在独派IR Taiwan主办的港台人权餐会上发出警讯：新疆集中营已在香港发生，就是在边界，外人很难到达的新屋岭拘留中心。

  <br/>
  陈浩天接受自由亚洲电台专访说：“很多被送进去的人出来作证就是说，被强奸、被性侵、轮奸、被打、重伤，严重的骨折，还有传闻说其实有死了人。被抓进去的都是因为政治原因，所以那是一个集中营的开始。林郑月娥政府说，新屋岭拘留中心会停用，但是他们会在岛上建一个拘留中心，那就是一个真正的集中营。”
  <br/>
  <br/>
  香港歌手何韵诗在台湾参加929游行，遭统派人士泼洒红漆。陈浩天到台湾之前，在香港遭恐袭，他负伤到台湾参加独派聚会，并参加929撑港反极权台北大游行。

   陈浩天在香港遇袭
  </b>
  <b>
   负伤抵台吁认清“
  </b>
  <b>
   纳粹中国”
  </b>
  <br/>
  <br/>
  陈浩天29号对自由亚洲电台说：“前天我去法院应讯的时候，在路途上，被几个人伏击，有一个中国人在那边指挥，然后其他人都是戴口罩、戴墨镜、戴帽子，所以我也看不清楚他们是什么人，他们用的手电筒，黑色很长很硬的那种，比警棍还硬的，就伏击我。主要受伤是肩膀两边都有，比较严重的是盆骨这边，还有最严重的是头后边。”
 </p>

  陈浩天出发到台湾前在脸书公布伤势，发文说，“幸好打中皮带，不是的话应该骨裂。两边膊头都中了棍，依然在痛。头部其实更严重，现要侧一边睡觉。”
  <br/>
  <br/>
  陈浩天在香港公开倡议港独后，被港府、北京视为乱港分子，香港民族遭禁止运作。他持续参加反送中行动，8月29日赴日本前在机场被捕，和香港众志秘书长黄之锋不到一天相继被抓。此次赴台前又遭不明亲中人群暴打。如何面对可能被以检控手段噤声的牢狱之灾？
  <br/>
  <br/>
  <b>
   面对可能牢狱？“
  </b>
  <b>
   只有眼前路，没有身后身”
  </b>
  <br/>
  <br/>
  陈浩天说：“其实没有人想要坐牢，但是我们也没有选择！你看现在香港还没真正完全落入中国的手中，已经这样，我们还没输掉，已经遭受这样的打压、这样的对待，你们可以想像，如果我们输了会怎样？所以我们只有眼前路，没有身后身，我们没有选择，唯有、只有抗争。”
 </p>

 </p>

  中共建政七十周年前夕，香港出现号召自愿在十一献出自己承担流血受伤，提供抚恤金的“死士召募令”。
  <br/>
  <br/>
  对此，陈浩天说：“中学生、大学生，香港的年轻人都有同样的想法，香港没有前途，我也没有前途。香港 的将来，就是我们的将来，所以他们本来就没有未来，没有将来，所以觉得我们出来抗争，我们出来牺牲也没差，因为我们本来就没有将来。”
  <br/>
  <br/>
  陈浩天认为，十月一日，不论支持反送中，或反对反送中的香港人，很多都会站出来。
  <br/>
  <br/>
  <b>
   陈浩天：北京不想在十一前弄出人命
  </b>
  <br/>
  <br/>
  对于十一港府、北京和抗争者会有什么冲突状况，陈浩天认为：“我觉得中国不想要在十一，或之前弄出人命，不想杀人。但是他们会用恐怖恐吓香港人，可能会制造一些谣言、白色恐怖，让香港人不敢出来。”
 </p>

 <p>
  不过，陈浩天认为，这只对少部份人有效，大部份人是愤怒，愤怒比恐惧的更多。他预料十一会有很多香港人出来，不管是支持或反对者都会出来。至于十一之后，他认为会不断抓人、不断起诉，白色恐怖，因为，“他们没有其他方法了”。
  <br/>
  <br/>
  <b>
   陈浩天：香港情势危急
  </b>
  <b>
   呼吁国际加快制裁香港
  </b>
 </p>
 <p>
  香港警方抓捕、镇压随着十一到来更升高力度，陈浩天30号透过自由亚洲电台向全球发出求救讯号，呼吁国际制裁香港：“制裁香港，快一点制裁，因为我们的情况非常危急。今天早上已经超过几十人被捕，他们是有计划大规模围捕、清算，情况很危急，其他国家应该尽快去制裁香港。以美国为例，如取消《美国—香港政策法》，限制香港警察入境，停售武器给香港警察，和经济制裁等。”
 </p>

  <br/>
  数以百万计香港人上街反送中，许多香港年轻人抱着愿为香港而死、捍卫自由民主的决心，但台湾则有广大支持者却拥护踏进香港中联办的政治人物选上台湾总统。
  <br/>
  <br/>
  陈浩天说，“我不理他是什么政党，是什么人，但是如果今天还在讲『一国两制』去谋取跟中国去签订和平协议的，不是共谍，就是江泽民讲的，太简单、太天真。你看香港、你看维吾尔、你看西藏，那么多的案例，现在已经在发生的事，为什么可以看不到，为什么那么笨、还去相信中国？中国已经是一个纳粹国家了。我们看二次大战历史，那时英国法国对纳粹德国采取绥靖政策，跟他签和平协议，就是因此给空间让纳粹德国变强大，大到不可能把他打倒。”

  <br/>
  陈浩天此行到台湾，主要目的是传递两个讯息，一是认清中国已是“纳粹中国”，一是“时代革命”战场不只在香港，也在台湾、在全世界。
