### 大陆法轮功学员忆4‧25 公安透露的秘密
------------------------

<p>
 【大纪元2019年04月25日讯】今年的4月25日是1999年逾万法轮功学员前往北京国务院信访办，和平上访20周年，大陆亲历者纷纷投稿明慧网，回忆事件过程，见证历史。
</p>
<p class="p2">
 <span class="s1">
  1999年7月20日，中共江泽民集团发动了对法轮功迄今长达20年的迫害，其主要借口之一，即“
  <a href="http://www.epochtimes.com/gb/tag/4%E2%80%A725.html">
   4‧25
  </a>
  ”法轮功学员上访，为给打压找理由，中共将这次和平请愿扣上了“围攻中南海”的罪名。
 </span>
</p>
<h4 class="p2">
 <span class="s1">
  上访原因
 </span>
</h4>
<p class="p2">
 <span class="s1">
  1999年4月11日，当时的中共政法委书记罗干的连襟何祚庥在天津教育学院办的《青少年科技博览》杂志上发表文章，无中生有诋毁法轮功。
 </span>
</p>
<p class="p2">
 <span class="s1">
  4月18日至24日，部分天津法轮功学员前往天津教育学院及相关机构反映实情。
 </span>
</p>
<p class="p2">
 <span class="s1">
  4月23日、24日，中共政法委书记罗干命令天津市当局出动防暴警察三百多名，驱散并殴打澄清事实的法轮功学员，并抓捕了45名法轮功学员。
 </span>
</p>
<p class="p2">
 <span class="s1">
  法轮功学员到天津市政府要求放人，被告知只有去北京才能解决问题。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  天津教育学院抓人事件
 </span>
</h4>
<p class="p5">
 <span class="s1">
  一位天津学员是当年
 </span>
 <span class="s3">
  4
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  23
 </span>
 <span class="s1">
  日在天津教育学院被天津当局非法逮捕的法轮功学员之一。
 </span>
</p>
<p class="p5">
 <span class="s3">
  “
 </span>
 <span class="s1">
  虽然整整
 </span>
 <span class="s3">
  20
 </span>
 <span class="s1">
  年过去了，仿佛就在昨天。
 </span>
 <span class="s3">
  ”他回想那天的经历。
 </span>
</p>
<p class="p5">
 <span class="s3">
  4
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  23
 </span>
 <span class="s1">
  日下午三四点钟，特警、防暴警察、武警等二三十人来到天津教育学院现场，向反映情况的法轮功学员们喊话，强迫让大家离场，
 </span>
 <span class="s3">
  “
 </span>
 <span class="s1">
  看那架势再不走就抓人
 </span>
 <span class="s3">
  ” 。
 </span>
</p>
<p class="p5">
 <span class="s1">
  坐在前排离门三米距离的男学员，第一个被两个武警抬着两臂双腿，扔出学院大门。紧接着陆续有学员不分男女老少被抬出学院扔到马路上，现场混乱。
 </span>
</p>
<p class="p5">
 <span class="s1">
  “我和一位军人男学员分别被四个武警，拧胳膊，压脖子，头根本就抬不起来，像押罪犯一样押上一辆大客车，车上还有几个武警。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  车开了大概十分钟到了一个派出所。屋子比院子低几个台阶，“武警将我反背双手按下车，走下几个台阶进屋，用力往前一扔一搡，我被重重摔在地上，裤子被摔破。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  警察又将他和两位学员分开，各自分别关押。
 </span>
</p>
<p class="p5">
 <span class="s1">
  “警察强行让我蹲在地上，面向墙，手背到头的后面。二十多分钟后，有个便衣过来审问：‘你家是哪的？好好想想，你们违法，你们闹事，谁指使你们来的？来这干嘛？’像审犯人一样，过程中没有座位，没有水喝。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  大概晚十点、十一点来钟，也不审问了，在众多法轮功学员去市政府反映情况的压力下，警察接受命令才允许他和两位学员离开。回到家已是后半夜。
 </span>
</p>
<p class="p5">
 <span class="s1">
  第二天，他才知道还有部分被抓的法轮功学员没有放。
 </span>
</p>
<p class="p5">
 <span class="s3">
  1999
 </span>
 <span class="s1">
  年
 </span>
 <span class="s3">
  7
 </span>
 <span class="s1">
  月
 </span>
 <span class="s3">
  20
 </span>
 <span class="s1">
  日迫害法轮功后，中共电视台报导，警方说天津事件没有抓人，没有打人，和平劝离。
 </span>
</p>
<p class="p5">
 <span class="s3">
  “
 </span>
 <span class="s1">
  这种公开造假、欺骗百姓，是邪党的一贯伎俩。我就是天津教育学院抓人事件的见证人。
 </span>
 <span class="s3">
  ”
 </span>
 <span class="s1">
  他说。
 </span>
</p>
<h4 class="p1">
 <span class="s1">
  “想去不去 自己的事”
 </span>
</h4>
<p class="p6">
 <span class="s1">
  天津公安局动用警察抓了法轮功学员以后，宣称这是“上边”（北京）的指示，就这样，天津及附近的法轮功学员决定25日去北京信访办上访。
 </span>
</p>
<p class="p7">
 <span class="s1">
  一位中国
  <a href="http://www.epochtimes.com/gb/tag/%E5%A4%A7%E9%99%86%E6%B3%95%E8%BD%AE%E5%8A%9F%E5%AD%A6%E5%91%98.html">
   大陆法轮功学员
  </a>
  在投稿明慧网的文章中回忆：
 </span>
</p>
<p class="p6">
 <span class="s1">
  4月24日上午，A 城一位同修（法轮功学员之间的称呼）打电话说，你们比较远，可以把消息告诉身边的同修，想去不想去，都是自己的事。
 </span>
</p>
<p class="p6">
 <span class="s1">
  于是，我就通知了全市的辅导员。大家一起交流，最后达成共识：“我们炼功，按照真善忍做好人，没有违反国家的法律，这件事虽然发生在天津，抓的是天津的大法学员，也等于抓我们一样。今天发生在天津，明天可能就会发生在我们地区。维护大法就是我们的责任，我们也应该去。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  当时还有人提议，别告诉更多同修，就参加交流的这些同修知道就行，去不去也是自愿的。
 </span>
</p>
<p class="p6">
 <span class="s1">
  随后，他们就各自去买票，因为已没有直达北京的火车，就只好乘坐路过天津的火车，参加交流的不到20人，25日到天津下车后，已有36人，因为有的家人同修知道后，也一同去了。
 </span>
</p>
<p class="p6">
 <span class="s1">
  25日夜，他们又转乘车到北京，已是26日清晨，联系上一个先前到北京的外地同修，将他们领到一个同修开的旅店，并详细地介绍了25日大上访盛况。
 </span>
</p>
<h4 class="p6">
 朱镕基的三点要求
</h4>
<p class="p6">
 <span class="s1">
  他介绍，当时去了多少人已无法计算，据一个警察说有三四万人。
 </span>
</p>
<p class="p6">
 <span class="s1">
  时任总理朱镕基接待了法轮功学员代表，并答复了代表提出的三点要求：
 </span>
</p>
<p class="p6">
 <span class="s1">
  一是立即释放天津警察抓捕的法轮功学员；二是放开对大法经书印刷的限制；三是给法轮功学员一个合法的修炼环境。
 </span>
</p>
<p class="p6">
 <span class="s1">
  朱镕基总理还说，政府一贯支持群众一切健身活动，从来没有限制过，今后也不会限制。
 </span>
</p>
<p class="p6">
 <span class="s1">
  “我们听了后，都很高兴，同时也为我们没有成为这次大上访中的一员而感到遗憾。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  4月26日早上，他们到住处附近一个公园去参加了晨炼，并于当天返回了当地。
 </span>
</p>
<p class="p6">
 <span class="s1">
  第二天，公安局国保支队教导员把他们辅导站的几位同修召集到公安局，询问“四·二五”大上访的情况，“我们毫无保留地向他讲述了我们所知道的整个情况，包括上访发生的背景等等。”
 </span>
</p>
<h4 class="p6">
 公安透露秘密调查
</h4>
<p class="p6">
 <span class="s1">
  这位教导员听了介绍，特别是听说（时任）朱镕基总理亲自接见学员代表，并给予答复后似乎也很震惊，但是他还是说：“天津的事我们不应该去参与，天津他们干预法轮功，是他们不够了解，这样的事（指诬陷法轮功和抓捕法轮功学员）在我们市是不会发生的。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  期间，这位国保教导员透露他自己就是管法轮功的，“我们已经跟踪你们几年了，近几年按照上级要求，已经对你们法轮功做过多次秘密调查，报告也已经打过多次了。”他说。
 </span>
</p>
<p class="p6">
 <span class="s1">
  “甚至为了了解你们的真实情况，我们很多人参加你们炼功，在你们的炼功点，我们都安装过微型监听器等等。报告都是我亲自写的，我们认为这个功法没有别的说头，确实很好，对祛病健身很有效果，后来我们X局长一家人都炼法轮功了。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  教导员还给法轮功总结有三多：年岁大的多；有病的多，女性多。
 </span>
</p>
<p class="p6">
 <span class="s1">
  他还说，“因为你是站长，我们还去过你们单位了解过你本人的情况，你们单位领导对你评价很高：说你人品非常好，为人友好，工作勤恳，作风扎实，廉洁自律，是个优秀干部。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  他表示：“别看你们当什么站长、辅导员，你们也不一定有我们对法轮功掌握的情况多，比如，有多少炼功点？大约多少人炼功等等？你们不一定能说清楚，这些我们都掌握。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  这位法轮功学员接着指导员的话说，“因为我们没有花名册，谁觉得好就来炼，不想炼就走，来去自由，所以我们也没必要掌握这些。我们站长、辅导员都是自愿为大家服务的，说白了就是修炼受益后，自愿给大家服务的。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  最后，国保支队教导员表示，你们看这样好不好，今后哪里出事，咱们不去管，咱们市里这一块谁要是干扰你们炼，你们就来找我。
 </span>
</p>
<p class="p6">
 <span class="s1">
  就这样，谈话在很友好气氛中结束了。
 </span>
</p>
<p class="p6">
 <span class="s1">
  1999年7月，中共迫害法轮功发生后，就是这个教导员把这位法轮功学员第一个送进了看守所。在去看守所的车上，法轮功学员问他，“你现在怎么看待法轮功？”他说，“这不是你们都往北京跑嘛，我们吃共产（邪）党的饭，我也不想往里送你，人家让先抓‘头’。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  到了看守所，教导员对看守所的所长说，给好好照顾一下，这是好人，没啥事，就是炼炼功。看守所所长笑着说，“你要这么说，你就得把人拉回去了，我这里收留的没有好人。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  教导员回应，“唉！没办法，这就是大气候。”然后他十分不好意思地看着法轮功学员说，“委屈你了，过两天，我就来接你回家。”
 </span>
</p>
<p class="p6">
 <span class="s1">
  20年过去了，今又逢“四·二五”，这位
  <a href="http://www.epochtimes.com/gb/tag/%E5%A4%A7%E9%99%86%E6%B3%95%E8%BD%AE%E5%8A%9F%E5%AD%A6%E5%91%98.html">
   大陆法轮功学员
  </a>
  表示，回忆这段历史，从中可以看出江泽民集团一意孤行迫害法轮功是蓄谋已久。当时中共高层对法轮功是比较了解的，他们中多是支持法轮功；基层警察，特别是国保部门，确实对法轮功很了解。
 </span>
</p>
<p class="p6">
 <span class="s1">
  但是，正是由于江泽民利用了中共邪党，所以把很多无辜者绑架在贼船上参与了迫害修心向善的法轮功学员。
 </span>
</p>
<p class="p6">
 <span class="s1">
  20年过去了，法轮功学员依然坚持自己的信仰，法轮功不但没有被打倒，还洪扬全世界一百多个国家和地区，这段历史中也记载了每个生命善与恶的表现。#
 </span>
</p>
