 <p>
  香港警察1号在多处以实弹射击示威者，在荃湾造成一名中学生中弹受重伤后，在台港生指控港警犯下反人类罪行，应交由国际法庭公审。前铜锣湾书店店长林荣基说，港警十·一枪杀学生，印证中共七十年始终是杀人政权。
  <br/>
  <br/>
  香港警察1日在荃湾驱离抗争者，持真枪近距离对中学生左胸开枪，学生中弹倒地，鲜血涌出命危，震惊各界。

  张俊豪强调，被开枪击中的，不过是十五、十六岁的学生，为了争取自由、民主而已。这根本是谋杀，不是自卫，港警已经完全没办法控制，没办法专业使用武器 。
  <br/>
  <br/>
  张俊豪认为，香港已形同进入“内战”状态，港警、港府犯下违反人类罪行，应诉诸国际法庭审判。
  <br/>
  <br/>
  张俊豪说：“如果他今天生命有什么事情的话，香港人绝对不会原谅这个政权，因为这个政权手上已经沾满鲜血。我相信今天开第一枪，可以燃起我们香港人的革命之火。”

  林荣基接受自由亚洲电台访问时讽刺说，印证中国共产党就是杀人政权，七十年不变：“今天是七十年前他拿了政权的第一天，今天杀人就最好，中国大陆就是杀人，他就是杀人拿了这个政权，所以他选择今天杀人是对的，等于是香港人的港殇日。”

   台湾总统府：对渴望民主的人民动用国家暴力
  </b>
  <b>
  </b>
  <b>
   只会恶化情势
  </b>
  <br/>
  <br/>
  在台湾，总统府表示，对参与抗争的香港中学生遭港府执法人员开枪致重伤，表达强烈遗憾。事实证明，对渴望民主的人民动用国家暴力，只会让情势不断恶化，最终导致悲剧的发生。呼吁香港政府当局，停止镇压手段，尽速并诚意开启社会对话，落实对港人自由民主的严肃承诺，以化解不必要的冲突。
  <br/>
  <br/>
  主管两岸事务的大陆委员会发言人邱垂正呼吁港府：“一再以失控的国家暴力打压自由民主，只会落入恶性循环，应该尽速理性倾听，及妥善回应民众诉求，才是促使香港社会回归稳定的最佳途径。”
