 <p>
  台湾高雄市长韩国瑜18日会晤美国在台协会主席莫健的时候说，他决定取消访问美国，理由是要忙着下乡拼选举。有学者告诉本台，这说明韩国瑜是“亲中拒美”。
  <br/>
  <br/>
  美国在台协会（AIT）主席莫健16日会见民进党总统参选人蔡英文之后，18日南下高雄会见国民党总统参选人韩国瑜。刚刚宣布请长假跑选举行程的韩国瑜，销假半天，西装笔挺地迎接莫健，不过却告诉莫健，他不去美国访问了。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  <b>
   韩国瑜新衣新鞋见
  </b>
  <b>
   AIT
  </b>
  <b>
   主席
  </b>
  <b>
  </b>
  <b>
   告知取消访美
  </b>
  <br/>
  <br/>
  韩国瑜开口就对莫健说：“我们台湾小孩子过新年的时候，大家都会穿新衣、穿新鞋，快快乐乐过新年，所以莫健大使一来，我今天就穿了新衣、新鞋，用了新的领带，代表心情是非常开心的。”
 </p>
 <p>
 </p>
 <p>

  <br/>
  莫健开场白则秀了一段中文：“市长，我是高高兴兴地到高雄来，我是第三次到这儿来。”
  <br/>
  <br/>
  韩国瑜对莫健说，中华民国的对外关系中，美国是最重要且坚定不移的盟友，他要致上最高感谢。莫健则说，《 台湾关系法》生效四十年至今，此时是台美关系最强健时期。
  <br/>
  <br/>
  两人会面开放短暂问候谈话和互赠礼物给媒体拍摄。随后闭门交谈时，韩国瑜对莫健说，取消访美。
  <br/>
  <br/>
  此事引发哗然。韩国瑜受访表示，他向莫健表达总统大选只剩80多天，台澎金马都还没完全跑完第一趟，时间压力非常大。
  <br/>
  <br/>
  <b>
   韩国瑜
  </b>
  <b>
   :
  </b>
  <b>
   莫健接受他取消访美的理由
  </b>
  <br/>
  <br/>
  韩国瑜说：“我也跟莫健大使说明这一趟美国之行，可能因为时间压力，我没有办法成行。他当然理解！因为他知道只剩85天左右，高雄市政工作非常繁忙，没有办法请假太长，他可以了解我一站一站、一个地方一个地方来奔跑、拜票、倾听、学习跟关怀之旅，他也能了解，我可以感觉得到，他也能够接受我的观点，我也再一次表达有机会明年一定到美国拜访。”
 </p>
 <p>
  韩国瑜说，若有时间，妻子李佳芬可代表他去美国拜访侨界。
  <br/>
  对韩国瑜决定在大选前取消访美行程，与会的韩国瑜国政顾问团两岸小组召集人赵建民会后接受自由亚洲电台访问说：“市长跟美国方面，并不欠缺管道，在当前、未来也一样，会是很畅通的管道。”
  <br/>
  <br/>
  至于莫健如何回应韩国瑜取消访美的说法，赵建民说，这不是他该回答的问题。
 </p>
 <p>
 </p>
 <p>
  <div class="image-inline captioned" style="width:1500px;">

  <br/>
  <b>
   莫健非常关心韩国瑜对香港问题的态度
  </b>
  <br/>
  <br/>
  赵建民提到，莫健关心韩市长未来对台湾的经济发展政策、两岸政策、台美关系，尤其主动谈到香港局势。
  <br/>
  <br/>
  赵建民说：“莫健大使很关心香港现在的（局势），也希望知道市长的立场。市长论述得很深入。他首先强调台湾跟香港不一样，不能类比，也反对当前有一些政治人物，刻意把香港跟台湾类比，市长特别说我们是中华民国，是主权完整的主权国家，香港则是已经在1997 年回归中华人民共和国是特别行政区。”
  <br/>
  <br/>
  赵建民说：“第二点，市长也特别强调，我们是反对‘一国两制’的，香港的老百姓走上街头，也反应出来‘一国两制’并不能适用在台湾。第三点，市长觉得北京跟香港政府都应该更主动去了解他们走上街头得民意之所在，了解问题，建立沟通的管道。”
  <br/>
  <br/>
  赵建民表示，韩市长强调，香港过去在亚洲四小龙是很成功的范例，台港关系非常友善。希望香港早日恢复稳定。赵建民认为，美国肯定关切香港会朝更多元民主方向发展，在这方面，韩市长跟美国方面有共同的语言。
 </p>
 <p>
  至于蔡英文总统日前会见莫健呼吁美方和台湾能尽速洽签台美双边自由贸易协定，以及中美贸易战的影响，赵建民表示，莫健和韩国瑜会面都未触及这方面的问题。
  <br/>
  <br/>
  台媒报导，蔡英文18号被问到对韩国瑜取消访美一事的看法，只灿烂一笑回答“喔”，未多作回应。
 </p>

   绿委：韩是首位表态

  <b>
   的总统参选人
  </b>
  <br/>
  <br/>
  民进党籍立法委员王定宇18号接受自由亚洲电台采访表示：“他（韩国瑜）可能是中华民国史上第一个要选总统的人，只去中国拜访，却不访美国。”
  <br/>
  <br/>
  王定宇透露，他九月份去华府两趟，到了参众两院，参访几个重要的美方智库，并参加台美国防会议，与美国政府及民间部门接触过程中，完全没有感受到韩国瑜要访美的迹象。韩阵营似乎没有进行访美的联系，当时莫健对他提到自己将访问台湾，拜访各界，也会到高雄市府与韩市长会面。
  <br/>
  <br/>
  王定宇说，韩国瑜以要访美为由向高雄市府请长假，却变成请假是为了跑竞选的地方行程，可能像他的国政顾问所说是“藏拙”，但把国际关系当儿戏的领导人非常危险。
  <br/>
  <br/>
  王定宇说：“台美关系是重中之重，他当市长跑港澳，进香港中联办，也见了国台办主任，却把访美当儿戏，正式的台湾总统候选人如此轻率，呈现出的态度就是‘亲中拒美’的态度，对国际事务不嫺熟，怕出丑就取消访美，这样处理会伤害两千三百万台湾人民的利益。”
  <br/>
  <br/>
  <b>
   学者猜测韩阵营与美方乔不拢
  </b>
  <b>
  </b>
  <b>
   赴美见不到高官遂取消
  </b>
  <br/>
  <br/>
  台湾师范大学政治学研究所教授范世平向本台直言，韩国瑜取消访美会使他的选情失分：“那是很大失分啊！怎么会取消咧？我蛮讶异。”
  <br/>
  <br/>
  对韩国瑜表示是为了下乡跑行程，范世平认为：“当然不是理由，我觉得他应该没有拿到对方要给他的位阶！去不了，问题就大了，美国连这点样子都不愿意做。”
 </p>

  范世平分析，韩国瑜可能是见不到想见的美方官员、议员层级，干脆不去，免得被拿来跟蔡英文上届选总统之前的访美规格比。
 </p>
 <p>
  范世平说，就他了解，韩国瑜今年四月访美，有美方前外交官就认为韩国瑜在两岸关系上讲那一套“一中各表”，根本行不通，美方也对韩的亲中立场十分担心。
  <br/>
  <br/>
  范世平说：“他（韩国瑜）过去跟北京关系太复杂，大概十年在大陆，关系是什么？美国驻北京大使馆应有所了解。”
  <br/>
  <br/>
  <b>
   独派讥笑韩：美方已泄题
  </b>
  <b>
  </b>
  <b>
   韩发觉不会作答落跑
  </b>
  <br/>
  <br/>
  独派政党台湾基进党则第一时间在脸书嘲笑韩国瑜说，媒体都已经“泄题”报导美方对韩国瑜会提出五个问题，包括两岸政策、对中国依赖度、对军购态度等等，希望韩国瑜到美国可以好好回答。
  <br/>
  <br/>
  台湾基进党新闻舆情部副主任张博洋觉得，韩国瑜像是打开考卷后发现一题都不会写，所以选择放弃以免尴尬。
  <br/>
  <br/>
  张博洋说：“他（韩国瑜）现在就像一个不会写考卷的小孩，离开考场跑去找韩粉取暖，跟莫健主席说要下乡 、很忙，事实上都是推拖之词。这些问题答案对韩国瑜、对国民党都很好回答，但答案一定都不会让美方满意，干脆不去。”
  <br/>
  <br/>
  张博洋说，对韩粉来说，韩国瑜去不去美国都不重要，因为韩国瑜不论做什么，韩粉都支持。但是对很多台湾人而言，韩国瑜没有通过美国这一关，他的两岸政策没办法让人信任。
