 <p>
  中国在国庆阅兵当天高调展示多个新型导弹，引发美国方面的警惕。美国国防部印太安全事务助理部长薛瑞福（Randall Schriver）表示，美中竞争源于双方对地区和平架构看法的不同，美国将保持在危机爆发时的威慑和打击能力。
  <br/>
  <br/>
  针对中国星期二举行的盛大国庆阅兵式，薛瑞福形容，那是“一场令人印象深刻的展示”。他认为，阅兵式的展示情况与预期一致，旨在对内和对外发出信号，美国国防部随后会作深入分析。
  <br/>
  <br/>
  与此同时，有美国军事专家注意到，中国解放军在国庆阅兵式上展示的多种新型装备，重点围绕着作为中国核威慑支柱力量的导弹，其中包括射程在1800-2500公里之间、据称可以突破美国导弹防御系统的高超音速导弹东风-17；还有射程约达两千至三千公里、速度快精度高、主要针对美国航空母舰战斗群的巡航导弹长剑-100。
 </p>
 <div>
 </div>
 <div>
  美国海军战争学院中国海事战略研究院教授艾力信（Andrew S. Erickson）在1日发表的博文指出，中国这种以导弹为中心的军事发展可能是侵蚀美国在亚洲影响力的最大因素。
  <br/>
  <br/>
  美国国际评估和战略中心(International　Assessment　and　Strategy　Center)的军事专家费学礼（Richard Fisher）接受本台记者采访时解释说，中国致力于开发中程导弹的原因是：“要发现和防御飞行速度极快的洲际和巡航导弹很难。中国现在发展出低空飞行、超音速导弹和核导弹列车，这都预示着中国希望拥有与美国和俄罗斯数量一样的弹头，最后甚至可能超越美国。”
  <br/>
  <br/>
  他举例说：“我们看到中国在阅兵式上所展示的东风-17常规导弹装载着世界第一个高超音速飞行器（hypersonic glide vehicle）。如果将东风-17和东风-16型弹道导弹配置在一起的话，射程是可以到达冲绳和台湾，也就是说，中国人民解放军是可以向目标进行高空、低空、极快的打击，让美国的导弹防御系统很难承受饱和攻击，而这套防御系统也是台湾政府斥巨资获得的。”
 </div>

 <div>
  美国哥伦比亚大学政治学博士王军涛则认为，中国虽然在军事硬件上具有不错的优势，但：“一个国家的军事战斗力除了硬件外还取决于软件，例如操控和综合使用武器的人的熟练程度，这对新型军事国家来说很难。我想中国不会马上威胁到美国的军事优势。中国在武器硬件发展是强的。如果美中双方都饱和射击的话，中国在硬件上是占很大的优势。”
  <br/>
  <br/>
  值得注意的是，中国在国庆日“大秀铁拳”的举动似乎解释了中国为什么不愿加入美俄中导条约（INF），还有美国为什么决定退出该条约，同时宣布加快全面开发新常规导弹。
  <br/>
  <br/>
  艾力信在文章中说，曾任美国太平洋司令部司令的哈里斯上将(Adm. Harry B. Harris Jr.)于2017年在美国国会作证时表示，“如果中国也是《中导条约》签署国的话，该国95%的弹道导弹和巡航导弹都是违规的”。美国退出《中导条约》意味着华盛顿可以发展射程超过5500公里的导弹来反制他国威慑。
  <br/>
  <br/>
  费学礼对此表示：“由于受到《中导条约》的约束，美国多年来眼睁睁看着中国制造大量的洲际和巡航导弹。美国现在正与中国进行全面的导弹军备竞赛，美国将全力保卫军队、盟友。美国必须防止中国向台湾、日本和韩国挑起战争。”

  美国国防部印太安全事务助理部长薛瑞福也提到，中国的军事发展正试图侵蚀美国的军事优势，美中竞争的根源在于对全球安全架构看法的不同，美国会以维持自由世界秩序为目标与中国进行竞争，在危机爆发时保持威慑和打击能力。
  <br/>
  <br/>
  据了解，美国国防部今年6月新设立的副助理部长职位会由通晓中文的前海军陆战队美中研究小组主任斯布拉吉亚（Chad Sbragia）担任。他将负责所有涉及中国的事务，包括在战略、计划、政策和双边安全关系的发展与落实等议题上，向国防部高层提供建议。
