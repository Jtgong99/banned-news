 <p>
  美国研究创新问题的专家最近发表的新书认为，中国科技领域全面创新的时代正在展开，并可能10年内在诸多重要科技领域占据世界主导地位。但有美国商业界和学术界人士认为，中国扶持国内产业的政策可能阻碍其公司的持续发展，而中国的生产率目前仅为美国的百分之二十五，需要几十年才能赶上美国目前的水平。
  <br/>
  <br/>
  全球创新问题专家丽贝卡·范宁10月10日在美国民间智库布鲁金斯学会向学者和媒体介绍了其新书《中国的科技巨擘》，并表示，她研究了中国在人工智能、电动汽车、金融科技、共享经济、人脸识别、无人机、按需送货等领域的快速发展，认为中国企业在这些方面近几年来的发展,已经超越了美国的科技公司。作者表示，她的新书还凸显了源于中国的新商业模式，即：多功能超级应用软件、移动支付、实时直播中购买虚拟商品、人工智能新闻和视频信息、社交媒体上的商店等。
  <br/>
  <br/>
  作者还指出，如今在中国风险投资的规模几乎与美国的风险投资市场相当。而来自中国的专利申请目前已占世界总量的22%；中国的阿里巴巴和腾讯已进入全球市值最高的10家公司之列。同时，中国政府的研发支出也正迅速赶上美国，中国已在学术论文和速度最快的超级计算机方面处于世界领先地位。
 </p>
 <div>
 </div>
 <div>
  范宁认为，现在中国公司已不再模仿美国了，恰恰相反，倒是美国科技公司在一些方面开始模仿中国公司。例如，美国的星巴克咖啡连锁公司近来模仿了中国“瑞幸咖啡”(Luckin Coffee)的送货上门服务模式；同时，中国的一些科技公司也正在走向全球，例如，下载流行视频的分享应用程序TikTok（海外版抖音）已经在全球有很多用户，这个数字娱乐应用很受欢迎。
  <br/>
  <br/>
  美国公司模仿中国的例子还包括，脸书公司在私人群组信息方面模仿微信的运营方式，它还模仿了视频应用程序抖音；而亚马逊公司则在电子零售方面，模仿了阿里巴巴。中国的百度、阿里巴巴和腾讯等这20多年来已经发展成上市的科技巨头，并拥有数亿用户。而它们的创始人在中国已成为令人羡慕的人物，并激励着下一代。
 </div>

 <div>
  范宁表示，十年前，中国在科技创新方面超越美国的可能性几乎超出人们的想象，但如今中国一些科技公司为自己打造了名声，其中一些在好几个竞争激烈的科技领域，颇显竞争实力，有些甚至在类似移动支付等方面占据着全球领先地位。她说：
  <br/>
  <br/>
  “一个新的时代正在兴起，在这个新时代不再仅仅是有关中国公司抄袭他国公司的科技成果。最初成立的一批中国科技公司，都是由在美国最好的大学和公司里学习和获得经验的中国留学生，回国后模仿美国公司建立的企业。苹果、脸书，亚马逊、谷歌、雅虎等这些都在中国被模仿。而目前，中国一些新成立公司的运作方式，被美国公司所模仿。”
  <br/>
  <br/>
  范宁推测，中国有一个要在未来10年内在大部分技术领域占主导地为的总体规划。中国还力争在第五代移动通信领域里领先。
  <br/>
  <br/>
  范宁介绍其新书之后，两位美国学者发表了他们对范宁书中一些观点的看法。布鲁金斯学会约翰·桑顿中国中心全球经济与发展高级研究员大卫.杜勒  (David Dollar) 表示，他当然对中国科技公司这些年来取得的发展和成就很了解，但是，这并不意味着中国在经济实力和科技发展上已经成为超人。例如，中国经济的生产率目前仍然仅为美国生产率的百分之二十五。而提高生产率幅度则需要很多年。他说：
  <br/>
  <br/>
  “我当然明白中国在一些科技领域里取得了显著的进展，但是，中国要想在生产率上赶超美国，还需要将现在的生产率提高四倍。而即使美国的生产率停止不前，这也还需要几十年才能达到。而美国当然不可能在生产率上停止不前。中国经历了40年的发展才得以把过去为美国百分之四的生产率，提高到今天的百分之二十五。”
  <br/>
  <br/>
  美国非政府机构 美中贸易全国委员会（U.S.-China Business Council）主席克雷格.艾伦 （Craig Allen）则指出，中国政府试图在各方面支撑国有和本土企业，但由政府主导的工业发展政策最终阻碍国内企业的持续发展。他说：
  <br/>
  <br/>
  “中国的2049发展计划和国家干预经济的工业政策，与过去日本经济产业省(METI, Ministry of Economy, Trade and Industry) 的做法很相似，但后来发现，这种由政府主导的发展模式其实最终反而起到了阻碍国内公司发展壮大的作用。很多最终发展成全球性大公司的日本公司并不是由于日本经济产业省的协助而成功的，而是靠自己成功的。中国政府应该好好研究日本的经验。”
  <br/>
  <br/>
  丽贝卡·范宁不仅是全球创新问题专家，她还是美国风投研究机构“硅谷之龙”的创始人，而她的第一本有关中国科技发展的书，也题为《硅谷之龙》，该书于2008年出版。
  <br/>
  <br/>
