  请愿十多人被抓
  <br/>
  4月22号，内蒙古乌拉特中旗的100多名牧民前往旗政府请愿，要求当局兑现承诺，发放“边民补贴款”，但遭到保安和特警拦截，10多人被抓。

  <a href="https://www.ntdtv.com/gb/“六四酒案”.htm">
   “六四酒案”
  </a>
  三当事人 被强戴监控手表
  <br/>
  成都
  <a href="https://www.ntdtv.com/gb/“六四酒案”.htm">
   “六四酒案”
  </a>
  3名被判缓刑的当事人，出狱后被强迫佩戴即时监控手表，24小时受到监控。当局要求他们即时对手表充电，以保持手表正常运作。如果因为断电而出现监控空白，就将收监羁押服刑。
 </p>

  <a href="https://www.ntdtv.com/gb/疫苗受害家长.htm">
   疫苗受害家长
  </a>
  何方美不认罪 遭持续羁押
  <br/>
  大陆毒
  <a href="https://www.ntdtv.com/gb/疫苗受害家长.htm">
   疫苗受害家长
  </a>
  何方美，被拘留已经50多天，4月23号，她的丈夫李新透露，律师当天在看守所会见了何方美，据说她的状态还好，未被批捕，当局要她认罪才释放她，但她坚持自己无罪。
 </p>

  <a href="https://www.ntdtv.com/gb/709案.htm">
   709案
  </a>
  维权人士吴淦受酷刑后遗症折磨
  <br/>
  <a href="https://www.ntdtv.com/gb/709案.htm">
   709案
  </a>
  中因拒不认罪而遭重判8年的维权人士吴淦，早前在天津公安局曾遭酷刑折磨，4月23号，他的好友传出消息说，目前正在福建监狱服刑的吴淦，酷刑后遗症严重，腰部、颈椎部因酷刑受伤，现在疼痛难忍。
