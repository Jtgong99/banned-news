<h4 class="p2">
 <span class="s1">
  任正非放软姿态？
 </span>
</h4>
<p class="p3">
 <span class="s1">
  华为集团创办人任正非
 </span>
 <span class="s3">
  21
 </span>
 <span class="s1">
  日上午在深圳召开的国内记者会上，一方面强调“华为的实力足以撑过这关”，另一方面也不断提醒“各方冷静”。
 </span>
</p>
<p class="p3">
 <span class="s1">
  他说，美国的“90天宽限期”意义不大。他强调，华为的生产能力还是很大的，不会超出公司的负增长，不会对产业发展做出伤害，“我们不会轻易狭隘地排除美国芯片，要共同成长”。他也表示，如果出现供应困难的时候，华为有备份，能够挺过困难期。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “大家要骂就骂美国政客，这件事不关美国企业什么事。”任正非
 </span>
 <span class="s1">
  还表示，爱华为不一定买华为手机，
 </span>
 <span class="s1">
  “像我的小孩就是不爱华为，因为他就爱苹果……我们家人现在也都用苹果的手机，苹果的生态很好，家人出国我还送他们苹果电脑，所以不能狭隘地认为爱华为就得用华为手机。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  经济学博士、时事评论员李松筠表示，任正非的这次讲话明显对美国放软了姿态，希望美国可以像对中兴一样取消禁令。
 </span>
</p>
<p class="p3">
 <span class="s1">
  “因为华为之前已经储备了很多芯片，可能会撑个半年以上，所以华为利用这个缓冲期，一方面希望像中兴公司一样，通过中美高层对话或者习川会，达成一个和解的协议；另一方面，华为对美国企业大加赞美，可能也是希望通过这些企业，去说服川普政府放松禁令，放华为一马。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  她对大纪元记者说：“华为并非像大陆媒体宣传的那样，是因为技术太强，而遭到美国的嫉恨。如果华为技术那么强，就用不着费力气去储备那么多芯片了，也用不着把自己研发的芯片压在箱底当‘备胎’了，也用不着表态说‘华为不会轻易狭隘地排除美国芯片，要共同成长’等等。”
 </span>
</p>
<h4 class="p3">
 <span class="s1">
  致命的打击
 </span>
</h4>
<p class="p3">
 <span class="s1">
  大纪元专栏作家、著名时事评论员章天亮博士认为，芯片及谷歌的“双重断供”对华为来说是致命的打击。
 </span>
</p>
<p class="p3">
 <span class="s1">
  获得信息工程博士学位的章天亮在其自媒体上介绍，搞集成电路设计的人都知道摩尔定律，“摩尔定律指的是在单位面积上能够集成晶体管的数量大概每十八个月会翻一番，也就是说晶体管越做越小，晶体管之间的距离也越来越小。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  他说：“现在这个集成芯片能够达到最大的精度是五纳米，就是两个晶体管之间线路的宽度是五纳米，一纳米是一百万分之一毫米。”
 </span>
</p>
<p class="p3">
 <span class="s1">
  什么样的工艺才能够把这个电线刻到这种程度？章天亮介绍说，实际上有一个非常先进的设备叫光刻机，这种设备只有荷兰有卖，但却不卖给中国，所以如果中国真的自己能够设计芯片，也根本无法生产。
 </span>
</p>
<p class="p4">
 <span class="s1">
  他表示，做一个芯片需要几十个企业的配合，从整个芯片的设计，到封装，中间要经过大概二千到五千道工序。
 </span>
</p>
<p class="p5">
 <span class="s1">
  另外，芯片试验，需要有一个仿真软件，如果没有这个仿真软件，就无法完成对芯片的测试，而中国没有那个仿真软件。
 </span>
</p>
<p class="p5">
 <span class="s1">
  “种种迹象表明，中国如果想做出这样芯片的话，不是五年十年的事情。像
 </span>
 <span class="s6">
  Intel
 </span>
 <span class="s1">
  这些比较成熟的芯片公司，都是几十年的积累，每年投入的研发费可能上百亿美元，然后不断地更新换代。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  他说：“如果五年十年的话，你都不知道比别人晚了多少代了。所以说，有些人说华为可以通过自己设计芯片，来度过这个寒冬，那是根本就不可能的。”
 </span>
</p>
<p class="p5">
 <span class="s1">
  此外，谷歌的安卓操作系统，是除苹果之外几乎所有的手机厂家都在使用的操作系统，如果谷歌不提供系统支持，那么华为手机在软件上也失去了安卓系统升级的可能。
 </span>
</p>
<p class="p5">
 <span class="s1">
  “安卓系统如果将来不授权给华为，华为可真就死掉了，因为你不可能自己花那么大的力气去开发一个操作系统。那等于是你从头到尾，从硬件到软件完全从头做起，这几乎是一个不可能的任务。”
 </span>
</p>
<p class="p1">
 章天亮表示，现代社会实际上是一个高度分工、高度协调的社会，就像苹果公司有超过二百个部件的供应商，在不同的国家。“苹果自己根本不生产手机，都是代工。所以如果美国不向华为提供硬件、软件的话，华为等于要把所有那些国家做的所有最好的技术，都得在中国复制一份，才能够做出这样好的手机来，所以我觉得这对华为来说，真的是致命的打击。”
</p>
<p class="p4">
 <span class="s1">
  他说：“所以华为现在唯一能寄予希望的就是跟美国达成协议，让美国继续卖芯片给他。当然美国对华为的这种盗窃别人知识产权、盗窃别人数据的行为，深恶痛绝。但是华为从美国采购大量芯片的同时，也给美国创造了一些就业机会，所以美国一些芯片制造商，在华为被封堵的时候，实际上他们的股价也在下降。”#
 </span>
</p>
