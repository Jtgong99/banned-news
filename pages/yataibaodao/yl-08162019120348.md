 <p>
  对“反送中”示威活动一直保持沉默的香港首富李嘉诚，8月16日首度公开表态，在香港多家报纸头版刊登广告。有评论认为，在北京压力下，李嘉诚不得不以登广告的方式表达立场，但他希望既不得罪北京、又不得罪港人，两边都讨好。
  <br/>
  <br/>
  香港多家报纸8月16日头版刊登香港首富李嘉诚以“一个香港市民”的名义的整版广告。广告共有两款，其中《东方日报》刊登的广告内容非常简短，只有一句话“正如我之前讲过：‘黄台之瓜，何堪再摘’”。《香港商报》头版刊登的广告字数稍多些，围绕中间的反暴力标志，上写“最好的因可成最坏的果”，左写“爱自由．爱包容．爱法治”，右写 “爱中国．爱香港．爱自己”，下面是“以爱之义 止息怒愤”。
 </p>
 <div>
 </div>
 <div>
  前香港记者协会副主席、独立评论人士谭自强8月16日接受本台记者电话采访时认为，李嘉诚显然是受到了来自北京方面的压力，不得不以登广告的方式公开表态。
  <br/>
  <br/>
  “根据香港目前的行情，买一个头版全版的广告至少要30万港元。香港目前每天的报纸大概有15份。你想想15份报纸，每份平均30万一个版面的话，有些报纸更贵了，像《东方日报》、《苹果日报》是更贵的，登一天最少要5、6百万。你无端花5、6百万登这么大规模的广告，如果没受到压力的话，你可能是神经病了，对不对？”
  <br/>
  <br/>
  谭自强认为，李嘉诚刊登的广告的内容十分巧妙，既不想得罪北京、又不得罪港人，两边都讨好。
  <br/>
  <br/>
  “因为这个广告有双重的意义。第一层就是说群众不要上街再闹了，黄台之瓜不能再摘下去，摘下去瓜都没了。这是第一层意思。可是相反的，像我们历史读得久的，很了解当年武则天跟她儿子章怀太子这个废太子的故事。你就发觉其实这个广告第二层意义，很隐晦的意义是说，妈妈不要再杀儿子了。香港这种情况，大家看得很清楚，妈妈当然是指北京，儿子当然是指香港。”
 </div>

 <div>
  除了刊登广告，李嘉诚还在8月16日透过发言人向媒体表示，他认为香港长期繁荣稳定系于“一国两制”行稳致远。今日香港，要停止暴力，坚守法治 。大家一定要以爱之义，止息怒愤，对“一国两制”，以谦和而珍之。对于为何刊登两个不同广告，李嘉诚表示，基于目前香港形势复杂，难以用单一语言或沟通方法回应。谈到对政府和年轻人的看法，李嘉诚说，现时年轻人给政府的声音和讯息震耳欲聋，政府已在绞尽脑汁。他投放资源在青年工作，永不后悔，因为投资青年，就是投资未来。不要让今天的激情，成为明天的遗憾。
  <br/>
  <br/>
  美国“波士顿港澳之友社”联合发起人谢中之8月16日向本台表示，他并不清楚李嘉诚的“黄台之瓜，何堪再摘”广告，到底是要中共停手还是要香港年轻人停止抗争，但他希望李嘉诚的表态是出于自己的良知。
  <br/>
  <br/>
  “李嘉诚跟中共关系挺深。我年轻的时候就知道，李嘉诚对他人造花塑胶厂的工人非常刻薄，可以说是为富不仁的。我希望李嘉诚现在说的话，是发自良心。”
  <br/>
  <br/>
  李嘉诚8月16刊发的另一则反暴力广告也引发舆论关注。有网民发帖说，该广告前八句的最后一个字连起来是：“因果由国，容港治己。” ，隐晦的指出香港近日之乱来自北京的干涉，应该允许港人自治。
  <br/>
  <br/>
  (记者：林坪 责编：申铧 网编：郭度)
