 <p>
  中国文革时期被称为北京五大造反派之一的聂元梓上周去世，被中国政府要求立即火化。这位文革中的风云人物到底经历了什么？
  <br/>
  <br/>
  聂元梓于8月28日（周三）在北京去世，享年98岁。与她当年在文革中暴得大名不同的是，她身后清凉。
  <br/>
  <br/>
  据香港媒体《星岛日报》报道，聂元梓的家人透露，她去世后，中国政府要求将她的遗体迅速火化。在火化前，仅有家人和几位好友向她的遗体道别。中国新浪、搜狐等门户网站刊登了她去世的消息后，又迅速删除了新闻页面。
  <br/>
  <br/>
  聂元梓出生于1921年，1930年代末就加入中国共产党，并有在延安工作的经历。1966年文革开始时，聂元梓担任北京大学哲学系党委书记。
  <br/>
  <br/>
  1966年，标志文革爆发的《五一六通知》发布后，聂元梓与哲学系另外6位教师在北大食堂张贴了名为《宋硕、陆平、彭珮云在文化革命中究竟干些什么？》的大字报，号召把社会主义革命进行到底，被毛泽东称为第一张马列主义大字报。
  <br/>
  <br/>
  聂元梓很快成为北大文革的主要领导，并与清华大学的蒯大富等人被并称为北京造反派五大领袖。她曾组织攻击朱德、邓小平等中共领导人。
  <br/>
  <br/>
  1969年，聂元梓当选为中共第九届中央委员会候补委员后不久就在政治上失势，被隔离审查。文革后，聂元梓被捕入狱，于1984年保外就医。
  <br/>
  <br/>
  聂元梓在80年代经商致富，于2005年在香港出版了《聂元梓回忆录》，对文革有所反思。
  <br/>
  <br/>
  美国《纽约时报》2014年刊登聂元梓朋友的回忆文章，其中提到，聂元梓晚年总结说，文革是毛泽东一个人发动，但全体党中央委员都要负责。
  <br/>
  <br/>
  聂元梓还表示，她当年牵头写那张大字报并不重要，关键是毛泽东高抬了这张大字报，导致了她后来所有的遭遇。对于她在文革中受到的政治迫害，聂元梓表示原谅，认为一切主要是由形势使然。
  <br/>
  <br/>
