 <p>
  本周六，旅居美国旧金山、洛杉矶两地的数百位中国大陆和香港人士，举行集会和游行，声援港人“反送中”运动。他们表示，正密切关注香港日益恶化的局势，警告中国政府，不要让六四屠杀在香港重演。
  <br/>
  <br/>
  17日中午，三十多名来自中国大陆的华裔人士在旧金山中国领事馆前集会，与此同时，约三百名香港人在旧金山湾畔的一个广场举行了集会。
  <br/>
  <br/>
  在中领馆前的集会人群喊出了“支持香港，香港加油，没有暴徒、只有暴政……”等口号。
 </p>

 <p>
  本次活动的组织者之一赵昕，是前89学运领袖。他表示，香港局势日益恶化，他们不会让香港人民孤独的抗争。
  <br/>
  <br/>
  赵昕警告中国政府，“香港人民的五大诉求，绝对不是港独，绝对不是你们栽赃陷害的暴力。中共暴政，如果你们敢于像89年天安门广场一样的血腥镇压香港市民，那这一次你们就必然自掘坟墓。全世界都会对你们采取全面的制裁，不出十年二十年，你们就自行死亡。”
  <br/>
  <br/>
  去年来到美国的政治异议人士赵常青，在集会上发表演说：“我们不希望今天的香港沦落为‘六四’屠杀下的北京，我们不希望香港沦落为‘六四’屠杀后的中国。我们希望香港人民能够坚持下去，抗争到底。希望大陆人民，尤其是中国青年，不要受官方的说教洗脑。香港这场运动是争取香港民主自由，就像我们要争取大陆的民主自由一样。中国公民，尤其是中国青年，觉醒起来，声援香港，支持香港，这也是支持大陆中国民主事业。”
 </p>

  刚刚从中国西安来的高喜梅女士告诉记者：“现在香港人民的抗争，大陆人民看到的都不是真相，我希望大陆人民多一点渠道了解香港的现状，不要被大陆媒体所蛊惑。香港人民争取正义自由，是为我们大陆人民打前站，我希望香港加油，我们大陆同胞要多多支持。”
  <br/>
  <br/>
  前89学运领袖方政的女儿方希冉，今天陪父亲参加声援港人“反送中”运动。她说，“我很支持香港的这些抗议者。香港的这些抗议者很勇敢。希望中国人和在美国的中国人和其他国家的中国人，都能够支持他们。”方希冉以全优的学业成绩已经被卫斯理女子学院录取，一个星期后她将到美国东部上学。
  <br/>
  <br/>
  洛杉矶的声援活动也是大陆人、香港人分头举行。一百多位大陆人在中领馆前集合，大约四百名香港人在海边的一个广场上集会。

  <br/>
  本次活动的组织者之一刘燕林接受记者电话采访时表示，“中国国内，尤其是年轻人，是非常支持香港的。但是他们在国内发不了声，他们就委托留学生，或者是过来旅游的大陆华人，表达他们对香港的支持。这些声音今天都能够发出来，大家也很兴奋，觉得为香港的自由和民主奉献了力量。”
  <br/>
  <br/>
  洛杉矶大陆人支持港人“反送中”的活动，从17日上午10点开始，一直延续18日上午。17日中领馆前的集会结束后，他们高呼“海外华人，支持香港；撑香港，反送中”等口号，开车在市内华人商业区巡游。接下来，他们又驱车四个钟头，前往拉斯维加斯。当晚返回洛杉矶“自由雕塑公园”。18日上午他们在公园的“六四纪念碑”前举行了“不能让8964在香港重演”的集会。
