 <p>
  【新唐人北京时间2019年06月05日讯】1. 30年前6月4日 波兰民众选举成功
  <br/>
  30年前6月4日的大屠杀，拖延了中国迈向民主的机会。但是30年前的同一天，波兰成功举行了二战后的第一次民主选举，摆脱共产统治、选出了首任民选总理马佐维斯基，使波兰成为东欧民主化先驱。周二，前波兰总理、现欧洲理事会主席图斯克，还有马佐维斯基、华勒沙、杜达等多位波兰前任及现任领导人，悉数出席在波兰格但斯克造船厂旁的欧洲团结中心，举办的盛大纪念仪式。他们共同签署“自由和团结宣言”并切下纪念蛋糕。
 </p>
 <p>
  2.
  <a href="https://www.ntdtv.com/gb/巴黎共和国广场.htm">
   巴黎共和国广场
  </a>
  现“
  <a href="https://www.ntdtv.com/gb/坦克人.htm">
   坦克人
  </a>
  ”
  <a href="https://www.ntdtv.com/gb/纪念六四.htm">
   纪念六四
  </a>
  <br/>
  6月4日，法国人权团体“基督徒”废除酷刑行动，在巴黎市中心的共和国广场放置了一个坦克模型，前面是一位身穿白衣的女士。这画面不言而喻，是在模仿30年前中国六四的著名照片“
  <a href="https://www.ntdtv.com/gb/坦克人.htm">
   坦克人
  </a>
  ”。主办方表示，每年的这一天，都要铭记发生了什么，要将这场大屠杀暴露在所有的聚光灯下。
 </p>
 <p>
  3. 澳大利亚昆士兰
  <a href="https://www.ntdtv.com/gb/罕见降雪.htm">
   罕见降雪
  </a>
  <br/>
  6月4日前夕，澳大利亚东部大规模降雪，处在亚热带地区的昆士兰州也罕见被白雪覆盖。处在新南威尔士州的悉尼也受大雪影响，风雪阻碍了交通。社交媒体上，许多澳洲人分享降雪后的图片。
 </p>
 <p>
  4. 美国出台对
  <a href="https://www.ntdtv.com/gb/古巴.htm">
   古巴
  </a>
  新的旅行禁令
  <br/>
  美国财政部日前出台了对美国人去
  <a href="https://www.ntdtv.com/gb/古巴.htm">
   古巴
  </a>
  的新的旅行禁令。被称作“人对人”的教育性和文化性的团体出访，不再被允许。而该项目早在2014年12月美古两国恢复正常邦交前就已存在。此外，私人或企业前往古巴的飞机或游船，也不再会获得许可。美国解释此举是应对古巴破坏西半球稳定，特别是支持委内瑞拉独裁的马杜罗政权。
 </p>
