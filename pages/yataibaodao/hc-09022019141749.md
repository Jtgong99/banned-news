 <p>
  周日，美中两国新一轮关税正式实行，标志着贸易战上了一个台阶。一年多以来，双方不仅在谈判桌上运筹帷幄，也在舆论圈轮番上演舌战。被网友戏称为“战略忽悠局”政委的中国知名教授金灿荣周一就双边贸易问题发文，批评社会中存在的两类人，并重申中国一定要斗争到底。
  <br/>
  <br/>
  9月1日，美国政府正式对价值约1250亿美元的中国商品加征15%的关税，其中包括鞋子、尿布和乳制品等日常用品。中国同日作出反制，开始对750亿美元美国商品中的一部分加征关税。
 </p>
 <p>
 </p>
 <p>
  至此，美中贸易战达到新规模。随着两国关税清单日益膨胀，此前一度幸免的日用品也终究难逃关税大棒。美国彼得森国际经济研究所(PIIE)的数据显示，周日美国加征的新一轮关税将把中国进口商品的平均关税从贸易战前的约3.1%提升至21.2%。
  <br/>
  <br/>
  中国人民大学国际关系学院教授金灿荣与社科院学者孙西辉周一在《人民日报》上联合发表文章，阐述了中方在贸易战中的坚定立场。
  <br/>
  <br/>
  文章开门见山，把这场贸易摩擦的起因再度归结为美国政府的单方面挑衅行为。文章说，美方行为严重损害两国利益，威胁全球产业链和供应链安全、托累了国际贸易和世界经济的增长。面对美方的极限施压和霸凌行为，中方保持了理性克制的态度，并反复强调贸易战中国“不愿打、不怕打、必要时不得不打”的思维。
 </p>
 <p>
  <br/>

 </p>
 <p>
  现居美国的中国经济问题专家梅凤杰认为，美国打响贸易战并非霸凌行为，而是对中国一方面希望进一步融入国际社会秩序，另一方面却屡屡背弃承诺的惩戒性动作。
  <br/>
  <br/>
  “美中贸易战其实跟当年大清帝国的贸易战（比如鸦片战争）有些神似，其实就是一个签约、反悔、再签约、再反悔的循环，进而触发了各种各样的战争。”
  <br/>
  <br/>
  但在金灿荣等人看来，贸易战淋漓尽致地展现了美国的霸权思维。文章说，美方一些人“只希望美国一家独大......认为中国迅速发展对美国霸权构成了挑战，对中国使出‘围追堵截’的百般手段。”对此，文章表示，中国社会上有少数人产生了“恐美崇美”的心理和言论，不是认为中国斗美国是“鸡蛋碰石头”，就是认为美国的做法总是有理有据的。
  <br/>
  <br/>
  由于金灿荣在中国国际关系领域颇负盛名，并身兼中共中央统战部和政府多个部门的咨询专家等职，这篇文章引发了公众的广泛关注。
  <br/>
  <br/>
  有网友评论说，如果大家都向金灿荣一样传播正能量，“美帝国主义”一定会认怂的。还有网友说，一味妥协换不来和平和尊重。迫于战略考量，中国过去也许的确妥协过，但现在到了它说“不”的时候了。
  <br/>
  <br/>
  这篇评论文章还指出，中国如果在美国霸权之下作出退让，就会犯下颠覆性历史错误。因此，只有坚持斗争，才能维护国家和人民的利益。
  <br/>
  <br/>
  梅凤杰表示，此类暗示中国政府官方立场的文章与中共宣传口的功能极为相似，到头来脱离不了它们的“忽悠”本质。
  <br/>
  <br/>
  “尽管当局在内心上极为希望与美国达成贸易协议，但是它的口号却喊得非常响亮，反美一定要旗帜鲜明。中共一直在言不由衷地做这些事情，所以像金灿荣这样的表态其实只是一个幌子，并不能够代表中共一定要把贸易战打下去。”
 </p>
 <p>
  <br/>
  <img class="image-inline captioned" src="../../../../resolveuid/540f88fb-226e-4ac3-b069-00892ed22842"/>
 </p>
 <p>
  近年来，中国国家主席习近平巩固了他的权力垄断地位，美国总统特朗普也站稳了政治脚跟。在这样的背景下，美国涌现出了一批直言不讳中共政治野心的人士。今年3月，20多位美国国防、政治、宗教等领域的知名人士在华盛顿成立了“应对中国当前危险委员会”( Committee on the Present Danger: China)，旨在通过教育和宣传等方式，帮助美国抵御中共对其造成的各种威胁。
  <br/>
  <br/>
  针对美国是否果真出于对中国国际实力的扩张，而决定通过贸易战对其进行遏制，美国独立政治学者顾为群表示，他的确注意到了美国对华强硬派政治分贝的增加，而从某种程度上，特朗普在贸易战中的政治意图也发生了遥相呼应的变化。
  <br/>
  <br/>
  “我认为特朗普和中国打贸易战最初的动机就是在双边贸易上逐渐达到一定程度的平衡。但随着时间的推移，他的目的也在变化。有迹象表明，特朗普目前把贸易战这个手段作为一套更大战略的一部分，就是说无论如何，美国不能让中国赶超。”
  <br/>
  <br/>
  但他说，美国社会的主流立场随着政府的更迭和民意的起伏，是充满变数的。这也意味着，中方某些人对美国战略目标的理解存在相当大的片面性。
  <br/>
  <br/>
  自由亚洲电台记者家傲华盛顿报道。 责编：申铧 网编：郭度
 </p>
